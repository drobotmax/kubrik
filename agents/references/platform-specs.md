# Ad Platform Specifications

## Meta Ads (Facebook / Instagram)

### Feed Ads
| Field | Recommended | Max | Notes |
|-------|------------|-----|-------|
| Primary text | 125 chars | 2,200 chars | Text truncates after ~125 on mobile |
| Headline | 27 chars | 40 chars | Below image |
| Description | 27 chars | 30 chars | Below headline, may not show |
| Image ratio | 1:1 | - | Also 4:5 for feed |
| Image size | 1080x1080 | - | Min 600x600 |
| Video length | 15 sec | 240 min | Under 15 sec recommended |

### Stories / Reels
| Field | Spec |
|-------|------|
| Format | 9:16 vertical |
| Resolution | 1080x1920 |
| Text safe zone | Central 60% of screen |
| Text overlay | Max 3 lines, large font |
| Duration (Stories) | Up to 15 sec |
| Duration (Reels) | Up to 90 sec |
| CTA | Swipe up / Link sticker |

### CTA Buttons (Meta)
Learn More, Shop Now, Sign Up, Book Now, Contact Us, Get Offer, Send Message, Download, Apply Now, Get Quote, Subscribe, Watch More

### Meta Ad Policies - Key Restrictions
- No personal attributes ("Are you overweight?", "Struggling with debt?")
- No misleading claims or fake urgency
- No before/after implying unrealistic results
- No discrimination by race, ethnicity, religion, sexual orientation
- No ALL CAPS text (occasional emphasis OK)
- Landing page must match ad content
- Special category: Housing ads have restricted targeting (no age, gender, zip code targeting)

## Google Ads

### Search Ads (Responsive)
| Field | Max chars | Count |
|-------|-----------|-------|
| Headline | 30 chars | Up to 15 (min 3) |
| Description | 90 chars | Up to 4 (min 2) |
| Display URL path | 15 chars | 2 fields |
| Final URL | - | 1 |

**Best practices:**
- Headline 1: include primary keyword
- Headline 2: benefit or value prop
- Headline 3: CTA
- Description 1: expand on benefit, include keyword
- Description 2: social proof or secondary benefit
- Pin critical headlines to positions 1-3

### Display Ads (Responsive)
| Field | Max chars |
|-------|-----------|
| Short headline | 30 chars (up to 5) |
| Long headline | 90 chars (1) |
| Description | 90 chars (up to 5) |
| Business name | 25 chars |

### Google Ads Policies - Key Restrictions
- No superlatives without third-party verification ("best", "#1")
- No excessive capitalization (ENTIRE WORDS)
- No gimmicky punctuation ("!!!", "???", "***")
- No misleading content
- No phone numbers in headlines/descriptions (use call extensions)
- No trademark violations
- No click-bait

## Character Counting Rules

- Spaces count as characters
- Emoji count as 2 characters on most platforms
- URLs don't count toward text limits (separate field)
- Line breaks count as 1 character
- Special characters (&, @, #) count as 1 character
