# Knowledge Base: Marketing Strategy Books

## Как книги связаны между собой

Эти 7 книг образуют **единую систему** для построения маркетинговой стратегии. Ниже — карта перекрёстных связей и порядок применения.

## Карта перекрёстных ссылок

### Breakthrough Advertising ↔ Другие книги

| Концепция Шварца | Связь с другой книгой | Как усиливают друг друга |
|-----------------|----------------------|------------------------|
| 5 Awareness Levels | **StoryBrand** — SB7 Framework | Awareness level определяет какой элемент SB7 выводить на первый план. Unaware → начни с проблемы (SB7 #2). Most Aware → сразу CTA (SB7 #5) |
| 5 Awareness Levels | **Cialdini** — Social Proof | Product Aware → Social Proof + Authority. Problem Aware → Reciprocity (бесплатная ценность). Unaware → Liking + Unity |
| Market Sophistication | **Blue Ocean** — ERRC | Stage 5 (насыщение) = Красный океан. Ответ: Blue Ocean ERRC для создания нового пространства |
| Market Sophistication | **Positioning** — Закон первенства | Stage 1 = будь первым. Stage 3+ = нужен новый механизм = новая категория (Positioning) |
| Mass Desire | **JTBD** — Jobs to be Done | Mass Desire = функциональная + эмоциональная + социальная работа. JTBD даёт метод исследования, Шварц даёт метод канализации |
| Механизм (Stage 3) | **Positioning** — Репозиционирование | Новый механизм = способ репозиционировать конкурентов ("их метод устарел, вот почему") |

### JTBD ↔ Другие книги

| Концепция JTBD | Связь | Как усиливают |
|---------------|-------|--------------|
| 4 Forces of Progress | **Sugarman** — 24 триггера | Push = триггеры #1 (вовлечённость), #16 (любопытство). Pull = #18 (мгновенное удовлетворение), #24 (надежда). Anxiety = #2 (честность), #9 (satisfaction conviction). Habit = #6 (обоснование), #14 (принадлежность) |
| 4 Forces of Progress | **Cialdini** — 7 принципов | Push → Social Proof ("другие тоже страдали"). Pull → Scarcity ("не упусти"). Anxiety → Authority + Reciprocity. Habit → Commitment (маленькие шаги) |
| 4 Forces of Progress | **StoryBrand** — Failure/Success | Push = Failure (SB7 #6). Pull = Success (SB7 #7). Anxiety = Plan (SB7 #4). Habit = Guide authority (SB7 #3) |
| Switch Interview | **Schwartz** — Mass Desire | Цитаты из switch-интервью = готовый копирайтинг на языке Mass Desire. Интервью выявляет доминирующее желание |
| Job Map (8 шагов) | **Blue Ocean** — Buyer Utility Map | Job Map 8 шагов ≈ Buyer Utility Map 6 стадий. Оба ищут pain points на каждом этапе клиентского пути |
| Functional/Emotional/Social jobs | **Schwartz** — 3 уровня продукта | Functional job = физический продукт. Emotional/Social job = продукт в действии |

### StoryBrand ↔ Другие книги

| Концепция SB7 | Связь | Как усиливают |
|--------------|-------|--------------|
| 3 уровня проблемы | **JTBD** — 3 типа работ | External = Functional job. Internal = Emotional job. Philosophical = Social job |
| 3 уровня проблемы | **Schwartz** — Awareness levels | Problem Aware → агитируй все 3 уровня. Solution Aware → фокус на external + механизм |
| Guide (Empathy + Authority) | **Cialdini** — Liking + Authority | Empathy = Liking + Unity. Authority = Authority + Social Proof |
| Plan | **Sugarman** — Slippery Slide | План = следующая ступенька горки. Каждый шаг плана тянет к следующему |
| Failure Stakes | **Cialdini** — Scarcity (Loss Aversion) | Failure = loss aversion. "Не потеряйте" работает в 2x сильнее чем "получите" |
| One-Liner | **Positioning** — Позиция в разуме | One-Liner = вербализация позиции. Оба требуют: одно слово/одна идея/сверхпростота |
| Villain | **Blue Ocean** — Red Ocean | Villain = проблема красного океана. "Хаос, перегрузка, непрозрачность" = враг, от которого Blue Ocean спасает |

### Positioning ↔ Другие книги

| Концепция | Связь | Как усиливают |
|-----------|-------|--------------|
| Закон первенства | **Schwartz** — Sophistication Stage 1 | Первый на пустом рынке = прямое заявление. Оба говорят: "Be simple. Be direct." |
| Репозиционирование | **Schwartz** — Механизм (Stage 3) | Репозиционирование = "их механизм сломан". Новый механизм = новая позиция |
| Line Extension Trap | **Blue Ocean** — Value Innovation | Расширение линейки = красный океан (больше того же). Value Innovation = голубой океан (принципиально новое) |
| "Против"-позиция | **StoryBrand** — Villain | Позиция "против" = назвать villain. 7UP "The Uncola" = villain это кола |
| Владеть словом | **StoryBrand** — One-Liner | Одно слово в голове = ядро one-liner'а. Volvo = "безопасность". FedEx = "overnight" |

### Blue Ocean ↔ Другие книги

| Концепция | Связь | Как усиливают |
|-----------|-------|--------------|
| Не-покупатели (3 уровня) | **JTBD** — Не-потребители Кристенсена | Те же люди. JTBD объясняет почему не покупают (работа не закрыта). Blue Ocean показывает как их привлечь (ERRC) |
| Strategy Canvas | **Schwartz** — Market Sophistication | Canvas визуализирует sophistication рынка. Stage 5 = все кривые слились. Blue Ocean = нарисовать новую кривую |
| Путь 5 (Функция vs Эмоция) | **StoryBrand** — Internal Problem | Функциональные отрасли игнорируют internal problem. Добавить эмоцию = StoryBrand internal + philosophical |
| ERRC | **Positioning** — Нишевое позиционирование | Eliminate/Reduce = отказ от части рынка (жертва по Трауту). Create = новая категория (закон первенства) |

### Cialdini ↔ Sugarman

| Чалдини | Шугерман | Синергия |
|---------|----------|----------|
| Reciprocity | Триггер #22 (Вина) | Оба: дай ценность бесплатно → создай обязательство |
| Commitment | Slippery Slide | Серия маленьких "да" (Чалдини) = серия предложений горки (Шугерман) |
| Social Proof | Триггер #14 (Принадлежность) | Оба: "другие уже с нами" |
| Authority | Триггер #8 (Авторитет) | Идентичная концепция, разные примеры |
| Scarcity | Триггер #17 (Срочность) + #19 (Эксклюзивность) | Чалдини даёт психологию, Шугерман — копирайтинг-формулы |
| Unity | Триггер #14 (Принадлежность) | Unity глубже: не "хочу быть частью", а "я уже часть" |

## Практическое применение: чеклист для стратега

### Перед написанием любого текста — ответь на 7 вопросов:

1. **Какой awareness level у аудитории?** (Schwartz) → определяет тип заголовка
2. **Какую работу нанимают продукт?** (JTBD) → определяет messaging
3. **Где мы на лестнице в голове?** (Positioning) → определяет позицию
4. **Красный или голубой океан?** (Blue Ocean) → определяет стратегию
5. **Кто герой, кто наставник?** (StoryBrand) → определяет нарратив
6. **Какие триггеры использовать?** (Sugarman + Cialdini) → определяет тактику
7. **Какая стадия sophistication рынка?** (Schwartz) → определяет формулу

### Порядок работы над стратегией клиента:

1. **Исследование:** JTBD switch-интервью + Blue Ocean strategy canvas
2. **Позиционирование:** Positioning (незанятая ниша) + JTBD (через работу)
3. **Messaging:** StoryBrand BrandScript + Schwartz awareness-adapted headlines
4. **Копирайтинг:** Sugarman structure + Cialdini principles embedded
5. **Итерация:** JTBD 4 forces analysis + Schwartz sophistication check

## Рекомендуемое чтение для углубления

### По каждой книге — что читать дальше:

**После Breakthrough Advertising:**
- "Great Leads" (Michael Masterson) — расширение системы заголовков Шварца
- "Cashvertising" (Drew Eric Whitman) — современная адаптация принципов

**После JTBD:**
- "Competing Against Luck" (Christensen) — JTBD в полном формате
- "Demand-Side Sales" (Bob Moesta) — практика switch-интервью
- "When Coffee and Kale Compete" (Alan Klement) — теория JTBD

**После StoryBrand:**
- "Marketing Made Simple" (Donald Miller) — воронка по StoryBrand
- "The Hero with a Thousand Faces" (Joseph Campbell) — оригинал мономифа

**После Sugarman:**
- "The Boron Letters" (Gary Halbert) — письма о директ-маркетинге
- "Ogilvy on Advertising" (David Ogilvy) — классика рекламы

**После Cialdini:**
- "Pre-Suasion" (Cialdini) — что делать ДО убеждения
- "Thinking, Fast and Slow" (Kahneman) — когнитивные искажения

**После Blue Ocean:**
- "Blue Ocean Shift" (Kim & Mauborgne) — практическое руководство
- "Zero to One" (Peter Thiel) — создание новых рынков

**После Positioning:**
- "The 22 Immutable Laws of Marketing" (Ries & Trout) — расширение законов
- "Obviously Awesome" (April Dunford) — современный positioning framework
