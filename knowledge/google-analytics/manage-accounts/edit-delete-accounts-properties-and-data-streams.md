Discover how to add, delete, create, and manage properties, accounts, and data streams in your Google Analytics account.

You need the [Editor role](https://support.google.com/analytics/answer/9305587) to edit and delete accounts, properties, and data streams in a Google Analytics account.

*   [Edit an account](https://support.google.com/analytics/answer/9304776?hl=en#edit-account)
*   [Edit a property](https://support.google.com/analytics/answer/9304776?hl=en#edit-property)
*   [Edit a data stream](https://support.google.com/analytics/answer/9304776?hl=en#edit-data-stream)
*   [Delete an account](https://support.google.com/analytics/answer/9304776?hl=en#delete-account)
*   [Delete a property](https://support.google.com/analytics/answer/9304776?hl=en#delete-property)
*   [Delete a data stream](https://support.google.com/analytics/answer/9304776?hl=en#delete)

## Edit an account

1.   In **[Admin](https://analytics.google.com/analytics/web/#/?pagename=admin&utm_source=gahc&utm_medium=dlinks)**, under _Account_, click [**Account details**](https://analytics.google.com/analytics/web/#/?pagename=admin-account&utm_source=gahc&utm_medium=dlinks). 
**Note**: The previous link opens to the last Analytics property you accessed. You must be signed in to a Google Account to open the property. You can [change the property](https://support.google.com/analytics/answer/12813202) using the property selector. You must be an [Editor or above](https://support.google.com/analytics/answer/9305587)at the account level to edit an account.

2.   Modify the account name, country of business, and data-sharing and Data Processing Amendment options as necessary.
3.   Click **Save**.

## Edit a property

1.   In **[Admin](https://analytics.google.com/analytics/web/#/?pagename=admin&utm_source=gahc&utm_medium=dlinks)**, under _Property_, click [**Property details**](https://analytics.google.com/analytics/web/#/?pagename=admin-property&utm_source=gahc&utm_medium=dlinks). 
**Note**: The previous link opens to the last Analytics property you accessed. You must be signed in to a Google Account to open the property. You can [change the property](https://support.google.com/analytics/answer/12813202) using the property selector. You must be an [Editor or above](https://support.google.com/analytics/answer/9305587)at the property level to edit a property.

2.   Modify any of these settings (Learn more about [property settings](https://support.google.com/analytics/answer/9304153#property)): 
    *   Property name
    *   Industry category
    *   Reporting time zone
    *   Currency

3.   Click **Save**.

#### Edit a subproperty or roll-up property

In general, you edit subproperties and roll-up properties using the method described above, though there are some differences:

*   With subproperties, you can also edit the filter that determines which data is available. Learn more about [editing subproperties](https://support.google.com/analytics/answer/11526072#edit-subproperty).
*   With roll-up properties, you can also add and remove source properties. Learn more about [editing roll-up properties](https://support.google.com/analytics/answer/11539651#add-remove).

## Edit a data stream (Google Analytics 4 properties)

1.   In **[Admin](https://analytics.google.com/analytics/web/#/?pagename=admin&utm_source=gahc&utm_medium=dlinks)**, under _Data collection and modification_, click [**Data streams**](https://analytics.google.com/analytics/web/#/?pagename=admin-streams&utm_source=gahc&utm_medium=dlinks). 
**Note**: The previous link opens to the last Analytics property you accessed. You must be signed in to a Google Account to open the property. You can [change the property](https://support.google.com/analytics/answer/12813202) using the property selector. You must be an [Editor or above](https://support.google.com/analytics/answer/9305587)at the property level to edit a data stream.

2.   Click the name of the data stream that you want to edit. The stream details screen is displayed.

#### Edit data stream name or URL (web)

From the stream details screen you can change the name or URL of a web data stream.

1.   Click the edit ![Image 1](https://storage.googleapis.com/support-kms-prod/YbU4raT0O6DuvMq0dr1swxJPIkq1SuqhcKTj) icon in the upper-right corner of the Web stream details screen.
2.   Enter the new name for the data stream in the **Stream name** field.
3.   To update the **Website URL**, select the protocol (http or https) and enter the new URL domain and path.
4.   Click **Update stream** to save your changes.

#### Edit data stream properties and settings

You can also take the following actions related to the data stream:

*   Download the latest config file for your app (app)
*   Change [enhanced measurement settings](https://support.google.com/analytics/answer/9216061) (web)
*   Add or remove [connected site tags (web)](https://support.google.com/analytics/answer/9973999)
*   [Generate Measurement Protocol API secrets](https://support.google.com/analytics/answer/9814495) (web)
*   [Modify and create events](https://support.google.com/analytics/answer/10085872)(app and web)
*   Collect Universal Analytics events in your Google Analytics 4 property
*   [Configure cross-domain measurement (web)](https://support.google.com/analytics/answer/10071811)
*   [Filter out internal traffic (web)](https://support.google.com/analytics/answer/10104470)
*   [Identify unwanted referrals (web)](https://support.google.com/analytics/answer/10327750)
*   Adjust the session timeout

## Delete a data stream (Google Analytics 4 properties)

1.   In **[Admin](https://analytics.google.com/analytics/web/#/?pagename=admin&utm_source=gahc&utm_medium=dlinks)**, under _Data collection and modification_, click [**Data stream**](https://analytics.google.com/analytics/web/#/?pagename=admin-streams&utm_source=gahc&utm_medium=dlinks). 
**Note**: The previous link opens to the last Analytics property you accessed. You must be signed in to a Google Account to open the property. You can [change the property](https://support.google.com/analytics/answer/12813202) using the property selector. You must be an [Editor or above](https://support.google.com/analytics/answer/9305587)at the property level to delete a data stream.

2.   Click the row for the data stream.
3.   In the upper right, click ![Image 2: More](https://lh3.googleusercontent.com/oLoRPrHJd7m46sWijX6zBWnEnfslP62AxJSwt5Nj0bNbpaYHz2pyscExleiofsH2kQ=h36)**> Delete stream**.

**Notes**:

*   If you delete a data stream, you can’t undelete or recover it.
*   After a data stream is deleted, any product integrations that rely on that data stream (for example, Big Query Export) will no longer work.
*   As of now, app data streams need to be deleted twice. After deleting the stream in the GA4 property, you must also remove it in Firebase. In the Firebase project, go to **Project settings**> select **App**> click **Remove App** at the bottom of the page.

### If you're using connected site tags

If you're using connected site tags with a web data stream you want to delete, remove the connected site tags before you delete the data stream. Learn more about [removing connected site tags](https://support.google.com/analytics/answer/9973999#add-remove).
