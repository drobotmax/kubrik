# О партнерских торговых интеграциях для событий на сайте
Это статья с пошаговыми инструкциями по интеграции [пикселя Meta](https://www.facebook.com/business/help/952192354843755?helpref=commercepartner) и [Conversions API](https://www.facebook.com/business/help/2041148702652965?helpref=commercepartner) с популярными торговыми платформами, такими как WordPress, OpenCart, WooCommerce, Shopify, BigCommerce и Ecwid. Партнерские интеграции Meta упрощают настройку, улучшают отслеживание событий и результативность рекламы благодаря точному измерению конверсий и поведения клиентов.
## Прежде чем начать
  * **Доступ администратора к бизнес-портфолио Meta.** У вас должен быть доступ администратора или разрешение на управление объектами компании на платформах Meta (Страницей Facebook, профилем Instagram, каталогом, рекламным аккаунтом, пикселем Meta).
  * **Двухфакторная аутентификация.** Включите двухфакторную аутентификацию для бизнес-портфолио Meta.
  * **Соответствие товаров требованиям.** Убедитесь, что товары соответствуют [Нормам рекламной деятельности Meta и Правилам торговли](https://www.facebook.com/business/help/488043719226449).
  * **Доступ администратора к платформе.** Убедитесь, что у вас есть доступ администратора к бэкенд-панели управления вашей платформой электронной коммерции.
  * **Актуальные плагины и приложения.** Установите последние версии приложений или плагинов интеграции, чтобы обеспечить совместимость.


## Ключевые понятия
  * **Пиксель Meta** — фрагмент кода, который отслеживает действия посетителей на вашем сайте.
  * **Conversions API** — средство отслеживания событий на стороне сервера, которое дополняет пиксель для повышения точности данных.
  * **Партнерские интеграции** — готовые коннекторы между Meta и торговыми платформами, которые упрощают настройку пикселя и Conversions API без редактирования кода вручную.


## Преимущества
  * Настройка автоматического отслеживания событий.
  * Улучшение точности данных и сопоставления событий.
  * Упрощенный процесс обновления и поддержки работы.
  * Доступ к расширенным функциям пикселя Meta, таким как дедупликация событий и улучшенное измерение.

## Пошаговые инструкции по настройке интеграции для каждой платформы
Выполните эти шаги, чтобы интегрировать пиксель Meta и Conversions API с выбранной торговой платформой. Общие этапы настройки применимы ко всем платформам. Далее приведены инструкции для отдельных платформ.
## Общие этапы настройки (для всех платформ)
  1. **Откройте Meta Events Manager.** Перейдите в [Events Manager](https://www.facebook.com/events_manager2).
  2. **Откройте вкладку "Источники данных".** Выберите свой пиксель или создайте новый.
  3. Нажмите **Настройки**. Прокрутите до пункта **Выбрать партнера**.
  4. **Выберите коммерческого партнера.** Доступные варианты: WordPress, OpenCart, WooCommerce, Shopify, BigCommerce, Ecwid.
  5. **Следуйте инструкциям для своей платформы.** Инструкции по интеграции отличаются в зависимости от платформы.


## Инструкции для разных платформ
  * **OpenCart.** Партнерская интеграция через Events Manager или [расширение OpenCart](https://www.facebook.com/business/help/1494437460610744?helpref=commercepartner). Некоторые версии могут требовать установки расширения вручную.
  * **WooCommerce.** Используйте [официальный плагин Meta для WooCommerce](https://www.facebook.com/business/help/900699293402826?helpref=commercepartner). Поддерживает автоматическое сопоставление событий и интеграцию Conversions API.
  * **Shopify.** [Нативная интеграция через Meta Events Manager.](https://www.facebook.com/business/help/646757258759189?helpref=commercepartner) Обеспечивает бесперебойную передачу и точность событий.
  * **BigCommerce.** [Партнерская интеграция через Events Manager.](https://www.facebook.com/business/help/1581202675515016?helpref=commercepartner) Поддерживает отслеживание событий на стороне сервера через Conversions API.
  * **Ecwid.** [Партнерская интеграция в Events Manager.](https://www.facebook.com/business/help/227503504841825?helpref=commercepartner) Поддерживает улучшенное отслеживание событий с минимальной настройкой.


## Отслеживание и устранение проблем
  * **Проверяйте статус пикселя.** Используйте расширение для браузера [Помощник по пикселям Meta (Pixel Helper)](https://www.facebook.com/business/help/198406697184603?helpref=commercepartner).
  * **Отслеживайте данные в Events Manager.** Проверяйте данные о диагностике и качество подключения.
  * **Подтверждайте события Conversions API.** Убедитесь, что события сервера регистрируются и дедуплицируются.


## Распространенные проблемы
  * **Пиксель не срабатывает.** Убедитесь, что плагин и расширение установлены.
  * **События не сопоставляются.** Проверьте сопоставление событий в партнерской интеграции.
  * **Низкое качество подключения.** Повторно подтвердите или назначьте наборы данных.


## Рекомендации
  * Регулярно обновляйте плагины и расширения до последних версий.
  * Используйте партнерские интеграции, чтобы избежать ошибок, характерных для редактирования кода вручную.
  * Выбирайте события с учетом бизнес-целей.
  * Еженедельно отслеживайте качество подключения и данные о диагностике событий.
  * Используйте Conversions API вместе с пикселем, чтобы повысить точность данных.
