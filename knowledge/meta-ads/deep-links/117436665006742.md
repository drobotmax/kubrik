# Устранение проблем с подтверждением адреса для выставления счетов по кредитной карте
Когда вы пытаетесь добавить карту в аккаунт, мы проверяем, совпадает ли указанный вами адрес с адресом для выставления счетов по кредитной карте.
Если вы получили сообщение о том, что ваш адрес для выставления счетов по кредитной карте не удалось подтвердить:
  * **Проверьте почтовый индекс**. Убедитесь, что почтовый индекс для кредитной карты, который вы ввели на нашем сайте, совпадает с указанным в ваших выписках по этой карте. 
  * **Обратитесь к поставщику платежных услуг**. Если индекс правильный, обратитесь в свой банк или к эмитенту кредитной карты, чтобы узнать, почему она не принимается на Facebook.
