# Добавление правил для данных товаров в Commerce Manager
Вы можете создать правила для данных, чтобы автоматически изменять сведения о товарах каждый раз, когда они загружаются в ваш каталог из фида данных или с партнерской платформы. Правила для данных позволяют исправлять проблемы с форматированием, обеспечивать соответствие информации о товарах нашим [требованиям](https://www.facebook.com/business/help/120325381656392) и применять к ней сразу несколько изменений.
## Прежде чем начать
У вас должен быть [полный доступ к каталогу](https://www.facebook.com/help/1953352334878186).
## Добавление правил для данных товаров
Чтобы добавить правила для данных товаров:
  1. Откройте [Commerce Manager](https://www.facebook.com/commerce_manager) и выберите нужный каталог.
  2. Откройте вкладку **Каталог** и нажмите **Источники данных**.
  3. Выберите нужный фид данных или партнерскую интеграцию. Откройте вкладку **Настройки**.
  4. В разделе **Правила фида данных** или **Правила для данных товаров** нажмите **Добавить правило** и выберите в раскрывающемся меню нужное правило. С помощью правил можно:
     * **Изменить регистр**. Вы можете выбрать один из следующих способов капитализации начальных букв: Каждое Слово, Первое слово предложения, все строчные или ВСЕ ПРОПИСНЫЕ. Например, если описания ваших товаров написаны заглавными буквами, к полю **description** ("описание"). Это позволит избежать ошибок, поскольку описание не должно состоять только из заглавных букв.
     * **Сопоставить поле**. Сопоставьте предоставленное вами поле с полем, которое соответствует нашим [требованиям](https://www.facebook.com/business/help/120325381656392). Это правило можно применить, если вы указали неподдерживаемое название поля либо использовали фид данных Amazon или Google, требования которых могут немного отличаться от наших. Например, если идентификаторы ваших товаров указаны в поле "SKU", вы можете добавить правило, чтобы оно сопоставлялось с полем **id** ("идентификатор").
     * **Найти и заменить** определенную информацию в поле. Это правило позволяет заменять отредактированные или неправильные значения. Например, в поле **price** ("цена") не поддерживается использование в десятичных числах запятых вместо точек. Вы можете добавить правило, согласно которому "," будет заменяться на ".". В результате 10,50 EUR будет заменено на 10.50 EUR. Это поле чувствительно к регистру. Так, правило для замены слов "красный" на "синий" не исправит слова "Красный". Вам потребуется добавить ещё одно правило для замены "Красный" на "синий" или выполнить замену по регулярному выражению.
     * **Установить значения по умолчанию**. Подстановка значения по умолчанию в пустое поле. Это правило позволяет заполнять пустые или добавлять недостающие поля. Например, если для нескольких позиций поле **availability** ("наличие") не заполнено, вы можете добавить правило, по умолчанию задающее для поля **availability** значение **in stock** ("в наличии"). Если вы продаете женскую одежду и не заполнили поле **gender** ("пол"), то можете установить значение по умолчанию **female** ("женский") для поля **gender**.
     * **Заменить по регулярному выражению**. Поиск и замена строки текста или чисел с использованием регулярных выражений. Регулярные выражения позволяют задавать точность поискового запроса. Вы можете использовать дополнительные символы либо определенную последовательность символов или чисел для поиска. Например, можно добавить правило для замены всех 10-значных чисел на другое число. При желании вы можете заменить все варианты слова или фразы. Например, "цвет" и "цвета" на "оттенок" либо "красный" и "Красный" на "синий".
  5. Выберите поле, к которому нужно применить правило, и укажите нужные изменения. Затем нажмите **Создать**. Повторите эти действия, чтобы добавить больше правил. К каждому полю можно применить несколько правил для поиска и замены, а также замены по регулярному выражению, но не больше одного правила другого типа.
  6. Когда всё будет готово, нажмите **Применить правила** и подтвердите действие. Обычно все правила применяются к данным товаров одновременно. Однако если вы добавите несколько правил для поиска и замены или замены по регулярному выражению, они будут применяться в порядке добавления.


Правила для данных товаров будут применяться ко всем будущим загрузкам фида данных или синхронизациям информации с партнерской платформы.
Правила можно в любой момент отредактировать или удалить. Чтобы отредактировать правило, найдите его в настройках источника данных в разделе **Правила фида данных** или **Правила для данных товаров**. Нажмите **Смотреть** рядом с правилом и выберите **Редактировать**. Внесите изменения и нажмите **Применить**. Чтобы удалить правило, нажмите на значок ![](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=5178541075543443&version=1761104322&transcode_extension=webp), выберите **Удалить**
**Примечание.** Метка **Применяется автоматически** рядом с правилом указывает на то, что мы применили его, чтобы исправить проблему с данными ваших товаров.
