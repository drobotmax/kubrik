# О базовой настройке
Компания Meta внедрила [_ограничения_](https://www.facebook.com/legal/businesstech) в отношении видов информации, которой компании могут делиться с нами через [_инструменты Meta для бизнеса_](https://www.facebook.com/help/331509497253087). Мы не разрешаем рекламодателям использовать инструменты Meta для бизнеса, чтобы делиться информацией о людях, запрещенной [_Условиями использования инструментов Meta для бизнеса_](https://www.facebook.com/legal/terms/businesstools?_rdr). Сюда относятся данные, которые могут считаться конфиденциальными, в том числе согласно применимым законам и отраслевым нормам, а также данные, которые иным образом не разрешены нашими условиями. [_Подробнее о запрещенной информации_](https://www.facebook.com/business/help/361948878201809).
Базовая настройка — это набор ограничений передачи данных, который вы или Meta можете применить к инструменту Meta для бизнеса. Включить ограничения передачи данных для используемого вами инструмента Meta для бизнеса можно в [_Meta Events Manager_](https://www.facebook.com/events_manager2/).
Базовая настройка ограничивает передачу пользовательских [_параметров_](https://www.facebook.com/business/help/308258403676519) и частей URL, следующих за доменом.
Ограничения данных на уровне базовой настройки не заменяют ваши внутренние механизмы соблюдения правил, однако помогают предотвратить передачу информации, которая может быть запрещена [Условиями использования инструментов Meta для бизнеса](https://www.facebook.com/legal/businesstech).
## Типы данных в инструментах Meta для бизнеса, затрагиваемые ограничениями на уровне базовой настройки
Ограничения на уровне базовой настройки применяются к следующим типам данных:
  * **Пользовательские параметры.** Стандартные параметры — это предопределенные поля, которые рекламодатели могут добавлять к своим событиям, например параметры value и currency для события покупки или параметр content_id для события просмотра контента. Если ваш параметр не входит в [список стандартных](https://developers.facebook.com/docs/marketing-api/conversions-api/parameters/custom-data), он считается пользовательским. [Подробнее о стандартных и пользовательских параметрах.](https://www.facebook.com/business/help/308258403676519)
  * **Части URL, следующие за доменом.** Например, URL https://jaspersmarket.com/clothes/summer/dresses?item=10 сокращается до [https://jaspersmarket.com/](https://l.facebook.com/l.php?u=https%3A%2F%2Fjaspersmarket.com%2F&h=AT4HmI5ThXHRrkK1FVrh8zV-b0EzeYh0v5w5zatoXlJ83LBWaJaQLkqTBsyRDrqY54TR8tsP1wFdEpWRvniv-l8NTAWU4UTzED-4g69sSRp3-8hetN1avr110gUYcpEJJR6nnNldPircIA).


## Влияние ограничений данных на уровне базовой настройки
Если вы используете инструмент Meta для бизнеса, к которому применяются ограничения данных на уровне базовой настройки, то можете заметить ограничения приведенных ниже функций для рекламодателей.
  * **Пользовательские аудитории могут не работать должным образом.**[Пользовательские аудитории](https://www.facebook.com/business/help/744354708981227) могут не работать должным образом, если в них используются пользовательские параметры или части URL, следующие за доменом.
    * Пользовательские аудитории, правила которых зависят от пользовательских параметров или URL, не могут предварительно заполняться. Кроме того, они могут быть недоступны или наполняться дольше.
    * Пользовательские аудитории невозможно создать и они не будут обновляться, если содержат правила устройств или значения агрегирования вместе с пользовательскими параметрами либо URL.
    * Чтобы просмотреть и обновить пользовательские аудитории, перейдите в раздел [Аудитории](https://adsmanager.facebook.com/adsmanager/audiences).
  * Уменьшение пользовательской аудитории может затронуть **показ групп объявлений**. Также группы объявлений могут перестать обновляться из-за того, что для инструмента Meta для бизнеса включена базовая настройка. Вы можете обновить пользовательскую аудиторию или создать новую для своих групп объявлений. Чтобы просмотреть и обновить пользовательские аудитории, перейдите в раздел [Аудитории](https://adsmanager.facebook.com/adsmanager/audiences).
  * **Информация в[Meta Events Manager](https://www.facebook.com/events_manager2/) и других продуктах может стать недоступной.** К примеру, пользовательские параметры и некоторые части URL, следующие за доменом. Это касается выборки действий и инструмента "Тестирование событий". Подробнее в [Meta Events Manager](https://www.facebook.com/events_manager2/).
  * **Автоматический расширенный поиск совпадений может быть недоступен** , если для вашего инструмента Meta для бизнеса включена базовая настройка. В таком случае эту функцию можно настроить вручную. Подробнее на сайте [Meta for Developers](https://developers.facebook.com/docs/meta-pixel/advanced/advanced-matching).
  * **Обновление каталога с помощью пикселя Meta** может не работать. Однако вы можете добавлять позиции в каталог вручную. [Подробнее.](https://www.facebook.com/business/help/384041892421495)
  * **Специально настроенные события требуют проверки.** Проверьте специально настроенные события (на сайте, в приложении и офлайн), которые отправляются через инструменты Meta для бизнеса. Вы должны решить, какие специально настроенные события нужно подтвердить, а какие — заблокировать. Подтверждайте события, которые хотите передавать в Meta, только если они соответствуют [Условиям использования инструментов Meta для бизнеса](https://www.facebook.com/legal/terms/businesstools?_rdr) и их данные правильно отформатированы. Подтвержденные специально настроенные события можно использовать для рекламных целей, например в кампаниях, специально настроенных конверсиях и пользовательских аудиториях. Подробнее о том, [как проверить специально настроенные события в Meta Events Manager](https://www.facebook.com/business/help/642275438032995). 


## Причины, по которым к инструментам Meta для бизнеса могут применяться ограничения данных на уровне базовой настройки
Ограничения на уровне базовой настройки могут применяться к инструментам Meta для бизнеса по одной из следующих причин:
  1. Согласно определению Meta источник данных относится к категории, для которой требуются ограничения данных на уровне базовой настройки. [Подробнее о категориях источников данных в Meta Events Manager.](https://www.facebook.com/business/help/1402913027039332)
  2. Вы (или кто-то другой в вашем аккаунте) назначили набору или источнику данных категорию, для которой установлены ограничения. Подробнее о том, [_как присваивать категории наборам и источникам данных в Meta Events Manager_](https://www.facebook.com/business/help/1666235380728268).
  3. Вы получили несколько уведомлений о том, что передаваемые вами данные могут нарушать [Условия использования инструментов Meta для бизнеса](https://www.facebook.com/legal/terms/businesstools?_rdr). Подробнее о том, [как проверить заблокированные параметры в Meta Events Manager](https://www.facebook.com/business/help/3720004518245751).
  4. Вы включили уведомления об ограничениях данных на уровне базовой настройки в Meta Events Manager. Подробнее о том, [как включить или выключить ограничения данных для инструментов Meta для бизнеса в Meta Events Manager](https://www.facebook.com/business/help/423150200627651).
