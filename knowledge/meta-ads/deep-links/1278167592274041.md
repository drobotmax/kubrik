# О партнерских интеграциях для офлайн-событий
![](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=438223955063449&version=1666716831&transcode_extension=webp)
Поддержка Offline Conversions API прекратится в мае 2025 года. Вы не сможете использовать Offline Conversions API, чтобы загружать события в группы офлайн-событий. Рекомендуем загрузить события в группы событий и интегрировать Conversions API вместо Offline Conversions API. [Подробнее об этих изменениях и подготовке к ним.](https://www.facebook.com/business/help/1835755323554976)
Многие партнерские интеграции упрощают загрузку данных об офлайн-событиях. Ваш партнер может специализироваться на точках продаж (PoS), системе управления взаимодействием с клиентами (CRM) или других системах, используемых для работы с клиентами. Эти интеграции позволяют автоматически загружать данные об офлайн-событиях и сокращают количество задач, которые приходится выполнять вручную.
Чтобы использовать партнерскую интеграцию, перейдите по ссылке на **документацию партнера** ниже (если она доступна) или свяжитесь с партнером и получите помощь по настройке интеграции.
**Примечание.** Отправку офлайн-событий через Conversions API поддерживают не все партнерские интеграции.
