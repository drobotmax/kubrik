# Настройка каталога гостиниц
[Реклама гостиниц](https://www.facebook.com/business/help/1320285461344901) — это [объявления из каталога Meta Advantage+](https://www.facebook.com/business/help/733979527611858), оптимизированные для охвата людей, которые планируют забронировать номер в гостинице. Реклама гостиниц использует ваш пиксель Meta или SDK, чтобы показывать людям наиболее актуальные предложения с учетом их интересов и поведения. 
Чтобы запустить рекламу гостиниц, нужно создать и заполнить каталог — файл, который содержит данные о гостиницах.
Если вы разработчик, ознакомьтесь с документацией по API на сайте [Meta for Developers](https://developers.facebook.com/docs/marketing-api/hotel-ads/).
  * [Шаг 1. Создание каталога](https://www.facebook.com/business/help/1291772977600258?id=221593351918232#create). 
  * [Шаг 2. Создание или загрузка фида данных о гостиницах](https://www.facebook.com/business/help/1291772977600258?id=221593351918232#setup).
  * [Шаг 3. Добавление групп гостиниц](https://www.facebook.com/business/help/1291772977600258?id=221593351918232#add).
  * [Шаг 4. Подключение пикселя или SDK для мобильных устройств](https://www.facebook.com/business/help/1291772977600258?id=221593351918232#associate).


[](https://www.facebook.com/business/help/1291772977600258?id=221593351918232)
## Шаг 1. Создание каталога
Узнайте, [как создать каталог в Commerce Manager](https://www.facebook.com/business/help/1275400645914358).
[](https://www.facebook.com/business/help/1291772977600258?id=221593351918232)
## Шаг 2. Загрузка информации о гостинице
Фид содержит информацию о рекламируемых вами гостиницах, в том числе название гостиницы, местоположение, начальная цена и изображения. Вы можете указать все гостиницы в одном фиде или загрузить разные фиды для определенных групп гостиниц (например, расположенных в одной стране или принадлежащих одному бренду). 
**Примечание**. Рекламодатели, которые рекламируют гостиницы, могут создать каталог непосредственно из [фида со списком гостиниц от Google](https://l.facebook.com/l.php?u=https%3A%2F%2Fdevelopers.google.com%2Fhotels%2Fhotel-ads%2Fxml-reference%2Fhotel-list-feed&h=AT5A97KVnZS83opSScviwKjg451C8UjOzTvo-XVT-onXHAyI4Lw9L3RQPZbp04Q5aFgNP7PCe7oOfeOZZYkc_I-Jtw1rsRYDcWQjotfcnC_tybgB1ctRh73VtyoeS3RgrQ6sFNP-W28Jtw). Если у вас есть такой фид, вы можете пропустить этот шаг и сразу перейти к инструкциям по загрузке. 
**Создание фида данных о гостиницах**
Узнайте, [как создать фид данных](https://www.facebook.com/business/help/1898524300466211). Убедитесь, что вы добавили все [обязательные поля для гостиниц](https://developers.facebook.com/docs/marketing-api/hotel-ads/catalog) (Meta for Developers), и выполните все требования, чтобы обеспечить правильную загрузку данных о гостиницах.
**Загрузка фида данных о гостиницах**
Создав фид данных о гостиницах, [загрузите его в каталог](https://www.facebook.com/business/help/125074381480892). Вы можете загрузить фид один раз или составить расписание загрузок (ежечасно, ежедневно или еженедельно), чтобы поддерживать информацию в каталоге в актуальном состоянии.
[](https://www.facebook.com/business/help/1291772977600258?id=221593351918232)
## Шаг 3. Добавление групп гостиниц 
В каталоге вы можете объединять свои гостиницы в группы с помощью фильтров (таких как бренд или цена), чтобы определить, какие гостиницы следует добавлять. [Подробнее о создании группы](https://www.facebook.com/help/620275848114281).
[](https://www.facebook.com/business/help/1291772977600258?id=221593351918232)
## Шаг 4. Подключение пикселя или SDK для мобильных устройств
Чтобы запустить рекламу гостиниц, нужно подключить к каталогу пиксель Meta. Так вы сможете показывать позиции из каталога людям, которые уже искали их или взаимодействовали с вашим ассортиментом. Если у вас нет настроенного пикселя или SDK для мобильных устройств, нажмите [здесь](https://www.facebook.com/business/help/308454139621869). После создания пикселя и каталога, узнайте, [как их подключить](https://www.facebook.com/business/help/1044262445604547).
