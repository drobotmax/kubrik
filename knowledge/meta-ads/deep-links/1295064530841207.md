# Подготовка компании к использованию Conversions API
Если вы уже слышали об [_API Conversions_](https://www.facebook.com/business/help/AboutConversionsAPI), то, возможно, хотите начать работу с этим инструментом. Как правило, прежде чем составить список задач и выполнить интеграцию, маркетологи консультируются с разными специалистами организации и обсуждают время внедрения, необходимые ресурсы и преимущества API Conversions.
Эти советы помогут вам и специалистам вашей компании принять правильные решения и подготовиться к использованию инструмента.
  * **Подумайте, какую пользу может принести API Conversions вашей компании**. Этот инструмент обеспечивает прямую связь между вашими маркетинговыми данными и системами Meta, которые помогают оптимизировать таргетинг рекламы, снизить цену за действие и измерить результаты. Узнайте больше о [_преимуществах API Conversions_](https://www.facebook.com/business/help/2041148702652965?id=818859032317965) и ознакомьтесь с [_историями успеха других компаний, использующих этот инструмент_](https://www.facebook.com/business/success/categories/conversions-api).
  * **Выберите подходящий способ настройки и убедитесь в наличии необходимых ресурсов**. Разные способы настройки требуют привлечения разных ресурсов и инструментов. Например, шлюз API Conversions и партнерская интеграция — более простые и быстрые варианты настройки, в то время как прямая интеграция позволяет лучше контролировать процесс, но требует поддержки разработчиков. [_Сравните варианты настройки API Conversions_](https://www.facebook.com/business/help/433493041367251?id=818859032317965).
  * **Заранее обсудите внедрение API Conversions с соответствующими специалистами вашей компании**. Вам может понадобиться время, чтобы заручиться поддержкой нужных специалистов, поэтому рекомендуем начинать подготовку как можно раньше. Ниже вы найдете полезные ресурсы об инструменте, которые помогут в коммуникации с коллегами.
    * Для руководителей отделов маркетинга и людей, ответственных за принятие бизнес-решений:
      * [_Вводная информация об API Conversions_](https://www.facebook.com/business/tools/facebook-conversions-api)
      * [_Истории успеха, связанные с API Conversions_](https://www.facebook.com/business/success/categories/conversions-api)
    * Для юристов:
      * [_Инструменты Meta для бизнеса: рекомендации по соблюдению конфиденциальности и использованию данных_](https://www.facebook.com/business/help/PrivacyBestPractices)
      * [_Условия использования инструментов для бизнеса_](https://www.facebook.com/legal/terms/businesstools)
    * Для разработчиков и специалистов по технологиям:
      * [_Сравнение вариантов настройки API Conversions_](https://www.facebook.com/business/help/CompareConversionsAPISetups)
      * [_О партнерских веб-интеграциях_](https://www.facebook.com/business/help/WebPartnerIntegrations)
      * [_Начало работы с API Conversions_](https://developers.facebook.com/docs/marketing-api/conversions-api/get-started)
      * [_Рекомендации по работе с API Conversions_](https://developers.facebook.com/docs/marketing-api/conversions-api/best-practices)
    * Для специалистов по измерению и аналитике:
      * [_О тестах прироста конверсий_](https://www.facebook.com/business/help/221353413010930)
  * **Разработайте план действий, который учитывает получение одобрений от нужных специалистов и требования выбранного вами способа настройки**. Внедрение API Conversions может занять от нескольких часов до нескольких недель или месяцев. Это зависит от целого ряда факторов, например от выбранного [_способа настройки_](https://www.facebook.com/business/help/CompareConversionsAPISetups) и от того, как быстро специалисты примут нужные решения. Вы можете включить в свой план все или только некоторые из приведенных ниже шагов.
    * Заручиться поддержкой нужных специалистов.
    * Привлечь к работе специалистов по техническим вопросам и партнеров по разработке системных решений (если необходимо).
    * Подготовиться к дополнительной проверке того, соблюдаете ли вы обязательства, предусмотренные законами о сборе и обработке данных; получить одобрение на передачу данных у юристов. Способы использования инструментов Meta для бизнеса и предоставления к ним доступа регулируются [_Условиями использования инструментов для бизнеса_](https://www.facebook.com/legal/terms/businesstools) и (в применимых случаях) [_Условиями использования платформы_](https://developers.facebook.com/terms).
    * Настроить Conversions API. Поскольку разные способы настройки требуют разного количества времени, рекомендуем [_сравнить варианты настройки API Conversions_](https://www.facebook.com/business/help/433493041367251?id=818859032317965).
    * Протестировать обмен данными. Следуйте [_видеоруководству по выполнению первого тестового вызова API_](https://developers.facebook.com/docs/marketing-api/conversions-api/using-the-api#video).
