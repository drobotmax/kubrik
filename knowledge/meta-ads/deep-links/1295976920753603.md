# Инструменты кастомизации креативов для разных мест размещения
Кастомизация креативов позволяет оптимизировать их для всех доступных мест размещения, включая ленты, истории и рекламу In-Stream в видео Reels.
Выполните кастомизацию рекламных креативов, чтобы ваши объявления соответствовали рекомендациям для каждого места размещения. Вы можете [изменить соотношение сторон креатива](https://www.facebook.com/help/376859690417674) для каждого отдельного места размещения, [редактировать креатив для определенных мест размещения](https://www.facebook.com/help/127128577862845) или [выполнить кастомизацию рекламы, созданной на основе существующей публикации](https://www.facebook.com/help/2883760805204712).
Для кастомизации рекламы можно использовать инструменты оформления, указанные ниже.
**Шаблоны**
  * **Преобразовать в видео** ![](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=1279541239478372&version=1766555586&transcode_extension=webp) (только для изображений) — превратить изображение в видео.


**Медиафайл**
  * **Изменить** ![](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=512255090451219&version=1766555586&transcode_extension=webp) — показывать другое изображение или видео.
  * **Кадрировать** ![](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=1094967267838813&version=1766555586&transcode_extension=webp) (или **Кадрировать и укоротить** для видео) — изменить соотношение сторон изображения или видео.
  * **Миниатюра** ![](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=525441728993860&version=1766555586&transcode_extension=webp) (только для видео) — выбрать изображение, которое будет показываться в начале и в конце видео.


**Текст**
  * **Субтитры** ![](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=576345960958059&version=1766555586&transcode_extension=webp) (только для видео) — сгенерировать или загрузить субтитры.
  * **Основной текст** ![](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=235219651849335&version=1766555586&transcode_extension=webp) — отредактировать или добавить текст над изображением или видео.
  * **Заголовок** ![](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=235219651849335&version=1766555586&transcode_extension=webp) — отредактировать или добавить текст под изображением или видео.
  * **Место назначения** ![](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=2132854233567332&version=1766555586&transcode_extension=webp) — отредактировать или добавить место назначения (на сайте или в приложении, если вы используете [оптимизацию для многоканальных конверсий](https://www.facebook.com/help/187685303269334)).


**Инструменты оформления**
  * **Опрос** ![](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=434157332202260&version=1766555586&transcode_extension=webp) (только для видео) — добавить вопрос для пользователей и два варианта ответа.
  * **Текстовый оверлей** ![](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=641325197666835&version=1766555586&transcode_extension=webp) (только для видео) — добавить текст на видео. Рамки помогут вам не размещать оверлеи в [свободном пространстве](https://www.facebook.com/help/980593475366490).
  * **Оверлей с логотипом** ![](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=1856001998082049&version=1766555586&transcode_extension=webp) (только для видео) — добавить в видео логотип.
  * **Цвет фона** ![](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=1491768751332957&version=1766555586&transcode_extension=webp) (только для историй и видео Reels) — выбрать цвет или добавить шестнадцатеричные значения, чтобы создать собственный оттенок.
