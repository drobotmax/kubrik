# Советы по повышению эффективности A/B-тестирования
С помощью [A/B-тестирования](https://www.facebook.com/help/1738164643098669) вы можете опробовать разные варианты рекламы, чтобы понять, какой из них больше понравится аудитории или принесет наилучшие результаты. 
Вы сможете посмотреть результаты A/B-тестирования в электронном письме или в Meta Ads Manager. Если у вас появятся вопросы о полученных результатах, ознакомьтесь с [этим руководством](https://www.facebook.com/help/1376548572415613?helpref=related). 
Иногда результаты показывают, что самый эффективный вариант рекламы не выявлен либо получено слишком мало показов. На этот случай мы предлагаем советы, которые помогут вам устранить проблемы с A/B-тестированием и оптимизировать будущие тесты.
В этой статье также приведены примеры дополнительных тестирований. Они подскажут вам, как повысить результативность рекламы или улучшить ее стратегию. 
## Устранение проблем с A/B-тестированием и повышение его эффективности 
Для начала изучите [рекомендации по A/B-тестированию](https://www.facebook.com/help/290009911394576) и [руководство по устранению проблем с результативностью рекламы](https://www.facebook.com/help/945693298836131). Если вы учли эти рекомендации, но вам не удалось устранить проблемы, сделайте следующее:
  * **Убедитесь, что ваша аудитория достаточно большая, чтобы избежать недостаточного количества показов**. Поскольку мы делим вашу аудиторию для проведения A/B-тестирования, то количество показов групп объявлений может быть недостаточным из-за маленьких аудиторий. Возможно, вам необходимо расширить аудиторию. 
  * **Увеличьте бюджет**. Маленький бюджет тоже может привести к небольшому количеству показов. Увеличив бюджет для A/B-тестирования, вы сможете охватить больше людей.
  * **При тестировании вариантов аудитории, места размещения или оптимизации показа используйте разные группы объявлений**. Если группы будут слишком похожи, мы не сможем точно определить самую эффективную из них. Например: [людей](https://www.facebook.com/business/help/283579896000936), которые уже совершали покупки на вашем сайте, в сравнении с группой людей, которых могут заинтересовать ваши товары (на основе таргетинга по интересам). 


## Проведение дополнительных тестов
Одно A/B-тестирование дает ценную информацию, но на ее основе можно создать дополнительные тесты, чтобы выработать эффективную стратегию их проведения или получить больше полезных данных. 
Ниже приведено несколько вариантов дополнительного тестирования, которые помогут улучшить рекламную стратегию. Это лишь рекомендации. Стратегии тестирования для разных рекламодателей и отраслей отличаются.
После тестирования аудиторий: 
  * **Проведите такое же тестирование для кампании с другой целью**. Реклама для аудитории, настроенной в тестировании, может показать другие результаты в кампании другого типа. Повторите тестирование с этой аудиторией для каждой нужной вам цели кампании. 
  * **Протестируйте разные рекламные креативы с аудиторией, которая показала наилучшие результаты**. Когда вы определите, какую аудиторию хотите использовать, узнайте, какие креативы эффективнее всего для ее охвата.
  * **Протестируйте разные места размещения с аудиторией, которая показала наилучшие результаты**. Узнайте, какие места размещения (например, ресурсы для мобильных устройств или для компьютеров) эффективнее всего для охвата нужной аудитории.


После тестирования креативов:
  * **Проведите такое же тестирование для кампании с другой целью**. Реклама с креативами, настроенными в тестировании, может показать другие результаты в кампании другого типа. Повторите тестирование с этими креативами для каждой нужной вам цели кампании. 
  * **Протестируйте разные элементы рекламных креативов**. Если вы протестировали один элемент креатива, например заголовок или текст, теперь можете протестировать другой. 
  * **Протестируйте разные креативы одного формата**. Если вы сравнивали видеорекламу с рекламой с одним изображением и победила видеореклама, попробуйте протестировать два разных видеообъявления. Так вы поймете не только, какой формат эффективен, но и что для него лучше всего работает.

После тестирования мест размещения:
  * **Проведите такое же тестирование для кампании с другой целью**. Реклама в местах размещения, настроенных в тестировании, может показать другие результаты в кампании другого типа. Повторите тестирование с этими местами размещения для каждой нужной вам цели кампании. 
  * **Протестируйте разные аудитории в самом результативном месте размещения**. Если вы выяснили, какие места размещения больше всего вам подходят, протестируйте разные аудитории для каждого из них, чтобы усовершенствовать свою стратегию. 

После тестирования оптимизации показа:
  * **Проведите тест с одинаковым окном конверсии**. После тестирования разных окон конверсии необходимо выделить событие для оптимизации. Для этого используйте победившее окно конверсии для каждой группы объявлений. 
  * **Проведите тестирование с одинаковым событием для оптимизации**. После тестирования разных событий для оптимизации необходимо выделить окно конверсии. Для этого используйте одинаковое событие для оптимизации для каждой группы объявлений.
