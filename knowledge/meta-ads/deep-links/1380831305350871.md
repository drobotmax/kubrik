# Рекомендации по настройке уровней списков заблокированных издателей
![](https://scontent-prg1-1.xx.fbcdn.net/v/t39.2365-6/30175777_958483094316933_3593890876511748096_n.png?stp=dst-webp&_nc_cat=104&ccb=1-7&_nc_sid=e280be&_nc_ohc=-yfgZtFMlgUQ7kNvwHqwrvy&_nc_oc=AdokFxkTAq_urWn3Pg-DNkj59FjbDpVU1QDUeEkuP13gdblFh-asjX9IGd3t2wjCmnw&_nc_zt=14&_nc_ht=scontent-prg1-1.xx&_nc_gid=MByp7MQLdpMOW9AYjbzgGw&_nc_ss=7a389&oh=00_AfxZ9aI-hFu-BtaCIqKsBThSnTKApRhe__SqsBxgYG7DLQ&oe=69E3B47D)
Вы можете настроить списки заблокированных издателей на уровне компании, аккаунта и кампании.
Количество записей в списке блокировки ограничено ([подробнее](https://www.facebook.com/help/176672796210945)). Лимит для кампании ниже, чем для аккаунта или компании. Если вы получили сообщение об ошибке, в котором говорится, что ваша реклама не будет показываться в некоторых местах размещения из-за превышения ограничения на число записей для кампании, попробуйте заблокировать некоторые объекты для аккаунта или компании.
Только у людей с полным доступом к бизнес-портфолио есть разрешение на применение списков заблокированных издателей на уровне компании. Подробнее о том, как [изменить чей-либо доступ или разрешения](https://www.facebook.com/business/help/299504287548592) либо [предоставить доступ к списку заблокированных издателей](https://www.facebook.com/business/help/240346943246169). 
  * **Примените списки заблокированных издателей для всей организации на уровне компании**. [Подробнее](https://www.facebook.com/help/427707251064766). 
Допустим, организация не хочет показывать свою рекламу на некоторых URL. В этом случае следует загрузить список заблокированных издателей с этими URL в бизнес-портфолио и применить его ко всем рекламным аккаунтам. Это поможет предотвратить ошибки, поскольку при создании рекламы URL блокируются автоматически. 
  * **Примените дополнительные списки заблокированных издателей для аккаунта**. [Подробнее](https://www.facebook.com/help/427707251064766). 
Допустим, у организации есть 2 бренда — у каждого свой рекламный аккаунт. Нужно заблокировать разные URL для каждого аккаунта. В этом случае следует загрузить 2 списка заблокированных издателей в бизнес-портфолио и применить по одному к каждому рекламному аккаунту.
  * **Примените дополнительные списки заблокированных издателей с корректировками для определенных рекламных кампаний.** [Подробнее.](https://www.facebook.com/help/775487392843817)
Списки заблокированных издателей для кампании не предназначены для регулярного использования. Допустим, организация хочет заблокировать определенные URL в рекламной кампании одного своего бренда. Также некоторые URL нужно блокировать не во всех странах. При создании рекламной кампании можно загрузить список заблокированных издателей в бизнес-портфолио, а затем применить его в Ads Manager.

## Будьте в курсе новостей от Meta for Business
Укажите электронный адрес, на который вы хотите получать новости о мероприятиях и продуктах, а также другую информацию от Meta for Business. 
Отправляя эту форму, вы соглашаетесь получать по электронной почте маркетинговые материалы от Meta, в том числе новости, информацию о мероприятиях, обновления и специальные предложения. Вы можете отказаться от рассылки в любой момент. Для этого нужно, например, нажать ссылку для отмены подписки в нашем электронном письме.
