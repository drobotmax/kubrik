# About Signals Gateway SDK

The Signals Gateway SDK allows businesses to manage data collection and sharing from their [Signals Gateway](https://www.facebook.com/business/help/1393493081525641) instance. By integrating the SDK into your app, you can take control of your data and make informed decisions about how it is used.
##  Create the Signals Gateway SDK 
To set up a new Signals Gateway SDK, follow these steps:
  1. **Access the data pipeline** : Log into your Signals Gateway and navigate to the **Data pipelines** section in the menu.
  2. **Create pipeline** : Click **Create Data pipeline**.
  3. **Choose SDK** : Select **Signals Gateway SDK** as your data source and choose a data destination.
  4. **Terms and conditions** : Acknowledge the **Terms and Conditions**. 
  5. Enter a name for your SDK, and click **Continue**.


##  Next steps 
  1. **Select event types** : Set your app category, and choose the events that will be part of your personalized instructions for your app developer to implement.
  2. **Configure iOS SDK** : Set up the Signals Gateway SDK for iOS by selecting the features and settings you want to enable. You can modify these configurations anytime, and updates will automatically apply to all active client accounts.
  3. **Share instructions** : Provide your developer with the setup instructions for the Signals Gateway SDK for iOS.
  4. **Connect data** : Follow on-screen instructions to connect your data to a destination in the pipeline. Admin access to a[ Meta Business Account](https://www.facebook.com/business/help/1710077379203657) is required to connect to Meta.


Implement the Signals Gateway SDK using the [developer documentation for Signals Gateway SDK](https://developers.facebook.com/docs/marketing-api/gateway-products/signals-gateway).
