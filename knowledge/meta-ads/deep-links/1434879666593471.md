# Добавление вопроса типа "Локатор точек" в рекламу для сбора лидов с моментальной формой
Создавая [рекламу для сбора лидов с моментальной формы](https://www.facebook.com/business/help/761812391313386), вы можете использовать функцию **Локатор точек** , чтобы узнать, где находятся потенциальные клиенты. Предположим, у вас есть несколько фитнес-клубов одной сети. Вы можете добавить в моментальную форму вопрос о предпочитаемой точке, чтобы узнать, какой фитнес-клуб потенциальный клиент посетит с наибольшей вероятностью. Это поможет вам работать с лидами и находить клиентов, которые с большей вероятностью пойдут в фитнес-клуб в определенном районе.
Чтобы использовать эту функцию, нужны Страницы для нескольких точек. Вы можете [настроить точки компании](https://www.facebook.com/business/help/1664726753843273) в бизнес-портфолио, а затем добавить их в рекламу для сбора лидов. 
**Примечание**. Вопрос типа "Локатор точек" сейчас нельзя добавить на компьютере. Реклама для сбора лидов с вопросом типа "Локатор точек" будет показываться только на мобильных устройствах.
## Чтобы добавить вопрос типа "Локатор точек" в моментальную форму:
  1. [Создайте моментальную форму](https://www.facebook.com/business/help/375478503258484) для рекламы для сбора лидов.
  2. В разделе **Вопросы** нажмите **Добавить вопрос**.
  3. Выберите **Локатор точек**.
  4. Введите вопрос, связанный с вашими точками.
  5. Нажмите **Варианты источника точек** и выберите **Точки с Facebook**. Так вы добавите все точки, настроенные в бизнес-портфолио.
  6. Завершите создание моментальной формы и рекламы для сбора лидов. 


Когда реклама будет одобрена, все люди будут видеть в ее моментальной форме вопрос типа "Локатор точек". Они смогут просмотреть все точки, которые вы настроили в бизнес-портфолио, и отфильтровать их по индексу.
