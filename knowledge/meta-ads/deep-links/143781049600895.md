# Настройка каталога для рекламы ассортимента автосалона
Чтобы показывать [рекламу ассортимента автосалона](https://www.facebook.com/business/help/1510143265745613), нужно создать каталог с информацией обо всех транспортных средствах, которые вы хотите продвигать. Такая реклама использует ваш пиксель Meta или SDK, чтобы показывать людям наиболее актуальные предложения с учетом их интересов и поведения.
**Примечание**. Если вы разработчик, ознакомьтесь с документацией по API на сайте [Meta for Developers](https://developers.facebook.com/docs/marketing-api/dynamic-ads-auto/auto-catalog).
##  Шаг 1. Создание каталога
[Подробнее о создании каталога в Commerce Manager](https://www.facebook.com/business/help/1275400645914358). Выберите тип каталога **Авто** и нажмите **Транспортные средства**.
Вы также можете настроить каталог при помощи сторонних поставщиков данных об ассортименте, таких как [DealerOn или Naked Lime](https://www.facebook.com/help/2671106459788829).
##  Шаг 2. Добавление ассортимента автосалона
Создав каталог в Commerce Manager, [настройте фид данных](https://www.facebook.com/business/help/1898524300466211) со всеми транспортными средствами, которые хотите продвигать. Вы можете указать все автомобили в одном фиде или создать разные фиды для определенных групп автомобилей (например, для всех внедорожников определенной марки). Убедитесь, что добавили все [обязательные поля для рекламы ассортимента автосалона](https://developers.facebook.com/docs/marketing-api/auto-ads/reference#vehicle) (Meta for Developers), и выполните все требования, чтобы обеспечить правильную загрузку данных о транспортных средствах.
Создав фид данных о транспортных средствах, [загрузите его в каталог](https://www.facebook.com/business/help/125074381480892). Вы можете загрузить фид один раз или составить график загрузок (ежечасно, ежедневно или еженедельно), чтобы информация в каталоге всегда была актуальной.
##  Шаг 3. Создание групп товаров
Группа товаров — это набор позиций каталога, определяющий, какие транспортные средства будут показываться в рекламе. Например, можно создать группу для всех седанов или для всех внедорожников в каталоге. Подробнее о том, [как создать группу товаров](https://www.facebook.com/help/620275848114281).
##  Шаг 4. Подключение пикселя или SDK для мобильных устройств
Чтобы показывать рекламу ассортимента автосалона, вы должны подключить к каталогу пиксель Meta или SDK приложения. Так вы сможете показывать транспортные средства из каталога людям, которые уже искали их или взаимодействовали с вашим ассортиментом. Подробнее о том, как [установить пиксель или SDK для рекламы ассортимента автосалона](https://www.facebook.com/business/help/1989760861301766) и [подключить их к каталогу](https://www.facebook.com/business/help/1044262445604547).
