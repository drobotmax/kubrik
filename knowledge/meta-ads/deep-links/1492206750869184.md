# Добавление оверлея в туристическую рекламу
**Примечание**. Это инструкция предназначена для туристической рекламы. Подробнее о том, [как добавить оверлей в динамическую рекламу](https://www.facebook.com/business/help/517601898571657?helpref=faq_content). 
## Настройка оверлея
В оверлее может быть информация о цене или скидке. 
![](https://scontent-prg1-1.xx.fbcdn.net/v/t39.2365-6/22810575_226204577915367_4767396286762582016_n.jpg?stp=dst-webp&_nc_cat=111&ccb=1-7&_nc_sid=e280be&_nc_ohc=NuI-pSNXY3QQ7kNvwGWQzjV&_nc_oc=AdpDz3SJnexWKMljkcD_-z08a6-A6W1Neybg0rDEYjCl-MigK6F-FpgGL-D8kRB3A3M&_nc_zt=14&_nc_ht=scontent-prg1-1.xx&_nc_gid=u5jThPoWpF0qn3qJgBQS5Q&_nc_ss=7a389&oh=00_AfznMIeMP6sEciEM4TeVdPXelH-Ahzyk3KXulcPfEbeqrQ&oe=69E3AB72)
_Пример туристической рекламы с перечеркнутой ценой на оверлее_
## Параметры оверлеев
Добавляя оверлей в туристическую рекламу, вы можете настроить его внешний вид. В зависимости от типа создаваемой туристической рекламы выберите один из следующих вариантов оверлея:
  * [Реклама гостиниц](https://www.facebook.com/help/1320285461344901). Вы можете указать **цену** , **перечеркнутую цену** или **скидку в процентах**.
  * [Реклама авиарейсов](https://www.facebook.com/help/306561846428189) и [туристических направлений](https://www.facebook.com/help/821596348016209). Поддерживается только оверлей с **ценой**. 


Для каждого типа оверлея можно выбрать варианты индивидуальной настройки:
  * **Форма**. Вы можете показывать информацию поверх изображения рекламируемого товара в прямоугольнике, треугольнике или круге.
  * **Шрифт**. Выберите шрифт текста в оверлее.
  * **Цвет**. Выберите цвет оверлея.
  * **Положение**. Укажите, где будет размещен оверлей в объявлении.


## Начало работы
Прежде чем создать оверлей, связанный с ценами, убедитесь, что в фиде данных каталога указана правильная информация.
**Для любой туристической рекламы**
Если цена по какой-либо причине недействительна (например, цена номера в гостинице со скидкой не указана или не ниже обычной, цена для туристического направления не указана и т. д.), оверлей не будет отображаться. Показ рекламы при этом не прекратится.
**Для рекламы авиарейсов**
Вы можете использовать только цену из фида (например, нельзя использовать динамическое ценообразование или цены из глобальной дистрибьюторской системы).
## Добавление оверлея
  1. Откройте инструмент для создания объявлений и начните создавать рекламу [гостиниц](https://www.facebook.com/help/1320285461344901), [авиарейсов](https://www.facebook.com/help/306561846428189) или [туристических направлений](https://www.facebook.com/help/821596348016209). 
  2. На уровне объявления прокрутите до раздела **Показывайте больше информации из каталога**.
  3. Установите флажок **Добавить оверлей к изображениям**.
  4. Раскройте меню и выберите тип оверлея: 
     * **Цена**. 
     * **Перечеркнутая цена**. 
     * **Скидка в процентах**. 
**Примечание**. Оверлей **Перечеркнутая цена** содержит две цены. При настройке такого оверлея укажите **базовую цену** (которая будет зачеркнута) и **цену со скидкой**.
![](https://scontent-prg1-1.xx.fbcdn.net/v/t39.2365-6/22509439_377943319305104_5384748387604103168_n.jpg?stp=dst-webp&_nc_cat=104&ccb=1-7&_nc_sid=e280be&_nc_ohc=UNTmFTipcU0Q7kNvwG9DUBO&_nc_oc=AdquQyEUsBzwUzHEv3bk7QY2mg27NjyQ7ucRYksG-mWXC2FXSmSXuuChK0r-RSSJ9RE&_nc_zt=14&_nc_ht=scontent-prg1-1.xx&_nc_gid=u5jThPoWpF0qn3qJgBQS5Q&_nc_ss=7a389&oh=00_AfzIv9A7XTvVVXAXXw6m6bmQqyHYH9zy1DAP1IDr5eFQPQ&oe=69E3974A)
  5. Чтобы изменить оверлей, нажмите **Редактировать внешний вид оверлея**. Вы можете выбрать форму, шрифт, цвет и положение оверлея. 
![](https://scontent-prg1-1.xx.fbcdn.net/v/t39.2365-6/22509230_124327421616519_3521772345477824512_n.png?stp=dst-webp&_nc_cat=105&ccb=1-7&_nc_sid=e280be&_nc_ohc=lOKP6DmyaFgQ7kNvwHgY9QI&_nc_oc=AdpOSmKa23dBlbolzdFoLmhZWYU4LRO2sAP_XnRRRHUClnGYwyspCXwk5uQvdzV6njY&_nc_zt=14&_nc_ht=scontent-prg1-1.xx&_nc_gid=u5jThPoWpF0qn3qJgBQS5Q&_nc_ss=7a389&oh=00_AfyJndya_7TVJY8fw-zAXSTXTEMCceJV0H26RXQaK28uvw&oe=69E38F76)
  6. По завершении нажмите **Сохранить**. Оверлей готов. Теперь вы можете закончить создание динамической рекламы.
