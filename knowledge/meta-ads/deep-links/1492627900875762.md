# Как найти ID рекламного аккаунта на Facebook?
Когда вы запускаете рекламу на Facebook, вашим рекламным аккаунтам присваиваются уникальные ID. Они могут понадобиться, если вы хотите [добавить рекламный аккаунт в бизнес-портфолио](https://www.facebook.com/business/help/915885887059947). ID рекламных аккаунтов можно найти:
  * В раскрывающемся меню аккаунтов в Meta Ads Manager.
  * В адресной строке браузера в Ads Manager.


## Как найти ID рекламного аккаунта в раскрывающемся меню аккаунтов в Meta Ads Manager?
Чтобы найти ID рекламного аккаунта в раскрывающемся меню аккаунтов в Ads Manager:
  1. Откройте [Meta Ads Manager](https://www.facebook.com/adsmanager/manage).
  2. Название и ID рекламного аккаунта показываются в раскрывающемся меню "Объект компании" в верхней части страницы. Раскройте меню, чтобы просмотреть ID других рекламных аккаунтов.


## Как найти ID рекламного аккаунта в адресной строке в Meta Ads Manager?
Чтобы найти ID рекламного аккаунта в адресной строке в Ads Manager:
  1. Откройте [Meta Ads Manager](https://www.facebook.com/adsmanager/manage).
  2. Найдите ID вашего рекламного аккаунта в адресной строке браузера. Он указан после фрагмента **act=** в URL.
