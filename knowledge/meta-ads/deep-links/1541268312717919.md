# Просмотр информации о событиях сервера в Meta Events Manager
![](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=438223955063449&version=1666716831&transcode_extension=webp)
Эти разбивки доступны для событий сайта, отправляемых через Conversions API, но могут быть недоступны для других типов событий от Conversions API и пикселя Meta.
В Meta Events Manager можно посмотреть подробные разбивки для каждого события сайта, отправляемого с помощью [Conversions API](https://www.facebook.com/business/help/2041148702652965).
Эти сведения позволяют улучшать работу Conversions API, помогая вам:
  * Отслеживать объем событий и убедиться, что вы отправляете достаточно избыточных событий (вкладка **Обзор события**).
  * Расширить охват событий (вкладка **Охват событий**). 
  * Повысить качество сопоставления событий (вкладка **Качество событий**).
  * Улучшить дедупликацию (вкладка **Дедупликация событий**).
  * Отправлять события в Meta в режиме реального времени (вкладка **Регулярность обновления данных**).


Ознакомьтесь с [рекомендациями по работе с Conversions API](https://www.facebook.com/business/help/308855623839366).
## Просмотр информации о событии сервера
Чтобы посмотреть информацию о событии сервера в Events Manager:
  1. Откройте [Events Manager](https://business.facebook.com/events_manager2/list).
  2. Выберите название и ID нужных данных.
  3. Найдите нужное событие и нажмите "Подробнее".
  4. Откройте одну из приведенных ниже вкладок, чтобы посмотреть подробную информацию о событии.


![](https://scontent-prg1-1.xx.fbcdn.net/v/t39.8562-6/146597050_455534749152435_1802453867954835002_n.svg?_nc_cat=1&ccb=1-7&_nc_sid=f537c7&_nc_ohc=aXVV8xvJ_GcQ7kNvwHlDc6r&_nc_oc=AdpxRCyIaZgWtCiyYVpouqc9iu5YqUDJMwrDTBJEFNEJZUEPEvWvXOLbI_2mhUNO9Is&_nc_zt=14&_nc_ht=scontent-prg1-1.xx&_nc_gid=UtIn1MUVoQZ6DrVJx1cT8A&_nc_ss=7a389&oh=00_Afy6y9N8V6HfCtaWG09I92e9ZwkbT-U3BRufJwb6keRX-g&oe=69CF18D8)
### Обзор события
На этой вкладке показаны полученные события. Это метрика, которая учитывает события, отправляемые через пиксель Meta и Conversions API.
Вам также доступна визуальная линия тренда для зарегистрированных дополнительных конверсий, которая показывает влияние качества данных на конверсии. По этой линии тренда можно понять, как изменение качества данных со временем влияет на коэффициенты конверсии.
**Полученные события**. Общее число событий от Conversions API или вашего пикселя перед обработкой. Если вы используете оба источника, это значение суммируется без дедупликации.
**Проверка избыточности**. Событий из Conversions API должно быть не меньше, чем из пикселя Meta. Убедитесь, что события дедуплицированы. Если событий из пикселя больше, скорректируйте конфигурацию Conversions API.
![](https://scontent-prg1-1.xx.fbcdn.net/v/t39.8562-6/146597050_455534749152435_1802453867954835002_n.svg?_nc_cat=1&ccb=1-7&_nc_sid=f537c7&_nc_ohc=aXVV8xvJ_GcQ7kNvwHlDc6r&_nc_oc=AdpxRCyIaZgWtCiyYVpouqc9iu5YqUDJMwrDTBJEFNEJZUEPEvWvXOLbI_2mhUNO9Is&_nc_zt=14&_nc_ht=scontent-prg1-1.xx&_nc_gid=UtIn1MUVoQZ6DrVJx1cT8A&_nc_ss=7a389&oh=00_Afy6y9N8V6HfCtaWG09I92e9ZwkbT-U3BRufJwb6keRX-g&oe=69CF18D8)
### Охват событий
Охват событий соответствует сумме событий браузера и сервера, отправленных в Meta. На этой вкладке показана процентная доля событий, полученных от Conversions API, по сравнению с уникальными событиями браузера от пикселя Meta. Вы получите персонализированные рекомендации по улучшению охвата событий, которыми можно поделиться со своим разработчиком для лучшего измерения конверсий.
Оптимальный коэффициент охвата событий от Conversions API и пикселя Meta для рекламодателей — 75 %. Такой показатель обеспечить точность отчетности и поможет добиться целей по результативности.
![](https://scontent-prg1-1.xx.fbcdn.net/v/t39.8562-6/146597050_455534749152435_1802453867954835002_n.svg?_nc_cat=1&ccb=1-7&_nc_sid=f537c7&_nc_ohc=aXVV8xvJ_GcQ7kNvwHlDc6r&_nc_oc=AdpxRCyIaZgWtCiyYVpouqc9iu5YqUDJMwrDTBJEFNEJZUEPEvWvXOLbI_2mhUNO9Is&_nc_zt=14&_nc_ht=scontent-prg1-1.xx&_nc_gid=UtIn1MUVoQZ6DrVJx1cT8A&_nc_ss=7a389&oh=00_Afy6y9N8V6HfCtaWG09I92e9ZwkbT-U3BRufJwb6keRX-g&oe=69CF18D8)
### Сопоставление событий
На этой вкладке оценивается точность сопоставления событий с параметрами информации о клиенте, содержащимися в событии сервера.[дополнительных зарегистрированных конверсий](https://www.facebook.com/business/help/453888373437795). 
**Информация о клиенте**. Просмотрите процентную долю событий с полными данными клиента. Как правило, чем больше данных, тем лучше сопоставление. 
  * **Советы по улучшению**. Отправка параметра информации о клиенте не гарантирует, что событие будет сопоставлено с аккаунтом Meta. Например, один такой параметр может содержать недостаточно информации для сопоставления события с аккаунтом Meta или событие может быть связано с клиентом, у которого нет аккаунта Meta. В результате у вас может быть высокий процент отправленных событий, но низкое качество их сопоставления. Следуйте рекомендациям, приведенным в окне сопоставления событий или нашем [руководстве](https://www.facebook.com/business/help/308855623839366).


![](https://scontent-prg1-1.xx.fbcdn.net/v/t39.8562-6/146597050_455534749152435_1802453867954835002_n.svg?_nc_cat=1&ccb=1-7&_nc_sid=f537c7&_nc_ohc=aXVV8xvJ_GcQ7kNvwHlDc6r&_nc_oc=AdpxRCyIaZgWtCiyYVpouqc9iu5YqUDJMwrDTBJEFNEJZUEPEvWvXOLbI_2mhUNO9Is&_nc_zt=14&_nc_ht=scontent-prg1-1.xx&_nc_gid=UtIn1MUVoQZ6DrVJx1cT8A&_nc_ss=7a389&oh=00_Afy6y9N8V6HfCtaWG09I92e9ZwkbT-U3BRufJwb6keRX-g&oe=69CF18D8)
### Дедупликация событий
На этой вкладке можно узнать, необходимо ли улучшить дедупликацию событий. [Подробнее о дедупликации](https://www.facebook.com/business/help/823677331451951).
В таблице "Ключи дедупликации" представлен процент событий от пикселя и Conversions API, полученных с каждым ключом дедупликации. Рекомендуем отправлять ключи дедупликации для всех событий. Чем выше значение в этом разделе, тем лучше.
  * В разделе "Совпадение" указан процент событий с заданным ключом дедупликации, полученных и Conversions API, и пикселем. Чем выше процент, тем эффективнее мы определяем и исключаем дублированные события. Это позволяет получить более полное представление о ваших событиях.


![](https://scontent-prg1-1.xx.fbcdn.net/v/t39.8562-6/146597050_455534749152435_1802453867954835002_n.svg?_nc_cat=1&ccb=1-7&_nc_sid=f537c7&_nc_ohc=aXVV8xvJ_GcQ7kNvwHlDc6r&_nc_oc=AdpxRCyIaZgWtCiyYVpouqc9iu5YqUDJMwrDTBJEFNEJZUEPEvWvXOLbI_2mhUNO9Is&_nc_zt=14&_nc_ht=scontent-prg1-1.xx&_nc_gid=UtIn1MUVoQZ6DrVJx1cT8A&_nc_ss=7a389&oh=00_Afy6y9N8V6HfCtaWG09I92e9ZwkbT-U3BRufJwb6keRX-g&oe=69CF18D8)
