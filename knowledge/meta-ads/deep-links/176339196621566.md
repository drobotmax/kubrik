# Рекомендации для стратегии "Цель по цене за результат"
Ниже вы найдете рекомендации для стратегии [Цель по цене за результат](https://www.facebook.com/help/272336376749096). 
## В каких случаях стоит использовать стратегию "Цель по цене за результат"?
  * **Если средняя цена за событие оптимизации — это важная метрика для оценки результативности, и вы хотите выполнять оптимизацию с учетом этой цены**. Так вы сможете значительно увеличить количество конверсий, контролируя значение цены. Допустим, вы хотите развивать свой бизнес и получить как можно больше подписок или покупок, но при этом поддерживать единую цену за событие оптимизации. В таком случае стоит использовать стратегию "Цель по цене за результат". Когда группа объявлений завершит [этап обучения](https://www.facebook.com/business/help/112167992830700), мы будем стараться не превышать установленную вами цель по цене за результат, однако не можем гарантировать точную сумму затрат. 
  * **Если вам известна цена за действие**. Когда вы задаете цель по цене за результат, то можете установить цену за событие оптимизации. Это позволяет нам выполнять оптимизацию с учетом этой цены и получать максимально возможное при вашем бюджете число конверсий.


## Как настроить целевое значение?
  * **Сообщите нам цену за событие оптимизации важных для вас результатов**. 
  * **Если вам неизвестна цена за событие оптимизации, попробуйте определить среднюю CPA с помощью стратегии "Максимальное количество"**. Для этого показывайте группу объявлений без цели по цене за результат и посмотрите значение цены за событие оптимизации в отчетности. Если этап обучения группы объявлений завершен, используйте только указанную цену за событие оптимизации.
  * **Если вы используете стратегию "Предельная ставка", возможно, ваша** [**контрольная ставка**](https://www.facebook.com/business/help/586282251886816) **не подходит для использования в качестве контрольной цены**. Контрольная ставка — это максимальная сумма ставки, которую мы можем использовать в каждом аукционе, а цель по цене за результат — это желаемый максимальный показатель CPA. Другими словами, контрольная ставка не связана напрямую с вашими затратами, в отличие от цели по цене за результат.


## Что следует учесть при масштабировании бюджетов в рамках стратегии "Цель по цене за результат"?
  * **Цель по цене за результат позволяет сократить расходы, поскольку система в первую очередь ищет самые дешевые результаты**. 
  * **Чтобы улучшить результативность и получить максимальное число конверсий с помощью стратегии "Цель по цене за результат", рекомендуем использовать весь бюджет**. Так наша система сможет получить нужные данные в ходе обучения и выполнить оптимизацию в соответствии с вашими целями. Если вы хотите получать больше конверсий, увеличьте бюджет и поддерживайте среднюю цену на уровне цели по цене за результат. 
  * **Если сумма затрат достигла уровня установленной цены, показ может замедлиться. Причина в том, что мы выполняем оптимизацию для получения большего числа результатов в пределах цели по цене за результат, поскольку уже использовали основные более дешевые возможности для показа на рынке**. Если вы повысите бюджет, а цель по цене за результат останется прежней, вы вряд ли получите больше конверсий и сможете полностью израсходовать бюджет. Если изменить цель по цене за результат нельзя, попробуйте обновить креатив или расширить аудиторию. Это увеличит количество показов.
