# Сброс потраченной суммы аккаунта
![](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=438223955063449&version=1666716831&transcode_extension=webp)
Если в вашем рекламном аккаунте настроены [доступные средства](https://www.facebook.com/business/help/350519228479855), вы не сможете установить для него лимит затрат.
![](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=438223955063449&version=1666716831&transcode_extension=webp)
Описанные здесь действия можно выполнить только на компьютере.
По достижении лимита затрат показ ваших объявлений будет приостановлен, а средства рекламного аккаунта перестанут расходоваться. Чтобы сразу возобновить показ рекламы, вы можете сбросить потраченную сумму в разделе "Настройки оплаты". Ваш лимит затрат для аккаунта не изменится. Сумма затрат начнет накапливаться с нуля.
**Примечание.** Если лимит затрат вашего аккаунта составляет 0 долл. США, вы не сможете сбросить его.
## Прежде чем начать
Вы должны быть администратором рекламного аккаунта.
## Сброс потраченной суммы рекламного аккаунта 
Чтобы сбросить потраченную сумму рекламного аккаунта:
  1. Перейдите в [**настройки оплаты**](https://www.facebook.com/ads/manager/account_settings/account_billing).
  2. В разделе **Лимит затрат для аккаунта** нажмите **Параметры** ![](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=250425836038370&version=1773072306&transcode_extension=webp).
  3. Нажмите **Удалить** или **Изменить**.


Сумма будет сброшена до 0 долл. США. Изменения вступят в силу примерно через 15 минут. После этого показ вашей рекламы продолжится.
У вас может быть возможность настроить автоматический сброс лимита затрат для аккаунта 1-го числа каждого месяца, а также лимита затрат для аккаунта за всё время, который сбрасывается вручную. 
Лимит затрат для аккаунта — это максимальная сумма денег, которую ваш рекламный аккаунт может потратить на все кампании за всё время. Этот лимит не сбрасывается автоматически по истечении определенного периода времени, например в конце месяца.
