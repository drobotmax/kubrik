# Как создать фид данных для каталога с помощью шаблона?
Фид данных — это файл, который можно использовать для массовой загрузки товаров в каталог Commerce Manager. Чтобы товары показывались в рекламе и магазине, фид должен содержать обязательные данные о позициях, например названия, описания, изображения и цены. Чтобы создать фид, скачайте из Commerce Manager шаблон и заполните его.
## Прежде чем начать
Вам потребуется **полный доступ** к каталогу. [Подробнее о разрешениях для каталога](https://www.facebook.com/business/help/1953352334878186).
## Скачивание шаблона фида данных для каталога
Чтобы скачать шаблон фида:
  1. Перейдите в [Commerce Manager](https://www.facebook.com/commerce_manager) и выберите свой каталог.
  2. Откройте вкладку **Каталог** и нажмите **Товары** или **Источники данных**.
  3. Нажмите **Добавить товары** и выберите **Фид данных**.
  4. Нажмите **Скачать шаблон**.
  5. Выберите шаблон.
     * Чтобы скачать стандартный шаблон, выберите формат файла в пункте **Стандартный шаблон** : Excel (**.xlsx**) или CSV (**.csv**).
     * Чтобы создать пользовательский шаблон с полями для определенных категорий, нажмите **Пользовательский шаблон**. Выберите одну или несколько категорий товаров. Когда всё будет готово, нажмите **Скачать шаблон XLSX** или раскройте меню и нажмите **Скачать шаблон CSV**.


**Примечание**. Для каталогов, предназначенных для туристической рекламы или продвижения ассортимента автосалона, доступны только стандартные шаблоны фидов в формате CSV или XML.
Вы скачали шаблон фида данных. Теперь заполните его данными о товарах.
## Заполнение шаблона фида данных
Чтобы заполнить шаблон фида данными о товарах:
  1. Откройте скачанный файл XLSX или CSV в программе для работы с электронными таблицами, например Microsoft Excel. Вы также можете импортировать его в Google Таблицы. Для этого откройте Google Таблицы, нажмите "Файл", а затем — "Импортировать" (см. инструкции [здесь](https://l.facebook.com/l.php?u=https%3A%2F%2Fsupport.google.com%2Fdocs%2Fanswer%2F9331167%232.4&h=AT4MVXJ5LtUeSrhFJBHOkAy8bOGdh7AWNIy7zfHvPOWAekr28iPgsULTXi1xF0k7FgOIDBamgCzmhx40a5h4PtYmlrEeJ1WJx5g6Z6-misJ-zoN4ST_FZBu3gZrjY2MFFasv60JKo8caZg)).
**Примечание**. Если вы используете Google Таблицы, ваш фид должен находиться на первой вкладке, поскольку остальные вкладки игнорируются. Используйте личный аккаунт Google, а не аккаунт Google Workspace. В вашей организации может быть запрещено предоставлять доступ к файлам внешним пользователям. Предоставьте доступ к ссылке **всем, у кого она есть** (не **ограниченный доступ**), чтобы разрешить обмен файлами с Meta.
  2. Откройте [требования к фиду данных](https://www.facebook.com/business/help/120325381656392) в новой вкладке или окне, чтобы сверяться с ними при заполнении шаблона.
  3. Просмотрите данные в шаблоне. Первая строка (строка заголовка) содержит названия столбцов, представляющие собой свойства товаров. Не меняйте эти названия. Они должны быть указаны так, как описано в требованиях (на американском варианте английского языка). В каждой из строк ниже указывается информация об одном товаре, при этом все обязательные поля должны быть заполнены. Необязательные поля можно оставить или удалить.
  4. Удалите из шаблона строки с примерами и добавьте вместо них настоящие данные обо всех свойствах товаров. Внимательно читайте требования, чтобы избежать ошибок. Некоторые поля поддерживают свободный ввод текста, тогда как в других можно указывать только определенные значения. Ваш фид будет выглядеть примерно так:   
| id  | title  | description  | availability  | condition  | price  | link  | image_link  | brand  |  
| --- | --- | --- | --- | --- | --- | --- | --- | --- |  
| 1234  | Полосатый ковер  | Черно-белый…  | in stock  | new  | 24.99 USD  | http://www…  | http://www…  | Jasper's Market  |  
| 5678  | Уютный плед  | Для прохладных вечеров…  | in stock  | new  | 14.99 USD  | http://www…  | http://www…  | Jasper's Market  |  
| 9123  | Декоративная подушка  | Красный бархат…  | in stock  | new  | 9.99 USD  | http://www…  | http://www…  | Jasper's Market  |  
  5. Чтобы добавить варианты товара, например разные цвета и размеры, создайте отдельную строку для каждого варианта и заполните приведенные ниже поля.
     * Укажите в поле **item_group_id** одинаковый ID группы для каждого варианта одного товара, чтобы обозначить, что они входят в одну группу.
     * Добавьте любые свойства, указывающие на отличия вариантов, например **color** (цвет), **size** (размер), **material** (материал), **pattern** (узор), **gender** (пол) или **additional_variant_attribute** (дополнительное свойство варианта). Укажите те же свойства для каждого варианта в группе.
     * Подробнее о том, [как добавлять варианты](https://www.facebook.com/business/help/2256580051262113?id=725943027795860#Feed).
  6. Введите всю необходимые данные о товарах, а затем сохраните файл.


Вы создали файл фида данных. Теперь его можно [загрузить в каталог](https://www.facebook.com/business/help/125074381480892).
