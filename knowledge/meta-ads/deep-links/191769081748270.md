# Охват широкой аудитории с помощью рекламы ассортимента автосалона для привлечения клиентов
Реклама ассортимента автосалона для привлечения клиентов позволяет охватывать людей, которые проявили интерес к вашим или похожим предложениям. Вы сможете охватить этих людей, даже если они ещё не посещали ваш сайт или приложение.
Такая реклама задействует аудитории для ретаргетинга. Когда вы таргетируете рекламу на широкую аудиторию, люди из этой аудитории автоматически видят наиболее актуальные позиции вашего каталога.
## Требования
  * **Пиксель Meta**. Интегрируйте на свой сайт [пиксель](https://www.facebook.com/business/help/1989760861301766) и используйте стандартные события, чтобы отслеживать ID товаров из вашего каталога, которые люди просматривают, добавляют в корзину и покупают.
  * **Каталог**. Настройте каталог с фидом транспортных средств, чтобы использовать рекламу ассортимента автосалона. Если у вас нет такого каталога, узнайте, [как его создать](https://www.facebook.com/business/help/143781049600895). 


## Настройка
  1. Создайте шаблон для рекламы ассортимента автосалона. 
  2. На этапе выбора **аудитории** нажмите **Определите широкую аудиторию, и мы выберем из нее людей, которые будут видеть ваши товары**.
  3. Задайте параметры таргетинга, например местоположение, возраст и пол.
  4. Выберите **Показать расширенные настройки** , чтобы уточнить аудиторию и исключить из нее людей, которые с меньшей вероятностью выполнят нужные вам действия. Доступные варианты:
     * "Без исключений" — люди не будут исключаться из аудитории на основе информации о взаимодействии с вашими товарами.
     * "Исключить людей, которые совершили покупку" — в течение указанного количества дней ваша реклама не будет демонстрироваться людям, которые приобрели товары из вашей группы.
     * "Создать индивидуально настроенное исключение" — люди будут исключаться из аудитории на основе заданных вами правил.


Примечание. Опытные рекламодатели могут создавать аудитории с помощью API. Узнать больше можно [на сайте Meta for Developers](https://developers.facebook.com/docs/marketing-api/dynamic-ads-auto/audience-management).
