# Настройка рекламы ассортимента автосалона
Реклама ассортимента автосалона позволяет автоматически показывать рекламу транспортных средств целевой аудитории, которая использует разные устройства. Такая реклама может появляться на Facebook или в Instagram в зависимости от выбранных мест размещения.
Если вы разработчик, ознакомьтесь с подробной информацией и образцами файлов на сайте [Meta for Developers](https://developers.facebook.com/docs/marketing-api/dynamic-ads-auto).
Автосалонам, у которых есть Страница Facebook, рекомендуем [настроить рекламу ассортимента автосалона со Страницы или в Meta Business Suite](https://www.facebook.com/business/help/702745500398966).
В остальных случаях выполните приведенные ниже инструкции.

## Настройка рекламы ассортимента автосалона
Настройка рекламы ассортимента автосалона состоит из 4 этапов.
  1. [Создайте каталог](https://www.facebook.com/help/143781049600895). Каталог содержит информацию о всех товарах, которые вы хотите рекламировать. Внутри каталога вы можете создавать группы товаров, чтобы управлять тем, какие типы товаров будут показываться в рекламе. Например, вы можете создать каталог транспортных средств и добавить в него весь свой ассортимент автомобилей. Затем создайте группы транспортных средств для разных видов автомобилей, чтобы рекламировать отдельно грузовики или, например фургоны. Эти группы можно создать на основе разных параметров. 
  2. [Установите пиксель Meta и/или SDK для мобильных устройств.](https://www.facebook.com/help/1989760861301766) Пиксель Meta — это аналитический инструмент, помогающий понять, какие действия люди совершают на вашем сайте. Facebook SDK для мобильных устройств позволяет передавать Facebook данные о событиях в вашем приложении. С помощью этих данных вы сможете точно отслеживать и измерять действия, которые люди выполняют в вашем приложении. Это шаг для рекламы ассортимента автосалона, направленной на **привлечение покупателей на сайт**. Если вы хотите, чтобы покупатели переходили в [место назначения на Facebook](https://www.facebook.com/help/346345493170863), то можете пропустить шаги 2 и 3.
  3. [Подключите пиксель к каталогу](https://www.facebook.com/help/1044262445604547). Этот шаг позволит вам показывать товары из каталога людям, которые уже искали их или взаимодействовали с ними.
  4. [Создайте шаблон рекламы](https://www.facebook.com/help/887605908064816). На завершающем этапе необходимо создать в Meta Ads Manager рекламу ассортимента автосалона: указать параметры целевой аудитории, выбрать форматы креативов и задать другие настройки.
