# Настройка параметров отслеживания в рекламе для сбора лидов с моментальной формой
Вы можете создать [рекламу для сбора лидов с моментальной формой](https://www.facebook.com/business/help/761812391313386) с [параметрами отслеживания](https://www.facebook.com/business/help/357992214682464), чтобы улучшить измерение и отслеживание для лидов. Если вы собираете лиды из нескольких источников, это поможет вам определить, какие маркетинговые каналы наиболее эффективны для вашей компании.
Вы можете просмотреть параметры отслеживания в информации о лидах. Люди, отправляющие форму, их не увидят. Параметры отслеживания показываются в файле, скачанном из [Meta Ads Manager](https://www.facebook.com/business/help/734933888443065) или [Meta Business Suite](https://www.facebook.com/business/help/734933888443065), и передаются в [интегрированную CRM-систему](https://www.facebook.com/business/help/908902042493104). Также их можно использовать для [уже настроенных Webhooks](https://developers.facebook.com/docs/graph-api/webhooks/) и [интеграции с API](https://developers.facebook.com/docs/marketing-api/guides/lead-ads/).
## Чтобы настроить параметры отслеживания для моментальной формы:
  1. [Создайте моментальную форму](https://www.facebook.com/business/help/375478503258484) для рекламы для сбора лидов.
  2. В разделе **Моментальная форма** скопируйте существующую форму или создайте новую.
  3. В окне **Создание формы** нажмите **Настройки**.
  4. Откройте раздел **Параметры отслеживания** , чтобы добавить нужные варианты. Например, если вы хотите добавить параметр источника UTM со значением Facebook, введите utm_source в поле **Название параметра** и Facebook в поле **Значение параметра**.
  5. Нажмите кнопку **Добавить параметр** , чтобы продолжить добавлять параметры в моментальную форму.
  6. Добавив нужные параметры, завершите создание моментальной формы и рекламы для сбора лидов.


Когда реклама будет одобрена, вы начнете получать заполненные моментальные формы с параметрами отслеживания.
**Примечание**. Поля названий и значений параметров не могут быть пустыми. Они должны быть уникальными и не могут совпадать с названиями других полей в форме.
Параметры отслеживания можно предварительно проверить с помощью [инструмента для тестирования рекламы для сбора лидов](https://developers.facebook.com/tools/lead-ads-testing/). Он позволяет создавать тестовые лиды, которые включают в себя параметры отслеживания. Вы можете получить данные о лидах и убедиться, что их параметры правильные.
