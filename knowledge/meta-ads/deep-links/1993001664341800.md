# Настройка автоматического расширенного поиска совпадений в Meta Events Manager
![](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=438223955063449&version=1666716831&transcode_extension=webp)
Вы можете получить пошаговые инструкции по настройке автоматического расширенного поиска совпадений от технического специалиста по решениям Meta. Узнайте, [_как запланировать звонок со специалистом Meta_](https://www.facebook.com/business/meta-pro-team/technical-pro?lead_source=sbg_2025_q4_ou_specialist_technical_helpcenter_xyz).
Если вы не можете отредактировать код сайта или приложения, для которого хотите использовать [расширенный поиск совпадений](https://www.facebook.com/business/help/611774685654668), включите автоматическую версию этой функции.

### Перед использованием автоматического расширенного поиска совпадений для сайта
Автоматический расширенный поиск совпадений будет работать на сайте, если:
  1. Ваш сайт содержит поля формы с запросом информации, которую вы хотите использовать для автоматического расширенного поиска совпадений (например, электронный адрес, номер телефона, имя, фамилия, город, регион, страна, почтовый индекс и пол).
  2. Пиксель Meta размещен на веб-страницах, где люди, скорее всего, будут вводить нужную информацию.
  3. Пиксель не находится в iFrame.
  4. Вы не используете пиксель IMG.
  5. Ваша компания не относится к отрасли с ограничениями.

### Перед использованием автоматического расширенного поиска совпадений для приложения
Автоматический расширенный поиск совпадений будет работать в приложении, если:
  1. Вы используете Facebook SDK версии 5.8.0 или более поздней.
  2. Ваше приложение содержит источники, такие как поля формы, запрашивающие информацию, которую вы хотите настроить с помощью автоматического расширенного поиска совпадений (например, электронный адрес или номер телефона).
  3. Ваша компания не относится к отрасли с ограничениями.


**Примечание.** Если компании (в том числе их [источники данных](https://www.facebook.com/business/help/1910428752389787) о событиях, например пиксель Meta) относятся к отрасли с ограничениями, им могут быть недоступны некоторые функции. Если в Meta Events Manager появилось сообщение "Невозможно использовать автоматический расширенный поиск совпадений", вероятно, ваша компания относится к отрасли с ограничениями. Примеры таких отраслей: банковская сфера, кредитование, финансовые услуги, страхование, здоровье и фармацевтика. Кроме этого, автоматический расширенный поиск совпадений может быть недоступен для пикселей IMG и пикселей в iFrame, а также если вы не используете Facebook SDK версии 5.8.0 или более поздней. В таких случаях вы можете настроить ручной расширенный поиск совпадений.
## Настройка автоматического расширенного поиска совпадений для сайта или мобильного приложения
  1. Откройте [**Events Manager**](https://business.facebook.com/events_manager2/list).
  2. Нажмите **Источники данных** ![](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=408108114296304&version=1774528167&transcode_extension=webp) в левой части экрана.
  3. Выберите название и ID нужных данных.
  4. Откройте вкладку **Настройки**.
  5. Для параметра **Автоматический расширенный поиск совпадений** переведите переключатель в положение **ВКЛ.**.
  6. Выберите **Показать параметры информации о клиентах**.
  7. Выберите **ВКЛ.** для параметров, которые хотите получать с сайта или из приложения. Чтобы добиться наилучших результатов, выберите варианты **Электронный адрес** и **Номер телефона**.


Результаты появятся в Meta Events Manager в течение 48 часов после завершения настройки.

## Будьте в курсе новостей от Meta for Business
Укажите электронный адрес, на который вы хотите получать новости о мероприятиях и продуктах, а также другую информацию от Meta for Business. 
Отправляя эту форму, вы соглашаетесь получать по электронной почте маркетинговые материалы от Meta, в том числе новости, информацию о мероприятиях, обновления и специальные предложения. Вы можете отказаться от рассылки в любой момент. Для этого нужно, например, нажать ссылку для отмены подписки в нашем электронном письме.
