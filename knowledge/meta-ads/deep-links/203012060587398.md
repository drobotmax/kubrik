# Рекомендации по выбору целей по результативности для просмотров целевой страницы
Выбор цели по результативности "Просмотры целевой страницы" помогает повысить качество трафика, который реклама привлекает на ваш сайт.
  * **Если вы хотите привлечь больше трафика на определенные страницы сайта** , выполните оптимизацию для просмотров целевой страницы, а не для кликов по ссылке. Это позволит повысить качество трафика.
  * **Если вы хотите увеличить число конверсий** , сначала выполните оптимизацию для конверсий. Однако цель "Увеличение числа конверсий" подходит не всем. Группа объявлений с оптимизацией для конверсий должна получать примерно 50 конверсий в неделю, чтобы приносить высокие результаты. Если ваша группа объявлений не настолько успешна, хорошей альтернативой будет оптимизация для просмотров целевой страницы. Это также подходящий вариант, если вы ещё не настроили в пикселе Meta события нижнего уровня воронки (например, покупку).
  * **Если формат вашей рекламы (например, "Предложения" или Messenger) предполагает переход на Страницу или к видео на Facebook** , выберите цель по результативности "Привлечь больше посетителей на Страницу Facebook", а не "Просмотры целевой страницы".


При определенных условиях в инструменте "Оценка возможностей" могут появляться проверенные на практике рекомендации по просмотрам целевой страницы. Оценка возможностей — это новый инструмент от Meta, который помогает определять и ранжировать эффективные рекомендации для кампаний практически в режиме реального времени. [Подробнее об оценке возможностей](https://www.facebook.com/business/help/804913634782260).
**Примечание.** Оценка возможностей (в том числе высокая) сама по себе не отражает текущую или будущую результативность. Фактическая результативность зависит от многих факторов, поэтому оценка возможностей не гарантирует ее.
