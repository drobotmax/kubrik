# Установка лимита затрат для рекламного аккаунта
![](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=438223955063449&version=1666716831&transcode_extension=webp)
Если в вашем рекламном аккаунте настроено использование [доступных средств](https://www.facebook.com/business/help/350519228479855), вы не сможете установить для него лимит затрат.
Вы можете установить лимит затрат для аккаунта, чтобы не потратить больше запланированного бюджета. По достижении лимита затрат показ ваших объявлений будет приостановлен, а средства рекламного аккаунта перестанут расходоваться.
Лимит затрат для аккаунта позволяет управлять общей суммой расходов, которую ваш рекламный аккаунт может потратить на все кампании за всё время. Этот лимит не сбрасывается автоматически по истечении определенного периода времени (например, в конце месяца).
**Примечание**. Лимит затрат не может быть меньше суммы, которую вы уже потратили. Если введенный вами лимит недействителен, повысьте его или сбросьте потраченную сумму до 0 долл. США.
## Прежде чем начать
  * Создать лимит затрат может только администратор рекламного аккаунта.
  * Любые изменения лимита (в том числе при его создании) вступают в силу примерно через 15 минут.
  * Для рекламных аккаунтов, в которых настроены [доступные средства](https://www.facebook.com/help/1730009227245741), нельзя установить лимит затрат.


## Установка лимита затрат для аккаунта 
Чтобы установить лимит затрат для аккаунта:
  1. Перейдите в [**настройки оплаты**](https://www.facebook.com/ads/manager/account_settings/account_billing).
  2. В разделе **Лимит затрат для аккаунта** нажмите **Параметры** ![](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=250425836038370&version=1773072306&transcode_extension=webp).
  3. Нажмите **Установить лимит**.
  4. Введите нужное значение.
  5. Нажмите **Сохранить**.
  6. Нажмите **Готово** , чтобы подтвердить изменения.


Вы установили лимит затрат для своего аккаунта. Изменения вступят в силу примерно через 15 минут. Теперь вы можете отслеживать, какую сумму можете потратить до достижения лимита. Чтобы реклама показывалась после достижения лимита, [измените](https://www.facebook.com/help/764293677236889) или [удалите](https://www.facebook.com/help/2135167003399680) его либо [сбросьте потраченную сумму до 0 долл. США](https://www.facebook.com/help/1892523930869907).
