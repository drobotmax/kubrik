# Форматирование списка клиентов для создания пользовательской аудитории
Список клиентов — это файл, который содержит информацию о ваших клиентах, например имена и электронные адреса. Такая информация называется личными данными. Мы используем их, чтобы сопоставлять ваших клиентов с пользователями Facebook и Instagram.
## Прежде чем начать
Убедитесь, что список клиентов:
  * создан в формате CSV или TXT;
  * содержит данные по крайней мере 100 клиентов;
  * содержит по крайней мере один тип личных данных;
  * содержит столбцы с правильным форматированием, которое соответствует каждому типу личных данных.


Meta использует криптографическую защиту (хэширование), чтобы безопасно сопоставлять список клиентов с данными на нашей платформе. [Подробнее.](https://www.facebook.com/business/help/112061095610075)
## Правила форматирования
**Основные личные данные**. Необходимо добавить хотя бы один тип личных данных, однако чем больше типов личных данных вы укажете, тем выше будет доля совпадения.
