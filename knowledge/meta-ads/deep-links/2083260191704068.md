# О цели "Продвижение приложения" в Meta Ads Manager
[Создайте кампанию](https://www.facebook.com/business/help/1292656978738967) с целью "Продвижение приложения", чтобы увеличить количество установок приложения, оптимизировать рекламу для событий в приложении или привлечь наиболее ценных клиентов, которые будут приносить максимально ценные конверсии. Эту цель стоит использовать, если вы, например, недавно выпустили приложение для доставки еды и хотите, чтобы люди его установили.
**Примечание.** Раньше эта цель называлась "Установки приложения". Ее название изменилось, а функции — нет. [_Подробнее об обновлении целей рекламы._](https://www.facebook.com/business/help/325793898950394)
## Чтобы создать кампанию с целью "Продвижение приложения", вам потребуются:
  * **Приложение, зарегистрированное на сайте Meta for Developers.** Зарегистрировать приложение можно [здесь](https://developers.facebook.com/quickstarts/).  
**Примечание.** Вы сможете создать и запустить рекламу с целью "Продвижение приложения", даже если не зарегистрируете приложение. Однако в таком случае оптимизация будет автоматически выполняться для кликов по ссылке (то есть по объявлению), а не для установок приложений.
  * **Рекламный аккаунт, связанный с приложением, которое вы хотите продвигать.** Разработчик или другой человек с правами администратора для приложения, зарегистрированного на Facebook, может связать рекламные аккаунты с приложением. Для этого откройте [панель приложений](https://developers.facebook.com/apps/) и выберите нужное приложение. Затем нажмите **Настройки приложения** и в раскрывающемся меню выберите **Дополнительно**.  
**Примечание.** Вы сможете создать и запустить рекламу с целью "Продвижение приложения", даже если не свяжете рекламный аккаунт с приложением. Однако в таком случае оптимизация будет автоматически выполняться для кликов по ссылке (то есть по объявлению), а не для установок приложений.
  * **Facebook SDK и события в приложении.** Узнать больше о Facebook SDK и событиях в приложении можно на сайте [Meta for Developers](https://developers.facebook.com/docs/app-ads/sdk). Чтобы оптимизировать рекламу для установок, событий в приложении или ценности, интегрируйте SDK и добавьте события в приложении. С помощью SDK вы также сможете отслеживать и измерять ценность, количество установок или событий в приложении.

## Будьте в курсе новостей от Meta for Business
Укажите электронный адрес, на который вы хотите получать новости о мероприятиях и продуктах, а также другую информацию от Meta for Business. 
Отправляя эту форму, вы соглашаетесь получать по электронной почте маркетинговые материалы от Meta, в том числе новости, информацию о мероприятиях, обновления и специальные предложения.
