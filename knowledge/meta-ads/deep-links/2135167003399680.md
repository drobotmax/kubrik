# Удаление лимита затрат для рекламного аккаунта
![](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=438223955063449&version=1666716831&transcode_extension=webp)
Если в вашем рекламном аккаунте настроены [доступные средства](https://www.facebook.com/business/help/350519228479855), вы не сможете установить для него лимит затрат.
Лимит затрат для рекламного аккаунта можно изменить в разделе "Настройки оплаты". По достижении лимита затрат для рекламного аккаунта показ ваших объявлений будет приостановлен, а средства аккаунта перестанут расходоваться. Чтобы сразу возобновить показ рекламы, вы можете удалить или [изменить лимит](https://www.facebook.com/help/764293677236889), а также [сбросить потраченную сумму](https://www.facebook.com/help/1892523930869907).
**Примечание**. Вы сами устанавливаете лимит затрат для рекламного аккаунта. Если ваша реклама приостановлена и бюджет не израсходован или лимит затрат для рекламного аккаунта не достигнут, возможно, вы исчерпали [дневной лимит затрат](https://www.facebook.com/business/help/563129151097553), который устанавливается Meta и сбрасывается каждый день. 
## Прежде чем начать
Вы должны быть администратором рекламного аккаунта. [Подробнее о разрешениях и ролях для рекламы](https://www.facebook.com/business/help/155909647811305).
## Удаление лимита затрат для рекламного аккаунта
Чтобы удалить лимит затрат для рекламного аккаунта:
  1. Перейдите в [настройки оплаты](https://business.facebook.com/billing_hub/payment_settings).
  2. В разделе **Лимит затрат для аккаунта** нажмите **Параметры** ![](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=250425836038370&version=1773072306&transcode_extension=webp).
  3. Нажмите **Удалить**.


Лимит затрат для рекламного аккаунта будет удален примерно через 15 минут. После этого показ вашей рекламы продолжится.
