# Об инструменте настройки событий на сайтах
Добавив базовый код пикселя Meta на сайт, вы можете настраивать события и параметры без написания кода с помощью инструмента настройки событий. [События](https://www.facebook.com/business/help/964258670337005) — это действия, которые люди совершают на вашем сайте, например покупки. Параметры предоставляют дополнительные сведения об этом действии, например о сумме покупки. События используются в рекламных инструментах Meta, позволяющих точнее измерять результативность кампании, расширять аудиторию и находить людей, которые могут совершить нужные вам действия.
Настроить события можно разными способами: путем добавления кода вручную, с помощью партнерских интеграций или инструмента настройки событий. Этот инструмент может находить и предлагать события на основе текста кнопок на вашем сайте, соответствующих названиям стандартных событий пикселя. Если вы хотите настроить события, которых нет в списке рекомендуемых, добавьте события для кнопок или URL страниц своего сайта вручную. 
После подключения инструмента настройки событий в Meta Events Manager, вы сможете:
  * Настраивать любое [стандартное событие сайта](https://www.facebook.com/business/help/402791146561655) напрямую на своем сайте без написания кода.
  * Настраивать параметры значения и валюты для всех стандартных событий без написания кода.
  * Настраивать параметры ID и типа контента для следующих событий: "Просмотр контента", "Добавление в корзину", "Начало оформления заказа", "Покупка", "Поисковый запрос", "Добавление в список желаний" и "Лид". Если вы настроили параметры ID и типа контента и в вашем каталоге есть хотя бы один товар, вы можете показывать объявления из каталога Meta Advantage+ (предыдущее название — динамическая реклама) для своих товаров. [Подробнее об обязательных событиях и параметрах пикселя для объявлений из каталога Advantage+](https://www.facebook.com/business/help/606577526529702).

Вы также можете использовать для событий кнопки, например, когда клиент нажимает кнопку для добавления товара в корзину, не переходя при этом на другую страницу. URL страниц помогают избежать ошибочных срабатываний пикселя. Например, если вы настраиваете событие "Покупка" для кнопки **Купить** , пиксель может регистрировать каждое нажатие кнопки как событие независимо от того, совершил ли клиент покупку.
Чтобы использовать инструмент настройки событий, перейдите в [Events Manager](https://www.facebook.com/events_manager2/) или следуйте [этим инструкциям](https://www.facebook.com/business/help/777099232674791).
