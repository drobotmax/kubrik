# Редактирование лимита затрат для кампании в Meta Ads Manager
В Meta Ads Manager можно отредактировать лимит затрат для кампании.
## Редактирование лимита затрат для кампании
Чтобы отредактировать лимит затрат для кампании:
  1. Откройте [Ads Manager](https://www.facebook.com/adsmanager/manage/campaigns).
  2. Найдите нужную кампанию в таблице в нижней части страницы.
  3. Наведите курсор на название кампании и нажмите **Редактировать** ![](https://scontent-prg1-1.xx.fbcdn.net/v/t39.2365-6/11404858_1683331455232654_27231468_n.png?stp=dst-webp&_nc_cat=100&ccb=1-7&_nc_sid=e280be&_nc_ohc=92Jw1tT6JZYQ7kNvwFFvAx6&_nc_oc=AdpJfSsoMzBK4F1-YlzeNdky7q-zrvoYgBrrnYZrflWccJ8aSrof2W3iahCZYO1svNE&_nc_zt=14&_nc_ht=scontent-prg1-1.xx&_nc_gid=pDKx1ZIlVAN_FekxKZpcfA&_nc_ss=7a389&oh=00_AfwhlK4FxsyCAlkjZhbL3bKlQv3sj8grWZPJgFgJRydO-g&oe=69E3A75D).
  4. В разделе **Сведения о кампании** найдите пункт **Лимит затрат для кампании**.
  5. Рядом с полем **Лимит затрат для кампании** нажмите кнопку **Редактировать**.
  6. Укажите новый лимит.
  7. Нажмите **Опубликовать**.


Внесенные изменения будут сохранены в течение 15 минут.
Чтобы продолжить показ рекламы по достижении лимита, отредактируйте или [удалите](https://www.facebook.com/help/390629621509687) лимит затрат для кампании.
