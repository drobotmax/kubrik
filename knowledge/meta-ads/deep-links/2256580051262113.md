# Управление вариантами в каталоге в Commerce Manager
[Варианты](https://www.facebook.com/business/help/363060785327110) — это версии одного товара в вашем каталоге, например футболки разных размеров и цветов. Их нужно настроить, чтобы они правильно показывались в ваших объявлениях и магазине.

##  [](https://www.facebook.com/business/help/2256580051262113?id=725943027795860)Добавление и удаление вариантов вручную
Вы можете добавить варианты, когда вручную: 
  * [Добавляете новый товар в каталог.](https://www.facebook.com/business/help/1670875312991238)
  * [Редактируете существующий товар.](https://www.facebook.com/business/help/1194178240952919)


Для каждого товара вручную можно добавить не более 100 вариантов. Создав варианты, вручную обновите для каждого из них данные в полях, например price (цена) и availability (наличие). Вариант, добавленный вручную, тоже можно [удалить](https://www.facebook.com/business/help/428079314773256) только вручную. 
##  [](https://www.facebook.com/business/help/2256580051262113?id=725943027795860)Добавление вариантов с помощью фида данных
Чтобы добавить варианты с помощью фида данных:
  1. Добавьте поле **item_group_id** (ID группы товаров) в фид данных. Укажите в этом поле одинаковый ID группы для каждого варианта одного товара, чтобы обозначить, что они входят в одну группу. Для каждого варианта также нужно указать уникальный идентификатор контента в поле **id**. ID групп должны отличаться от ID контента в вашем каталоге.
  2. Добавьте минимум одно свойство, указывающее на отличие варианта, например **color** (цвет), **size** (размер), **material** (материал), **pattern** (узор), **gender** (пол) или **additional_variant_attribute** (дополнительное свойство варианта). Укажите те же свойства для каждого варианта в группе. Например, если вы указали цвет и размер для одного варианта, введите эти данные и для остальных. У каждого варианта должна быть уникальная комбинация свойств. 
Ниже приведен пример фида данных, содержащий 3 варианта с разными свойствами цвета и размера.

  
| id  | title  | item_group_id  | color  | size  | link  |  
| --- | --- | --- | --- | --- | --- |  
| 1234  | Синяя рубашка  | Shirt_1  | Синий  | S  | http://www.jaspersmarket.com/products/blue_shirt  |  
| 5678  | Красная рубашка  | Shirt_1  | Красный  | S  | http://www.jaspersmarket.com/products/red_shirt  |  
| 9123  | Зеленая рубашка  | Shirt_1  | Зеленый  | L  | http://www.jaspersmarket.com/products/green_shirt  |  
Следующий пример показывает неправильно настроенные варианты: в первой строке поле **size** осталось пустым, а во второй и третьей у полей **color** и **size** одинаковые значения ("красный" и "L").  
| id  | title  | item_group_id  | color  | size  | link  |  
| --- | --- | --- | --- | --- | --- |  
| 1234  | Синяя рубашка  | Shirt_1  | Синий  |   | http://www.jaspersmarket.com/products/blue_shirt  |  
| 5678  | Красная рубашка  | Shirt_1  | Красный  | L  | http://www.jaspersmarket.com/products/red_shirt  |  
| 9123  | Красная рубашка  | Shirt_1  | Красный  | L  | http://www.jaspersmarket.com/products/red_shirt  |  
Когда всё будет готово, загрузите фид в свой каталог. Вы можете [загрузить новый фид данных](https://www.facebook.com/business/help/125074381480892) или [обновить существующий](https://www.facebook.com/business/help/1013894008673621).
**Примечание.** С помощью фида данных можно добавить не более 800 вариантов для каждого товара. Когда в группе будет 300 вариантов, вы увидите предупреждение.
##  [](https://www.facebook.com/business/help/2256580051262113?id=725943027795860)Советы по настройке вариантов
  * Укажите хотя бы одно свойство варианта: **color** (цвет), **size** (размер), **material** (материал), **pattern** (узор), **gender** (пол) или **additional_variant_attribute** (дополнительное свойство варианта). Для различения вариантов и правильного показа недостаточно использовать разные изображения. [определенных категорий товаров](https://www.facebook.com/business/help/1027180054842090) (например, одежды и обуви), если вы продаете их с возможностью оформления заказа на Facebook и в Instagram.
  * Добавьте другое основное изображение для вариантов, которые выглядят по-разному. У вариантов товара разных размеров может быть одинаковое изображение.
  * Указывайте уникальные ссылки для вариантов. Добавьте ссылку на страницу вашего сайта с предварительно выбранным товаром, чтобы клиентам не пришлось снова выбирать нужный вариант.
  * Все варианты товара должны принадлежать одному фиду данных.
  * Если вы добавляете в каталог варианты существующего товара, укажите его ID контента (**id**) хотя бы для одного варианта в фиде. Так вы сохраните все данные о событиях пикселя Meta для этого товара. Изменение ID может снизить результативность вашей рекламы. Узнать больше можно на сайте [Meta for Developers](https://developers.facebook.com/docs/commerce-platform/catalog/updating#2--add-variants).


##  [](https://www.facebook.com/business/help/2256580051262113?id=725943027795860)Удаление вариантов с помощью фида данных
  * Чтобы удалить вариант, [удалите товар](https://www.facebook.com/business/help/428079314773256) с помощью фида данных.
  * Чтобы удалить вариант из группы, [обновите свой фид данных](https://www.facebook.com/business/help/1013894008673621). Найдите нужный вариант в ленте и удалите данные в поле **item_group_id** (ID группы товаров), чтобы этот вариант больше не относился к этой группе.


##  [](https://www.facebook.com/business/help/2256580051262113?id=725943027795860)Управление вариантами на партнерской платформе
Если вы управляете товарами с помощью подключенной партнерской платформы, например Shopify, все настроенные вами варианты синхронизируются с Commerce Manager.
В Shopify можно сделать так, чтобы у каждого варианта в каталоге было выбранное вами изображение. Подробнее о том, [как изображения товаров синхронизируются с Shopify](https://www.facebook.com/business/help/1638980739993011).
