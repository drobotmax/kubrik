# Об отчетах по нескольким аккаунтам в инструменте "Отчеты по рекламе"
Выбирая разные рекламные аккаунты для сравнения результативности объявлений, вы получаете отчет по нескольким аккаунтам. 
## Что можно делать с помощью отчетов по нескольким аккаунтам?
  * Просматривать подробные разбивки и группировки по разным аккаунтам с учетом таких критериев, как возраст, пол, страна, регион, место размещения и устройства.
  * Анализировать информацию на уровне рекламного аккаунта, кампании, группы объявлений или отдельного объявления.
  * Анализировать общий уникальный охват и частоту по аккаунтам.
  * Анализировать сводные метрики по рекламным аккаунтам.
  * Стандартизировать настройку атрибуции. Вы можете выбрать одинаковую настройку для всех метрик в отчете. Эта функция доступна только для отчетов по нескольким рекламным аккаунтам.


## Какие функции не поддерживаются при использовании отчетов по нескольким аккаунтам?
Некоторые функции недоступны для отчетов по нескольким аккаунтам.
  * **В отчетах по нескольким аккаунтам не поддерживаются некоторые метрики и разбивки**. 
    * Примеры неподдерживаемых метрик: "Результат", "Диагностика и рейтинг объявлений", "Просмотры видео".
    * Примеры неподдерживаемых разбивок: "ID продукта", "Устройство конверсии", "Карточка кольцевой галереи".
  * **В отчетах по нескольким аккаунтам не поддерживаются разные часовые пояса**. 
    * В отчете по одному аккаунту используется часовой пояс из настроек рекламного аккаунта.
    * При выборе отчета по нескольким аккаунтам по умолчанию будет установлен часовой пояс бизнес-портфолио. 
    * Если в бизнес-портфолио он не выбран, в отчете по умолчанию будет установлен часовой пояс UTC.
  * **Отчеты по нескольким аккаунтам нельзя планировать по электронной почте**. 
  * **В отчетах по нескольким аккаунтам используются только данные за последние 24 месяца**. В отчете по одному аккаунту можно использовать данные за последние 37 месяцев. Для отчетов по нескольким аккаунтам доступны только данные за последние 24 месяца. 


Примечание. Если вы хотите использовать эти функции, переключитесь на отчет по одному аккаунту.
