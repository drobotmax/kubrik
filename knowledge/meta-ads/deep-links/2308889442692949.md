# Об оптимизации для событий в приложении
Оптимизация для событий в приложении позволяет наладить показ рекламы с использованием [стандартных и специально настроенных событий](https://www.facebook.com/business/help/235457266642587) для цели "Продвижение приложения". Благодаря этому вы сможете показывать свою рекламу [людям](https://www.facebook.com/business/help/283579896000936), которые с большей вероятностью совершат значимые для вашего приложения или компании действия. Предположим, вы хотите повысить количество покупок в приложении. В таком случае вы можете запустить рекламу с целью "Продвижение приложения" и оптимизировать ее для событий покупки.
Оптимизация для событий в приложении позволяет повысить количество установок и действий в приложении в рамках одной кампании по продвижению приложения. Например, вы запускаете кампанию для увеличения количества скачиваний нового игрового приложения и хотите, чтобы игру скачивали люди, которые действительно будут в нее играть. Оптимизируйте рекламу для события "Достижение уровня", и ее увидят люди, которые с наибольшей вероятностью скачают игру и продвинутся в ней дальше первого уровня. Аналогичным образом можно оптимизировать рекламу для увеличения количества покупок в приложении с помощью события "Покупка".
Перед оптимизацией для событий в приложении убедитесь, что вы подключили Facebook SDK или решение партнера по мобильным измерениям, а в вашем приложении используются стандартные или специально настроенные события. Затем убедитесь, что ваше приложение соответствует требованиям для протокола [Измерение на основе агрегированных данных о событиях](https://www.facebook.com/business/help/617021863580582) или [SKAdNetwork от Apple](https://www.facebook.com/business/help/2750680505215705). Если в кампаниях для iOS 14+ с целью "Продвижение приложения" и протоколом "Измерение на основе агрегированных данных о событиях" будет использоваться Facebook SDK для iOS 8.0 или более поздней версии, дополнительные требования для Facebook SDK выполнять не нужно.
## Оптимизация для стандартных событий в приложении
Некоторые стандартные события пока недоступны для оптимизации. Подробнее о доступных событиях в приложении можно узнать на сайте [Meta for Developers](https://developers.facebook.com/docs/app-ads/optimizing-your-app-ad?app-events-available%3Fhelpref=appl3).
## Оптимизация для специально настроенных событий в приложении
Вы можете выполнять оптимизацию не только для стандартных событий, но и для специально настроенных. Ее можно использовать в кампаниях по продвижению приложений, но оптимизация для ценности конверсии и параметры ценности при этом будут недоступны.
Если вы уже [сопоставили специально настроенное событие со стандартным](https://www.facebook.com/business/help/724097091659722), выполняйте оптимизацию для специально настроенного события, а не наоборот. [Подробнее о рекомендациях по оптимизации для событий в приложении](https://www.facebook.com/business/help/847716789206621).
Преимущества оптимизации для специально настроенных событий:
  * **Лучшая результативность рекламы**. Модель показа для специально настроенных событий в приложении помогает добиться лучших результатов.
  * **Больше возможностей**. Вы можете выполнять оптимизацию для событий, созданных специально под потребности вашего бизнеса.
  * **Упрощенная настройка**. Добавив в приложение специально настроенное событие, вы можете выполнить оптимизацию непосредственно в Ads Manager.


## Оптимизация для событий в приложении в рамках кампаний для iOS 14+, направленных на продвижение приложений
Чтобы использовать оптимизацию для событий в приложении в рамках кампаний для iOS 14+ на платформах Meta:
  * Убедитесь, что ваше приложение соответствует требованиям для протокола [Измерение на основе агрегированных данных о событиях](https://www.facebook.com/business/help/617021863580582) или [SKAdNetwork от Apple](https://www.facebook.com/business/help/2750680505215705). Если при создании кампании для iOS 14+, направленной на продвижение приложения, вы выберете метод атрибутирования "Измерение на основе агрегированных данных о событиях", у вас будет доступ к параметру атрибуции "7 дней после клика" и показу в Audience Network.
  * Если вы используете SKAdNetwork в качестве метода атрибутирования кампаний, добавьте событие, которое вы хотите оптимизировать, в конфигурацию событий SKAdNetwork в Events Manager. Это позволит вам выбрать событие для оптимизации при создании объявления в Ads Manager. Подробнее о том, [как настроить события в приложении для SKAdNetwork](https://www.facebook.com/business/help/670955636925518).
  * Если вы используете Facebook SDK для iOS, вам нужно также обновить его до версии 8.0 или более поздней, чтобы использовать оптимизацию для событий в приложении.
  * Для оптимальной результативности кампаний, которые используют оптимизацию для событий в приложении, рекомендуется отправлять все события конверсий.


**Примечание**. Обратитесь к юристу, чтобы убедиться, что ваши события не нарушают [Условия использования инструментов для бизнеса](https://www.facebook.com/legal/technology_terms) и правила использования данных. Не делитесь с нами конфиденциальными данными пользователей при настройке событий. [Подробнее о конфиденциальных данных](https://www.facebook.com/business/help/1057016521436966).
