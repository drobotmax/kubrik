# Реализация Conversions API с помощью персонализированных инструкций в Meta Events Manager
![](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=438223955063449&version=1666716831&transcode_extension=webp)
Для событий, отправленных через Conversions API, требуются определенные параметры данных. Эти параметры способствуют повышению качества событий, используемых для показа рекламы, и могут улучшить результаты кампании. Посетите сайт Meta for Developers, чтобы [узнать, какие параметры необходимы](https://developers.facebook.com/docs/marketing-api/conversions-api/parameters/server-event).
Вы или ваш разработчик можете настроить [Conversions API](https://www.facebook.com/business/help/2041148702652965) вручную с помощью кода, следуя пошаговому руководству в Meta Events Manager.
**Примечание**. Если вы не можете реализовать Conversions API вручную, попробуйте [другие способы настройки](https://www.facebook.com/business/help/433493041367251).
Настройка с помощью пошагового руководства состоит из 2 этапов:
  1. **Создание персонализированных инструкций по настройке** (обычно выполняется рекламодателем)  
Выберите события и параметры, которые вы хотите настроить. Иногда параметры и события выбираются по умолчанию, но их можно изменить. Информация о выбранных событиях и параметрах появится в персонализированных инструкциях по настройке, которые вы можете отправить себе или разработчику.
  2. **Реализация Conversions API** (обычно выполняется разработчиком)  
Вы или ваш разработчик можете настроить Conversions API, следуя созданным вами персонализированным инструкциям. Эти инструкции появляются в Events Manager и содержат информацию о выбранных событиях и параметрах, а также ссылки на документацию по реализации API и другие ресурсы для разработчиков.


## Прежде чем начать
  * Убедитесь, что у вас полный доступ к бизнес-портфолио, которому принадлежит пиксель Meta или набор данных. [Подробнее](https://www.facebook.com/business/help/2169003770027706).
  * Убедитесь, что у вас есть доступ к серверу компании.
  * Создайте пиксель Meta или набор данных в зависимости от отправляемых вами типов событий. Подробнее о том, [как выбрать оптимальный для вашей компании способ создания набора данных](https://www.facebook.com/business/help/5270377362999582).
  * Ознакомьтесь с [нашими рекомендациями по отправке событий на сайте через Conversions API](https://www.facebook.com/business/help/308855623839366) (необязательно).


## Этап 1. Создание персонализированных инструкций по настройке (обычно выполняется рекламодателем)
Чтобы реализовать Conversions API с помощью пошаговой настройки в Events Manager:
  1. Откройте [**Meta Events Manager**](https://www.facebook.com/events_manager2/). 
  2. Нажмите **Наборы данных** ![](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=408108114296304&version=1774528167&transcode_extension=webp) в меню в левой части страницы. 
  3. Выберите название и ID данных, которые хотите связать с Conversions API.
  4. Если Conversions API не настроены:
    1. Нажмите **Добавить события** под графиком действий.
    2. Выберите **Добавить новую интеграцию**.
    3. Выберите **Conversions API** и нажмите **Настроить**.
    4. Нажмите **Настроить вручную** , а затем — **Далее**.
  5. Если Conversions API настроены:
    1. Нажмите **Управление интеграциями** в правой части графика действий.
    2. Раскройте меню рядом с пунктом **Conversions API** и нажмите **Настроить события**.
  6. Прочитайте обзор и нажмите **Продолжить**.
  7. Выберите события, которые хотите отправлять через Conversions API. В раскрывающемся меню вы увидите рекомендуемые события для вашего типа компании.
  8. Нажмите **Продолжить**.
  9. Выберите для каждого события параметры, которые хотите отправлять.
     * Нужно выбрать хотя бы один [параметр информации о клиентах](https://developers.facebook.com/docs/marketing-api/conversions-api/parameters/customer-information-parameters).
     * Узнать, как создать новые специально настроенные события и пользовательские параметры, можно на сайте [Meta for Developers](https://developers.facebook.com/docs/).
  10. Нажмите **Продолжить**.
  11. Просмотрите выбранные события и параметры.
  12. Нажмите **Подтвердить настройку**.
  13. Выберите, что вы хотите: отправить персонализированные инструкции разработчику или реализовать Conversions API самостоятельно.
  14. Чтобы **отправить инструкции** :
    1. Установите или снимите флажок рядом с пунктом **Отправить мне копию электронного письма**.
    2. Нажмите **Отправить инструкции**.
    3. Введите электронный адрес получателя и нажмите клавишу **ENTER**.
    4. Нажмите **Отправить**. Этот человек получит приглашение настроить Conversions API для вашей компании. Если у него нет роли в вашем аккаунте Business Manager, он также получит электронное письмо с предоставлением доступа на уровне разработчика. Получив доступ, разработчик сможет настроить Conversions API в разделе **Управление интеграциями** в Events Manager.
  15. Чтобы **реализовать API самостоятельно** :
    1. Нажмите **Открыть руководство по реализации** , чтобы перейти к настройке Conversions API в Events Manager.


Созданные вами персонализированные инструкции будут содержать технические сведения о том, как создавать и редактировать код в вашей серверной среде, чтобы внедрить Conversions API. Вы или ваш разработчик можете получить доступ к персонализированным инструкциям по настройке, выполнив шаги, описанные ниже в этапе 2.
**Примечание**. Когда вы выполните все этапы в Events Manager, необходимо завершить процес, отправив разработчику запрос POST. Иначе на индикаторе выполнения не достигнет значения 100 %. Существует ещё несколько причин, по которым индикатор выполнения не достигает значения 100 %.
## Этап 2. Реализация Conversions API (обычно выполняется разработчиком)
  1. Откройте [**Meta Events Manager**](https://www.facebook.com/events_manager2/). 
  2. Нажмите **Наборы данных** ![](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=408108114296304&version=1774528167&transcode_extension=webp) в меню в левой части страницы. 
  3. Выберите название и ID данных, которые хотите связать с Conversions API.
  4. Нажмите **Управление интеграциями**.
  5. Найдите интеграцию Conversions API и нажмите **Завершить настройку**.   
  
**Примечание**. Если этого варианта нет, возможно, Conversions API был настроен с помощью [документации для разработчиков](https://developers.facebook.com/docs/marketing-api/server-side-api), а не пошагового руководства в Events Manager. Чтобы настроить Conversions API с помощью пошагового руководства, сначала создайте персонализированные инструкции, выполнив шаги из этапа 1.
  6. Чтобы узнать больше об Conversions API, ознакомьтесь с ресурсами в пункте **Подробнее** в разделе **Информация об Conversions API**.
  7. Нажмите **Сгенерировать маркер доступа**. Скопируйте маркер доступа и сохраните его в надежном месте.
  8. Ознакомьтесь с информацией в разделе **Отправка событий через API** , чтобы узнать, как настраивать и отправлять события. В этом разделе вы найдете следующие ресурсы:
    1. Инструкцию по [отправке запроса POST](https://developers.facebook.com/docs/marketing-api/conversions-api/using-the-api#send).
    2. Список доступных стандартных событий и [параметров](https://developers.facebook.com/docs/marketing-api/conversions-api/parameters).
    3. Кнопку для запуска [помощника по полезным данным](https://developers.facebook.com/docs/marketing-api/conversions-api/payload-helper), который показывает, как нужно их структурировать.
    4. Список событий и параметров, выбранных при создании персонализированных инструкций.
    5. Инструкцию по [настройке параметра "Время транзакции события"](https://developers.facebook.com/docs/marketing-api/conversions-api/using-the-api#event-transaction-time).
    6. Инструкцию по [использованию пакетных запросов](https://developers.facebook.com/docs/marketing-api/conversions-api/using-the-api#batch-requests).
    7. Инструкцию по [тестированию событий](https://www.facebook.com/business/help/1624255387706033).
    8. Инструкцию по [отладке Conversions API](https://developers.facebook.com/docs/marketing-api/conversions-api/support#debug).
  9. Когда вы завершите настройку Conversions API и убедитесь, что он работает должным образом, нажмите **Завершить**.


Настройка Conversions API завершена. Вы можете отслеживать события, проверять работу параметров, а также выявлять и решать проблемы. Для этого откройте Events Manager и выберите пиксель или набор данных, связанный с Conversions API. Если вы настроили события на сайте, узнайте, [как контролировать работу Conversions API](https://www.facebook.com/business/help/586304118741779).
