# Создание лимита затрат для кампании в Meta Ads Manager
Вы можете установить лимит затрат для кампании, чтобы не потратить больше, чем планируете. Если вы зададите лимит затрат для кампании, показ объявлений и групп объявлений прекратится по его достижении.
**Примечание**. Лимит затрат для кампании нельзя установить, если вы используете бюджет кампании на весь срок действия. Если вы включили [бюджет кампании Advantage+](https://www.facebook.com/business/help/733979527611858) (другие названия — бюджет кампании или **Бюджет** со **включенным параметром Advantage+**), то вместо лимита затрат для кампании применяется бюджет на весь срок действия.
## Создание лимита затрат для кампании
Чтобы создать лимит затрат для кампании:
  1. Откройте [Meta Ads Manager](https://adsmanager.facebook.com/adsmanager/manage).
  2. Выберите кампанию, которую хотите отредактировать.
  3. Наведите курсор на название кампании и нажмите **Редактировать**.
  4. В разделе **Сведения о кампании** нажмите **Ещё параметры** и найдите пункт **Лимит затрат для кампании**.
  5. Рядом с полем **Лимит затрат для кампании** нажмите кнопку **Редактировать**.
  6. Установите флажок **Лимит затрат для рекламной кампании**.
  7. Введите нужное значение лимита.


Внесенные изменения будут сохранены в течение 15 минут.
Чтобы продолжить показ рекламы по достижении лимита затрат для кампании, [отредактируйте](https://www.facebook.com/help/2218113641763765) или [удалите](https://www.facebook.com/help/390629621509687) его.

Укажите электронный адрес, на который вы хотите получать новости о мероприятиях и продуктах, а также другую информацию от Meta for Business.
