# О партнерских интеграциях для событий на сайте
Приглашаем вас ознакомиться со следующим разделом о партнерских интеграциях Meta. Из этой статьи вы узнаете, как партнерские интеграции упрощают настройку отслеживания событий на сайте, а также помогают измерять и оптимизировать бизнес-результаты с помощью инструментов Meta. 
## О событиях на сайте
События на сайте — это действия, которые посетители выполняют на вашем сайте, например покупки, регистрации или просмотры страницы. Вы можете отслеживать эти события и использовать их, чтобы улучшать рекламу и измерение. Партнерские интеграции Meta позволяют автоматически регистрировать эти события без редактирования кода вручную.
## Преимущества партнерских интеграций
  * **Быстрая и простая настройка.** Вам не потребуется редактировать код сайта.
  * **Автоматическое отслеживание событий.** Отслеживайте ключевые действия, например покупки, регистрации и многое другое.
  * **Повышенная точность данных.** Используйте пиксель вместе с Conversions API для улучшения измерения в браузере и на сервере соответственно.


## Поддерживаемые типы партнеров
Meta поддерживает множество партнерских платформ для отслеживания событий на сайте. К ним относятся:
  * Платформы для сайтов (WordPress, Shopify, Joomla и т. д.).
  * Торговые платформы (WooCommerce, BigCommerce, Ecwid, OpenCart, PrestaShop, Magento/Adobe Commerce, SHOPLINE).
  * Диспетчеры тегов, платформы клиентских данных (CDP), интеграторы систем и многое другое.


## Пиксель Meta
  * **Преимущества:** измерение влияния рекламы, поиск новых клиентов и повышение продаж. 
  * **Настройка:**
    1. Откройте [Events Manager](https://www.facebook.com/events_manager2/partner_integrations). 
    2. Выберите партнера по работе с сайтом.
    3. Следуйте инструкциям по настройке.


## Conversions API
  * **Преимущества:** отправка веб-событий напрямую с сервера в Meta для повышения надежности данных и конфиденциальности. 
  * **Настройка:**
    1. Откройте вкладку **Источники данных** в [Events Manager](https://www.facebook.com/events_manager). 
    2. Выберите **Настройки**.
    3. Прокрутите до пункта **Выбрать партнера**.
    4. Выберите партнера и следуйте инструкциям.


**Примечание.** Если вашей платформы нет в списке, вы можете [настроить Conversions API вручную](https://www.facebook.com/business/help/232481544843294) или обратиться за помощью к поставщику услуг.
