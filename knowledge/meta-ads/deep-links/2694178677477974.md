# Работа с инструментом настройки событий в приложении без использования кода
![](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=438223955063449&version=1666716831&transcode_extension=webp)
С помощью этой функции вы можете настроить события в приложениях, использующих Facebook SDK для Android или iOS (версии от 4.34.0 до 7.1.1). Этот инструмент не поддерживается в приложениях, использующих Facebook SDK для iOS версии 8.0.0 или более поздней. Версия SDK указана в настройках [панели приложений](https://developers.facebook.com/apps/).
Вы можете использовать [Facebook SDK](https://www.facebook.com/business/help/1471413626484885), чтобы показывать рекламу [людям](https://www.facebook.com/business/help/283579896000936), которые с большей вероятностью выполнят в вашем приложении нужное вам действие. Такие действия называются событиями в приложении. Вы можете использовать оптимизацию событий в приложении, чтобы побудить зрителей рекламы на Facebook к определенным действиям (например, совершению покупки). Такую оптимизацию можно выполнять для стандартных и специально настроенных событий. [Подробнее о событиях в приложении](https://www.facebook.com/business/help/235457266642587).
Если у вас уже установлен Facebook SDK для [iOS](https://developers.facebook.com/docs/ios/getting-started) или [Android](https://developers.facebook.com/docs/android/getting-started), настроить события в приложении можно 4 способами:
  * **Установить код события в приложении вручную**. Подробнее о том, [как добавить код события в приложении вручную](https://www.facebook.com/business/help/1634426896605026).
  * **Использовать партнерскую интеграцию**. Подключите аккаунт партнера по мобильным измерениям к Meta и добавьте события в приложении. [Подробнее о партнерских интеграциях для приложения](https://www.facebook.com/business/help/251324098890446).
  * **Использовать рекомендуемые события в приложении**. Вы можете добавлять в приложение рекомендуемые стандартные события, не изменяя код. Подробнее о том, [как управлять рекомендуемыми событиями в приложении](https://www.facebook.com/business/help/542463586317574).
  * **Использовать инструмент настройки событий без кода**. Если у вас установлена актуальная версия Facebook SDK для iOS или Android, управлять событиями можно с помощью нашего простого инструмента с интуитивно понятным интерфейсом. Посетите сайт Meta for Developers, чтобы [просмотреть доступные версии](https://developers.facebook.com/docs/app-events/codeless-app-events), а затем выполните указанные ниже действия.


## Использование инструмента настройки событий
Чтобы использовать инструмент настройки событий в приложении:
  1. Откройте [**Meta Events Manager**](https://www.facebook.com/events_manager2/). 
  2. Нажмите **Наборы данных** ![](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=408108114296304&version=1774528167&transcode_extension=webp) в меню в левой части страницы. 
  3. Выберите название и ID нужных данных. 
  4. Нажмите **Добавить событие** в левом верхнем углу графика действий и выберите **Используйте наш новый инструмент настройки событий без кода** в раскрывающемся меню.
**Примечание**. Если ваш набор данных не получил никакой информации о действиях, возможно, вы не увидите вариант "Управление интеграциями". Тогда прокрутите вниз до раздела **Передача данных из мобильного приложения** и нажмите **Настроить Facebook SDK** , а затем — **Используйте наш новый инструмент настройки событий без кода**.
  5. Ознакомьтесь с сообщением и нажмите **Начать**.
  6. Выберите нужную платформу приложения и нажмите **Начать настройку**.
  7. Запустите приложение на телефоне и несколько раз встряхните устройство, чтобы подключиться к Events Manager.
  8. После подключения экран вашего приложения будет дублироваться на экране Events Manager.
  9. Выберите категорию приложения. Система выделит события в приложении, соответствующие этой категории.
  10. Добавьте или удалите рекомендуемые события.
  11. Чтобы добавить событие, которого нет в списке рекомендуемых, выберите на экране компьютера кнопку или ссылку в приложении, которую хотите отслеживать.
  12. Следуйте подсказкам на экране.
  13. Чтобы подтвердить выбор событий, нажмите **Проверить и завершить**.


События появятся в Events Manager в течение 30 минут. Вы можете использовать их для настройки [кампаний по рекламе приложений](https://www.facebook.com/business/help/1471413626484885).
Сейчас мы поддерживаем только стандартные события, но в дальнейшем к ним добавятся специально настроенные. Из параметров пока поддерживаются цена и валюта.
Соблюдайте [Условия использования инструментов Meta для бизнеса](https://www.facebook.com/legal/technology_terms). Не делитесь с нами конфиденциальными данными пользователей при настройке событий. [Подробнее о данных деликатного характера](https://www.facebook.com/business/help/361948878201809).
