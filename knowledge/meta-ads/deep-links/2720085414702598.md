# Об управлении количеством рекламы
![](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=438223955063449&version=1666716831&transcode_extension=webp)
Начиная с июня 2024 года вы больше не сможете использовать динамические креативы при создании групп объявлений в Ads Manager, если выберете цель "Продажи" или "Продвижение приложения". Это не повлияет на существующие кампании, использующие динамические креативы, однако мы рекомендуем использовать гибкий формат рекламы для достижения аналогичного результата. [Подробнее о гибком формате рекламы](https://www.facebook.com/business/help/835561738423867). 
## Количество рекламы и результативность
Количество активных объявлений в рекламном аккаунте или на Странице имеет важное значение в достижении предсказуемых, стабильных и оптимальных результатов.
Каждый раз при показе объявления наша система узнает больше о том, где и каким людям следует показывать такую рекламу. Чем чаще показывается объявление, тем лучше наша система оптимизирует его результативность.
Когда рекламодатель запускает слишком много объявлений одновременно, снижается частота показа каждого из них. Это значит, что меньше объявлений завершают этап обучения и больше средств тратится, прежде чем система показа оптимизирует результативность. Другими словами, слишком большое количество объявлений приводит к низкой результативности.
Чтобы получать более предсказуемые, стабильные и оптимизированные результаты, рекламодатель должен контролировать количество активных объявлений. Он также должен следить, чтобы количество объявлений, которые показываются или находятся на проверке, не превышало лимит для Страницы. [_Подробнее о лимитах объявлений_](https://www.facebook.com/business/help/766697140509126).
Тестирование новых креативов и маркетинговых стратегий по-прежнему имеет большое значение. Однако использование подобных тактик не должно мешать системе показа полноценно изучать каждое объявление.
## Рекомендации 
Советы, представленные ниже, помогут вам эффективно управлять количеством объявлений.
  1. **Объединяйте группы объявлений.**[Объединив группы объявлений](https://www.facebook.com/business/help/2419480091640105), вы быстрее получите количество конверсий, необходимое для завершения этапа обучения. 
     * **Если вы создаете отчеты для каждой аудитории** , объедините группы объявлений и воспользуйтесь [разбивками](https://www.facebook.com/business/help/264160060861852). Так вы получите подробный отчет с полной информацией о результативности рекламы.
     * **Если вы настроили таргетинг на небольшие географические регионы** , объедините группы объявлений для таргетинга на более крупные регионы. Так вы получите больше конверсий и поможете системе показа узнать больше информации о группах объявлений.
     * **Если вы создаете разные группы объявлений для разных мест размещения** , объедините группы объявлений и используйте [места размещения Meta Advantage+](https://www.facebook.com/business/help/196554084569964) (предыдущее название — автоматические места размещения) и [кастомизацию креативов](https://www.facebook.com/business/help/1044825198987622). Благодаря кастомизации креативов в разных местах размещения будут показываться те версии объявления, которые вы выберете.
     * **Если вы создаете разные группы объявлений для разных языков** , объедините группы объявлений и настройте [несколько языков для одной группы объявлений](https://www.facebook.com/business/help/1852644738394400). Вы также можете создать [объявление из каталога Meta Advantage+](https://www.facebook.com/business/help/733979527611858) для разных языков и стран. Для этого нужно добавить фид для языков и/или стран в свой каталог.
  2. **Уменьшайте количество объявлений в группах, но оставляйте разные рекламные креативы в каждой из них.** Одно объявление может содержать несколько (до 10) рекламных креативов. 
     * **Если вы тестируете несколько вариантов текста, используйте оптимизацию вариантов текста.** Одно объявление с несколькими оптимизированными текстами обычно более эффективно, чем несколько неоптимизированных объявлений. Этот способ позволяет системе показа автоматически оптимизировать каждый показ рекламы для обеспечения более высокой результативности.
     * **Если вы тестируете несколько вариантов креатива, используйте**[**динамические креативы**](https://www.facebook.com/business/help/170372403538781)
     * **Если вы тестируете несколько форматов, используйте креатив Advantage+ для каталога**. Эта функция помогает показывать людям разные рекламные форматы и креативы на основе предположений о том, какой вариант объявления получит больший отклик.
  3. **Используйте решения, разработанные специально для вашей отрасли.** Meta предлагает множество решений для отраслей, которые требуют большого количества объявлений. Наши инструменты позволяют продвигать весь каталог товаров с помощью меньшего количества объявлений. 
     * **Если вы хотите повысить узнаваемость нескольких офлайн-точек** , используйте функции для продвижения адресов магазинов. Так для каждой точки в рекламе будет подставлена информация о ней и отдельный креатив.
     * **Если вы рекламируете много товаров одной аудитории** , используйте [объявления из каталога Advantage+](https://www.facebook.com/business/help/397103717129942). Они позволяют нашей системе автоматически выбирать оптимальный товар для показа конкретному человеку. 
     * **Если вы рекламируете несколько объектов недвижимости** , объедините группы объявлений и используйте [рекламу недвижимости](https://www.facebook.com/business/help/1993509814217759), чтобы показывать отдельные объекты людям, которые с большей вероятностью их приобретут.
     * **Если вы рекламируете ассортимент автосалона** , используйте [рекламу ассортимента автосалона](https://www.facebook.com/business/help/1930396467200635), чтобы продвигать весь ассортимент транспортных средств с помощью меньшего количества объявлений.
