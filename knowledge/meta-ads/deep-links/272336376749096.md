# О стратегии "Цель по цене за результат"
![](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=438223955063449&version=1666716831&transcode_extension=webp)
Группы объявлений со стратегией ставок "Цель по цене за результат" или "Цель по ROAS" и таргетингом на пользователей iOS 14.5 либо более поздних версий должны показываться не менее 3 полных дней.
Цель по цене за результат — это одна из стратегий ставок на [рекламном аукционе](https://www.facebook.com/business/help/430291176997542?id=561906377587030) на платформах Meta. Это средняя сумма, которую мы стараемся обеспечить на протяжении всей кампании, когда динамически увеличиваем ставку для получения наилучших результатов. Это значит, что средняя цена за результат может превысить указанную сумму. 
Например, если ритейлер оптимизирует рекламу для покупок, в качестве предельной цены он может установить цену за покупку, при которой в среднем будет получать прибыль.
**Стратегия "Цель по цене за результат" подходит вам, если** : 
  * Вы хотите, чтобы цена за действие всегда оставалась средней независимо от рыночных условий.


**Примечания** : 
  * Бюджет при установленной цели по цене за результат может расходоваться медленнее, чем в случае использования стратегии ставок "Максимальное количество". Если вам важнее расходовать весь бюджет, чем придерживаться целевой цены за действие, тогда вам подойдет стратегия "Максимальное количество".
  * При использовании стратегии "Цель по цене за результат" [этап обучения](https://www.facebook.com/business/help/112167992830700?id=561906377587030) может длиться дольше.
  * Этап обучения может сопровождаться более частыми колебаниями цены. 
  * Когда группа объявлений завершит этап обучения, показ должен стабилизироваться. Мы будем стараться оптимизировать вашу кампанию согласно установленной вами цели по цене за результат, однако не можем этого гарантировать. Цены могут превышать указанную вами цель.
  * Если обычно конверсии происходят намного позже, чем в течение 7 дней после клика, то в расходах и цене за действие могут наблюдаться значительные колебания. Однако со временем результативность стабилизируется.
  * Стратегия "Цель по цене за результат" поддерживается не для всех целей оптимизации. Ознакомьтесь с [рекомендациями для стратегии "Цель по цене за результат"](https://www.facebook.com/business/help/176339196621566?id=2196356200683573&helpref=faq_content).
