# Места размещения и форматы рекламы, доступные при выборе определенной цели
![](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=438223955063449&version=1666716831&transcode_extension=webp)
С января 2026 года место размещения "Интересное в Instagram" больше не будет доступно в Ads Manager. Нажав на плитку в сетке на вкладке "Интересное", человек откроет полноэкранный проигрыватель Reels, а не ленту "Интересное". Группы объявлений с местом размещения "Интересное в Instagram" будут показываться в других местах размещения, на которые настроен таргетинг. Группы объявлений с местом размещения "Интересное в Instagram" будут показываться в других местах размещения, на которые настроен таргетинг. Вы по-прежнему можете выбрать главный экран раздела "Интересное" в Instagram в качестве места размещения.
Из этой статьи вы узнаете, какие [места размещения](https://www.facebook.com/business/help/407108559393196) и форматы доступны для каждой цели рекламы. Здесь указаны только те места размещения, которые поддерживаются API Facebook для всех рекламодателей. Такие продукты, как Meta Ads Manager и платформы бизнес-партнеров Meta, могут предоставлять доступ ещё не ко всем местам размещения, которые поддерживаются API.
**Примечания** :
  * [Гибкий формат рекламы](https://www.facebook.com/business/help/835561738423867) пока доступен только для кампаний с целью "Трафик", "Вовлеченность","Продажи" или "Продвижение приложения".
  * Starting in March 2026, the flexible format will no longer be available in **Ad setup**.
