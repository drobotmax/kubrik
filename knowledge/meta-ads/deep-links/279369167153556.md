# Настройка CRM при выборе цели по результативности "Лиды для конверсий"
Реклама для лидов с целью по результативности "Лиды для конверсий" позволяет охватывать более качественные лиды. Если вы выбираете этот вариант во время создания рекламы, рекомендуется подключить систему управления взаимодействием с клиентами (CRM). Так информация о ваших лидах будет отправляться в Meta. Подключив CRM, вы сможете выбрать этапы воронки продаж в соответствии с вашими целями по результативности.
## Прежде чем начать
Убедитесь, что у вас есть [бизнес-портфолио](https://www.facebook.com/business/help/1710077379203657).
## Подключение CRM
Вы можете начать подключение CRM в Meta Ads Manager, во время создания рекламы для лидов или прямо в Meta Events Manager.
Чтобы начать подключение в Ads Manager:
  1. Откройте [Ads Manager](https://www.facebook.com/adsmanager/manage/campaigns).
  2. Следуйте [инструкции по созданию рекламы для лидов](https://www.facebook.com/business/help/375478503258484).
  3. Перейдите в раздел **Конверсии** , раскройте меню **Лиды** и выберите **Лиды для конверсий**.
  4. Нажмите **Далее**.


Чтобы начать подключение в Events Manager:
  1. Откройте [Events Manager](https://www.facebook.com/events_manager2/list).
  2. Нажмите на значок **плюса** , чтобы подключить новый источник данных.
  3. Выберите **CRM** , а затем — **Подключить**.


Запустив процесс подключения, сделайте следующее:
  1. Создайте новый или выберите существующий набор данных, чтобы использовать его в новой интеграции с CRM. Когда всё будет готово, нажмите **Продолжить**.
  2. Откройте вкладку настроек набора данных и нажмите **Продолжить**.  
  
Если вы хотите создать прямую интеграцию, то интегрируйте свою CRM-систему с [Conversions API](https://www.facebook.com/business/help/2041148702652965) при помощи разработчика. Ознакомьтесь с [руководством по интеграции с CRM](https://developers.facebook.com/docs/marketing-api/conversions-api/conversion-leads-integration/crm-integration/3-implementing-the-crm-integration).  
  
Если вы хотите использовать стороннюю [_партнерскую интеграцию_](https://www.facebook.com/business/help/317857030149451), ознакомьтесь с соответствующей документацией.
  3. Завершив начальную интеграцию с CRM, вернитесь на вкладку настроек набора данных, чтобы проверить свой статус интеграции.  
  
**Примечание.** Во время этапа обучения загружайте события CRM для каждого обновления статуса лида. Когда информация о лиде в вашей CRM-системе меняется, отправляйте последнее обновление в качестве события.   
  
Если в вашей интеграции есть ошибки, вы увидите сообщения о них на вкладках "Обзор" и "Диагностика" в Events Manager. Там вы также найдете инструкции по их устранению.
