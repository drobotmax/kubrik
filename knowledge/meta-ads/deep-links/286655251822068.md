# Добавление компонентов в Instant Experience
В [конструкторе](https://www.facebook.com/business/help/212388117255165) можно самостоятельно создать Instant Experience с помощью специальных компонентов, расположив их практически в любом порядке. Это позволит вам представить свой бренд, товар или услугу.
В режиме [предпросмотра](https://www.facebook.com/business/help/1703421759894088) в Meta Ads Manager можно узнать, как будет выглядеть ваша реклама с Instant Experience на Facebook и в Instagram.
## Добавление компонентов
Нажмите **Добавить компонент** в конструкторе Instant Experience. Выберите тип каждого компонента и нажмите **OK**.

### Добавление заголовка
Заголовок — это изображение, которое появляется и остается в верхней части Instant Experience. Если формат не соответствует ширине экрана, выберите цвет фона и выполните заливку пустого пространства. В отличие от других компонентов, заголовок может быть только один и должен размещаться в верхней части Instant Experience.
Чтобы добавить заголовок в Instant Experience:
  1. Нажмите **Добавить компонент**.
  2. Выберите **Заголовок**.
  3. Нажмите **ОК**.
  4. Нажмите **Загрузить фото** и выберите нужное изображение или логотип.
  5. При необходимости настройте параметр **Цвет фона**.


Вместо изображения или логотипа в заголовок можно добавить текст. Для этого нажмите **Текст** и введите заголовок в поле **Название**. При необходимости настройте параметры **Цвет текста** и **Цвет фона**.

### Добавление фото
Фото — это одно изображение, которое можно добавить в Instant Experience.
Чтобы добавить фото в Instant Experience:
  1. Нажмите **Добавить компонент**.
  2. Выберите **Фото**.
  3. Нажмите **ОК**.
  4. Нажмите **Загрузить фото** и выберите нужное.
  5. В разделе **Место назначения (необязательно)** можно указать адрес страницы, на которую будет направлять изображение. Эта функция доступна, только если в меню **Макет** выбран параметр **Заполнить по ширине (с возможностью вставки ссылки)**.
  6. Раскройте меню **Макет** и выберите параметры [показа изображения](https://www.facebook.com/business/help/1493359177637102) на экране.

### Добавление видео
Видео — это одно видео, которое можно добавить в Instant Experience.
Чтобы добавить видео в Instant Experience:
  1. Нажмите **Добавить компонент**.
  2. Выберите **Видео**.
  3. Нажмите **ОК**.
  4. Нажмите **Загрузить видео** и выберите нужное.


Раскройте меню **Макет** и выберите параметры [показа видео](https://www.facebook.com/business/help/1493359177637102) на экране.

### Добавление кнопки
Кнопки могут содержать ссылки, которые направляют людей в нужное место назначения, например в магазин приложений, на сайт или в другой Instant Experience.
Чтобы добавить кнопку в Instant Experience:
  1. Нажмите **Добавить компонент**.
  2. Выберите **Кнопка**.
  3. Нажмите **ОК**.
  4. В разделе **Название** укажите текст кнопки.
  5. Откройте раздел **Место назначения (обязательно)**. Выберите нужный пункт в раскрывающемся меню и введите адрес места назначения.
  6. Выберите **стиль** и **цвет кнопки** , а также **цвет фона**.
  7. Если вы хотите, чтобы кнопка показывалась в нижней части Instant Experience, установите флажок рядом с параметром **Закрепить кнопку в нижней части**.

### Добавление кольцевой галереи
Кольцевая галерея — это серия карточек с изображениями, которые можно пролистывать. При желании вы также можете добавить ссылку: одну для всех карточек или разные. При этом все изображения должны быть одного размера. Если нажать на изображение на карточке, откроется место назначения.
**Примечание**. В предпросмотре в правой части конструктора Instant Experience можно посмотреть, как будет выглядеть кольцевая галерея. Для этого нужно нажать на нее и, удерживая, перетаскивать карточки.
Чтобы добавить кольцевую галерею в Instant Experience:
  1. Нажмите **Добавить компонент**.
  2. Выберите **Кольцевая галерея**.
  3. Нажмите **ОК**.
  4. Добавляйте или удаляйте карточки, чтобы в галерее осталось нужное вам количество (от 2 до 10).

### Добавление текстового блока
Текстовый блок — это место для добавления текста в Instant Experience. Это может быть короткий заголовок или целые абзацы.
Чтобы добавить текстовый блок в Instant Experience:
  1. Нажмите **Добавить компонент**.
  2. Выберите **Текстовое поле**.
  3. Нажмите **ОК**.
  4. Введите текст и отформатируйте его.
  5. При необходимости настройте параметр **Цвет фона**.


**Примечание**. Если вы хотите использовать недоступный шрифт, загрузите текстовый блок в Instant Experience в качестве изображения, предварительно сохранив его.

### Добавление группы товаров
Группа товаров — это группа позиций из вашего каталога. Instant Experience динамически показывает те товары, которые предположительно будут продаваться лучше всего.
Чтобы добавить группу товаров в Instant Experience, сначала загрузите [каталог](https://www.facebook.com/business/help/890714097648074). Группу товаров можно создать в [каталоге](https://www.facebook.com/business/help/620275848114281) или конструкторе Instant Experience.
Чтобы добавить группу товаров в Instant Experience:
  1. Нажмите **Добавить компонент**.
  2. Выберите **Группа товаров**.
  3. Нажмите **ОК**.
  4. В раскрывающемся меню **Каталог** выберите нужный каталог.
  5. В раскрывающемся меню **Группа товаров** выберите нужную группу товаров.
  6. Укажите максимальное количество товаров, которые должны показываться в Instant Experience.


**Примечание.** Порядок показа товаров в Instant Experience будет определен автоматически с учетом их результативности. В каждую группу можно добавить не более 50 товаров.
Чтобы создать новую группу товаров в конструкторе Instant Experience:
  1. Добавьте группу товаров согласно указанным выше инструкциям.

### Добавление локатора точек
Локатор точек — это компонент, который направляет [_людей_](https://www.facebook.com/business/help/283579896000936) в ваш магазин, учитывая их текущее местоположение. Чтобы добавить локатор точек, включите для своих Страниц функцию [Места](https://www.facebook.com/business/help/167222830549113). Локатор точек не показывается в Instagram.
Чтобы добавить локатор точек в Instant Experience:
  1. Нажмите **Добавить компонент**.
  2. Выберите **Локатор точек**.
  3. Нажмите **ОК**.
  4. Настройте параметры **Семейство шрифтов** и **Тема**.


Локатор точек показывается в конце Instant Experience. Когда человек нажмет на объявление, чтобы увидеть маршрут к торговой точке, на его телефоне откроется приложение для работы с картами, которое используется по умолчанию.
## Расположение компонентов
Чтобы изменить порядок компонентов, нажмите на левую часть названия компонента и перетащите его в нужное место Instant Experience. Чтобы свернуть информацию о компонентах, нажмите на стрелку справа от каждого из них.
Большинство компонентов можно разместить в любом порядке. Заголовок всегда расположен в верхней части Instant Experience, а локатор точек — в нижней.
Чтобы увеличить расстояние между компонентами:
  1.   2. Выберите **Показать расширенные настройки**.
  3. Прокрутите до пункта **Отбивка компонента**.
  4. Добавьте больше места в **верхней** или **нижней** части компонента.
  5. При необходимости повторите эти действия для каждого компонента.
