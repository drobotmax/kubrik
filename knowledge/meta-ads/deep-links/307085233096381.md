# Устранение проблем с кастомизацией креативов для мест размещения
Некоторые продукты и функции не поддерживают кастомизацию креативов для разных мест размещения. Ниже приведены проблемы, с которыми вы можете столкнуться. 
  * **Нельзя использовать одинаковые изображения и текст в одной кампании.** Если вы видите код ошибки, возможно, проблема в добавленном тексте, изображении или видео. Проверьте загруженные медиафайлы.
  * **Нельзя использовать некоторые цели.** Посмотрите [список поддерживаемых целей рекламы и мест получения конверсий](https://www.facebook.com/business/help/1001112308024751). 
  * **Нельзя использовать следующие форматы** : [объявления из каталога Advantage+](https://www.facebook.com/business/help/397103717129942), объявления, [созданные на основе существующих публикаций](https://www.facebook.com/business/help/374756690038600), и [гибкий формат рекламы](https://www.facebook.com/business/help/835561738423867). Реклама, созданная не в [Ads Manager](https://adsmanager.facebook.com/adsmanager/manage) (например, с помощью API), имеет некоторые ограничения.
  * **Нельзя менять объявления для разных типов устройств** , например использовать один креатив для мобильных устройств, а другой — для компьютера.
  * **Нельзя использовать следующие кнопки призыва к действию** : "Купить билеты" и "Сохранить" (даже если они совместимы с вашей целью).
  * **Нельзя использовать следующие функции** : [реклама с динамическими креативами](https://www.facebook.com/business/help/170372403538781), [таргетинг по местоположению группы точек](https://www.facebook.com/business/help/628699626139328) и [объявления с переходом в магазин](https://www.facebook.com/business/help/786101482615028). 
  * **Нельзя дублировать существующие реакции, комментарии и репосты.** При дублировании рекламы с кастомизацией креативов для мест размещения эта функция недоступна.
  * **Нельзя редактировать текст объявления.** Текст может быть извлечен с места назначения "URL сайта", если вы: 
    * оставите текстовое поле в вашем рекламном креативе пустым.
    * используете [существующий пост со Страницы Facebook](https://www.facebook.com/business/help/347839548598012) для создания рекламы.

## Будьте в курсе новостей от Meta for Business
Укажите электронный адрес, на который вы хотите получать новости о мероприятиях и продуктах, а также другую информацию от Meta for Business. 
Отправляя эту форму, вы соглашаетесь получать по электронной почте маркетинговые материалы от Meta, в том числе новости, информацию о мероприятиях, обновления и специальные предложения. Вы можете отказаться от рассылки в любой момент. Для этого нужно, например, нажать ссылку для отмены подписки в нашем электронном письме.
