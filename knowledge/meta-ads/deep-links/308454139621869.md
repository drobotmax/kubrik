# Интеграция пикселя Meta и SDK для мобильных устройств для туристической рекламы
Чтобы показывать туристическую рекламу посетителям своего сайта или пользователям мобильного приложения, вам потребуется добавить на сайт пиксель Meta и интегрировать Facebook SDK в приложение. Пиксель и SDK позволяют измерять действия (например, бронирования и посещения страниц), совершаемые клиентами на сайте, а также выполнять оптимизацию для них.
## Настройка пикселя
Если вы хотите показывать туристическую рекламу, но прежде не настраивали пиксель, вам нужно его создать и добавить на свой сайт. Для этого воспользуйтесь [руководством по установке пикселя Meta](https://www.facebook.com/business/m/pixel-setup-get-started). 
Установив пиксель, вы сможете настроить его на ключевых страницах нижнего уровня воронки согласно целям своей рекламы. Инструкция приведена ниже. Если пиксель уже установлен, эта статья поможет вам настроить его для туристической рекламы. 
## Настройка пикселя для туристической рекламы
**установить код на все страницы сайта** , а также совместно с веб-разработчиком добавить в пиксель дополнительную строку кода для отслеживания 4 следующих событий: 
  1. Search (для отслеживания интересов). 
  2. ViewContent (для отслеживания просмотренных позиций). 
  3. InitiateCheckout (для отслеживания намерения совершить покупку). 
  4. Purchase (для отслеживания конверсий).


Эти события должны сопровождаться определенными параметрами, которые предоставляют дополнительную полезную информацию. Обязательные и рекомендуемые параметры приведены по ссылкам ниже. Ознакомиться с полным списком параметров и дополнительных требований к форматированию можно на сайте [Meta for Developers](https://developers.facebook.com/docs/marketing-api/travel-ads).
Добавьте соответствующие стандартные события на каждую страницу. Стандартные события нужно указывать точно так, как показано в ссылках ниже, без изменения правописания и регистра. Убедитесь, что вы выбрали подходящие стандартные события для вашего типа ассортимента:
  * [Стандартные параметры пикселя для рекламы гостиниц](https://www.facebook.com/business/help/661142797646824).
  * [Стандартные параметры пикселя для рекламы перелетов](https://www.facebook.com/business/help/2522836671061789).
  * [Стандартные параметры пикселя для рекламы туристических направлений](https://www.facebook.com/business/help/2215188722143939).


ID контента в пикселе должен совпадать со значением в столбце ID в каталоге. Также можно отправлять по несколько ID товаров с одной страницы путем отправки массива данных.

## Работа с партнером по управлению тегами
Многие поставщики услуг по управлению тегами интегрировали свои системы с Meta. Благодаря этому их клиентам проще создавать пользовательские аудитории на основе посетителей сайтов и аудитории туристической рекламы. Обратитесь напрямую к своему партнеру по управлению тегами, чтобы узнать, как настроить тег для показа туристической рекламы. Вы также можете добавить код пикселя прямо на свой сайт.
## Интеграция Facebook SDK для мобильного приложения
**Начало работы с Facebook SDK**
Чтобы использовать Facebook SDK для iOS, Android или JavaScript (на веб-платформах), вам потребуется ID приложения Facebook. Если у вас уже есть [приложение](https://developers.facebook.com/apps), можете использовать его. Если нет, создайте его с помощью [этого руководства](https://developers.facebook.com/quickstarts). Если вам нужна помощь, следуйте инструкциям из нашего [руководства по регистрации и настройке приложения](https://developers.facebook.com/docs/apps/register). 
Вам потребуется добавить SDK в свое приложение. [Подробнее](https://developers.facebook.com/docs/app-ads/sdk). 
Прежде чем добавить SDK в приложение, ознакомьтесь с разделом ниже, чтобы также интегрировать события в приложении, необходимые для туристической рекламы. 
**Настройка событий в приложении на ключевых страницах нижних уровней воронки**
Для динамического ретаргетинга рекламы актуальных товаров на пользователей вашего приложения вам потребуется добавить в него те же 4 обязательных события.  
| Событие  | Событие в приложении  |  
| --- | --- |  
| Search  | fb_mobile_search  |  
| ViewContent  | fb_mobile_content_view  |  
| InitiateCheckout  | fb_mobile_initiated_checkout  |  
| Purchase  | fb_mobile_purchase  |  
Дополнительную информацию об этих 4 событиях можно найти на [сайте для разработчиков](https://developers.facebook.com/docs/marketing-api/dynamic-product-ads/product-audiences/v2.4#setupappevents).

## Будьте в курсе новостей от Meta for Business
Укажите электронный адрес, на который вы хотите получать новости о мероприятиях и продуктах, а также другую информацию от Meta for Business. 
Отправляя эту форму, вы соглашаетесь получать по электронной почте маркетинговые материалы от Meta, в том числе новости, информацию о мероприятиях, обновления и специальные предложения. Вы можете отказаться от рассылки в любой момент. Для этого нужно, например, нажать ссылку для отмены подписки в нашем электронном письме. Узнать, как Meta обрабатывает вашу информацию, можно в [Политике использования данных](https://www.facebook.com/about/privacy).
