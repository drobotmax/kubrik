# Рекомендации для кампаний поставщиков туристических пакетов
Туристическая реклама позволяет настроить каталог авиарейсов, гостиниц или туристических направлений. Однако рекламодатели, продвигающие туристические пакеты в связке "авиарейс-гостиница" или "гостиница-аренда автомобиля", могут настроить уникальную кампанию, чтобы найти больше людей, которым интересны их предложения.
## Определение бизнес-цели 
  * Ретаргетировать рекламу на посетителей сайта или пользователей приложения. Мы рекомендуем использовать рекламу гостиниц для ретаргетинга и создать каталог гостиниц. Это позволит более точно настроить цены и аудитории.
  * Настроить таргетинг на широкую аудиторию, которая выражает разные уровни намерения забронировать путешествие. Мы рекомендуем использовать рекламу туристических направлений для привлечения клиентов и создать каталог туристических направлений. Это позволит вам расширить аудиторию. 


**Примечание**. Если вы не хотите сразу создавать и каталог гостиниц, и каталог туристических направлений, настройте сначала каталог гостиниц и задайте параметры, а затем протестируйте рекламу гостиниц для ретаргетинга и для привлечения клиентов. Так вы начнете создавать кампании для продвижения туристических пакетов. Вы можете [создать кампанию в Meta Ads Manager](https://www.facebook.com/business/help/1294482487347184?id=221593351918232) с помощью инструкций ниже или обратиться за помощью к бизнес-партнеру Meta.
## Настройка пикселя Meta для продвижения туристических пакетов
Мы рекомендуем настроить пиксель, чтобы передавать обратно все стандартные события, а также параметры **гостиниц** и **туристических направлений**. Если вы ещё не создали пиксель, узнайте, [как интегрировать пиксель и SDK для мобильных устройств](https://www.facebook.com/business/help/308454139621869) для туристической рекламы. Если у вас уже есть пиксель, обновите его, добавив события ниже.   
|   | **Гостиницы и туристические направления**  |  
| --- | --- |  
| **Требуемые события**  | 
  * Search 
  * ViewContent 
  * InitiateCheckout (если применимо)
  * Purchase (если применимо) 

 |  
| **Требуемые параметры**  | 
  * content_ids= ["example_hotel_id","example_destination_id"]
  * city (только для события Search)
  * region (только для события Search)
  * country (только для события Search)
  * value (только для события Purchase, если применимо)
  * currency (только для события Purchase, если применимо)

 |  
## Настройка фида каталога
Для туристических пакетов есть 6 переменных, которые необходимы для точных данных о туристическом пакете и цене: аэропорт отправления, гостиница, время отправления, длительность поездки, количество комнат и количество гостей.
Чтобы в точности указать все 6 переменных, вам придется заполнить несколько полей в каталогах **гостиниц** и **туристических направлений** иначе, чем при основной настройке:
  * Content ID
    * Для этого создайте уникальный ID контента, используя аббревиатуру аэропорта отправления и ID гостиницы, например MAN|017719 (аэропорт Манчестера + ID гостиницы 017719).
  * Num_children и Checkin_date
    * Используйте поле Num_children
    * Используйте поле Checkin_date


**Пример настройки каталога и пикселя**
Аэропорт отправления — Манчестер (MAN)
Content ID — MAN|017719 (аэропорт + ID гостиницы) 
Num_children — 71114 (обозначает длительность пребывания 7, 11, 14)
Checkin_date
**Другие примеры создания составного ID контента**
Аренда авто:
  * Откуда забрать/куда вернуть — Аэропорт Малаги (MAL)
  * Класс автомобиля — F
Content ID = MALMALF 


## Настройка динамического URL
Когда вы создаете шаблон объявления, настройте динамические глубокие ссылки, используя те же поля с измененным назначением:
  * Checkin_date = дата отправления 
  * Num_children = длительность поездки 


## Структура группы товаров
Группы товаров следует упорядочивать по аэропорту отправления. Это позволит убедиться в отсутствии пересечения с другими аэропортами поблизости. Если этого не сделать, ближайший аэропорт отправления может предложить рейс в связке с той же гостиницей, что приведет к дублированию изображений в кольцевой галерее.
