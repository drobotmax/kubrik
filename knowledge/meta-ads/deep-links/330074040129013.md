# О максимальной ценности конверсий с целью "Продвижение приложения"
Вы можете использовать кампании с целью "Продвижение приложения" с оптимизацией для ценных конверсий, которые помогут достичь высокой рентабельности инвестиций в рекламу (ROAS). При выборе цели по результативности **Максимальная ценность конверсий** в Meta Ads Manager система показа рекламы отдает приоритет людям, которые с наибольшей вероятностью совершат ценные действия в вашем приложении, например покупку или просмотр рекламы.
Эта цель подходит для тех, кто хочет сосредоточиться на прибыльности и ценности, а не просто на количестве конверсий. Она совместима с целями по ROAS для повышения эффективности.
## События конверсии
При использовании цели **Максимальная ценность конверсий** можно выбрать одно из следующих событий конверсии:
  * **Покупка в приложении.** Оптимизация для пользователей, которые с наибольшей вероятностью совершат покупку в приложении. Рекомендуется для приложений, которые генерируют доход за счет прямых покупок, подписок или других транзакций.  
**Примечание.** Доступно для iOS и Android.
  * **Показ рекламы в приложении.** Оптимизация для пользователей, которые с наибольшей вероятностью просмотрят вашу рекламу в приложении. Рекомендуется для приложений с рекламной монетизацией. Это, например, видео с вознаграждением или межстраничные объявления в играх.  
**Примечание.** Доступно только для Android.


## Преимущества использования цели "Максимальная ценность конверсий"
  * **Приоритетность ценных конверсий.** Система показа рекламы выполняет оптимизацию для конверсий с высоким показателем ROAS, а не просто для максимального числа действий.
  * **Оптимизация для определенных событий.** Выберите событие конверсии **Покупка в приложении** или **Показ рекламы в приложении** (в зависимости от модели монетизации приложения).
  * **Поиск самых ценных клиентов.** Система отдает приоритет пользователям и событиям, которые приносят максимальный доход в рамках рекламного бюджета.
