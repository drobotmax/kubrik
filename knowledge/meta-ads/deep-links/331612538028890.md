# Охват пользователей устройств с iOS 14.5 и более поздними версиями с помощью рекламы на платформах Meta
Узнайте, как создавать рекламу на платформах Meta для охвата пользователей устройств с iOS 14.5 и более поздними версиями. В этом руководстве приведена информация об особенностях кампаний для конверсий на сайте, кампаний для повторного вовлечения в приложении и кампаний для продвижения приложений.
## Основные особенности
  * **Настройка**. Вам не требуется выполнять дополнительные действия, чтобы использовать протокол "Измерение на основе агрегированных данных о событиях" в кампаниях для конверсий на сайте.
  * **Таргетинг**. По мере того, как всё больше пользователей будут обновлять устройства до iOS 14 и более поздних версий, будет уменьшаться размер пользовательских аудиторий на основе действий в приложении, данных с сайта и данных о подключении к приложению. Рекомендуем рассмотреть альтернативные стратегии — настройку [широкой аудитории](https://www.facebook.com/business/help/308474373366888) или [расширение таргетинга](https://www.facebook.com/business/help/128066880933676).


##  Создание кампаний для повторного вовлечения в приложении
Чтобы создавать кампании для возобновления вовлеченности в приложении, направленные на пользователей устройств с iOS 14.5 и более поздними версиями, используйте протокол "Измерение на основе агрегированных данных о событиях". Сейчас этот протокол доступен для кампаний с местом назначения "Мобильное приложение" и целями "Продажи", "Лиды" или "Вовлеченность". [Подробнее о протоколе](https://www.facebook.com/business/help/721422165168355).
## Основные особенности
  * **Настройка**. [Убедитесь, что приложение соответствует требованиям протокола "Измерение на основе агрегированных данных о событиях"](https://www.facebook.com/business/help/617021863580582). Если требования выполнены, вы можете запускать кампании для пользователей устройств с iOS 14+.
  * **Таргетинг**. Как и в кампаниях для конверсий на сайте, по мере обновления устройств может уменьшаться размер аудиторий. Рекомендуем рассмотреть настройку [широкой аудитории](https://www.facebook.com/business/help/308474373366888) или [расширение таргетинга](https://www.facebook.com/business/help/128066880933676).
  * **Настройки атрибуции**. Для определенных типов кампаний будут доступны не все настройки атрибуции. [Подробнее](https://www.facebook.com/business/help/460276478298895).
  * **Отчетность**. Отчеты по кампаниям для iOS 14+ могут отличаться от отчетов по другим типам кампаний. [Подробнее об отчетности о событиях в приложении, полученной через протокол "Измерение на основе агрегированных данных"](https://www.facebook.com/business/help/3499572073626051).


## Создание кампаний с целью "Продвижение приложения"
Чтобы создавать кампании с целью "Продвижение приложения" для пользователей устройств с iOS 14.5 и более поздними версиями (другое название — кампании для iOS 14+), используйте SKAdNetwork от Apple или протокол "Измерение на основе агрегированных данных о событиях" для атрибуции. [Подробнее о протоколе "Измерение на основе агрегированных данных"](https://www.facebook.com/business/help/721422165168355) и [настройке SKAdNetwork от Apple в Meta Events Manager](https://www.facebook.com/business/help/188126096313109).
## Основные особенности
  * **Настройка** :
    * Чтобы использовать протокол "Измерение на основе агрегированных данных о событиях" в кампаниях для iOS 14+ с целью "Продвижение приложения", [сначала убедитесь, что ваше приложение соответствует требованиям протокола](https://www.facebook.com/business/help/617021863580582). Если требования выполнены, вы можете запускать такие кампании.
    * Чтобы использовать SKAdNetwork в кампаниях для iOS 14+ с целью "Продвижение приложения", [сначала подготовьте приложение к настройке SKAdNetwork](https://www.facebook.com/business/help/2750680505215705). Затем вы сможете [настроить SKAdNetwork от Apple](https://www.facebook.com/business/help/670955636925518).
  * **Рекламные аккаунты и кампании**. Чтобы создать кампанию для iOS 14+, сначала вам может потребоваться внести несколько изменений в свои рекламные аккаунты, кампании и группы объявлений. [Подробнее о работе с рекламными аккаунтами для запуска кампаний, направленных на пользователей устройств с iOS 14.5 и более поздними версиями](https://www.facebook.com/business/help/651033805513936).
  * **Таргетинг**. По мере обновления устройств может уменьшаться размер аудиторий. Рекомендуем рассмотреть настройку [широкой аудитории](https://www.facebook.com/business/help/308474373366888) или [расширение таргетинга](https://www.facebook.com/business/help/128066880933676).
  * **Настройки атрибуции**. Для некоторых типов кампаний будут доступны не все настройки атрибуции. [Подробнее](https://www.facebook.com/business/help/460276478298895).
  * **Отчетность**. Отчеты по кампаниям для iOS 14+ могут отличаться от отчетов по другим типам кампаний. [Подробнее об отчетности о событиях в приложении, полученной через протокол "Измерение на основе агрегированных данных"](https://www.facebook.com/business/help/3499572073626051), и [отчетности Meta Ads Manager для SKAdNetwork от Apple](https://www.facebook.com/business/help/854500742637772).
