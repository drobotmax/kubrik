# Переключение между рекламным аккаунтом и бизнес-портфолио в Meta Events Manager
С помощью Events Manager можно измерять и оптимизировать события на сайте, в приложении или офлайн при размещении рекламы на платформах Meta.
Вы можете использовать Events Manager через рекламный аккаунт или бизнес-портфолио. При этом рекламный аккаунт может не принадлежать бизнес-портфолио. Важно понимать, какой аккаунт вы используете, поскольку некоторые функции доступны только для определенных типов аккаунтов.
Рекламным аккаунтам и бизнес-портфолио присваиваются уникальные ID. Чтобы проверить, какой тип аккаунта вы используете:
  1. Откройте [Events Manager](https://www.facebook.com/events_manager2).
  2. Посмотрите на URL в адресной строке и определите, какой ID он содержит: рекламного аккаунта, бизнес-портфолио или обоих. Если это ID рекламного аккаунта, в URL вы увидите фрагмент act=, а если это ID бизнес-портфолио, в URL будет фрагмент business_id=.


Например, следующий URL показывает только ID рекламного аккаунта:
![Пример URL только с ID рекламного аккаунта](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=247281743392949&version=1773385136&transcode_extension=webp)
Если URL содержит:
  * **Только ID бизнес-портфолио** —вы используете Events Manager через бизнес-портфолио.
  * **Только ID рекламного аккаунта** — вы используете Events Manager через рекламный аккаунт, не связанный с бизнес-портфолио.
  * **ID рекламного аккаунта и ID бизнес-портфолио** —вы используете Events Manager через рекламный аккаунт из бизнес-портфолио. Примечание. Бизнес-портфолио, ID которого указан в URL, может не владеть рекламный аккаунт, ID которого указан там же. Подробнее о том, [как найти ID бизнес-портфолио](https://www.facebook.com/business/help/1181250022022158).


Если вы хотите переключиться на другой рекламный аккаунт, выберите его из раскрывающегося меню в правом верхнем углу окна Events Manager. Если вы не видите нужный рекламный аккаунт, переключитесь на другое бизнес-портфолио с помощью раскрывающегося меню в левом верхнем углу окна Events Manager.
