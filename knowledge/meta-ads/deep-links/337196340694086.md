# Разница в подсчете событий в Meta Ads Manager, Отчетах по рекламе и Events Manager
Количество и типы событий в [Meta Ads Manager](https://www.facebook.com/business/help/200000840044554) и [Отчетах по рекламе Meta](https://www.facebook.com/business/help/487269218011981) могут отличаться от аналогичных данных в [Meta Events Manager](https://www.facebook.com/business/help/898185560232180). В Events Manager также можно увидеть разницу в количестве событий на страницах **Источники данных** и **Специально настроенные конверсии**. Расхождения связаны с тем, что эти инструменты предназначены для разных целей.
Ниже представлена сравнительная характеристика событий в этих трех инструментах.
![](https://scontent-prg1-1.xx.fbcdn.net/v/t39.8562-6/146597050_455534749152435_1802453867954835002_n.svg?_nc_cat=1&ccb=1-7&_nc_sid=f537c7&_nc_ohc=aXVV8xvJ_GcQ7kNvwHlDc6r&_nc_oc=AdpxRCyIaZgWtCiyYVpouqc9iu5YqUDJMwrDTBJEFNEJZUEPEvWvXOLbI_2mhUNO9Is&_nc_zt=14&_nc_ht=scontent-prg1-1.xx&_nc_gid=NTLER37PYOle8KSFn8spIA&_nc_ss=7a389&oh=00_AfwVw8xsXntg3yckrbXhTxYA2yCUldDXOiggLyhkWmZK9g&oe=69CF18D8)
### Типы событий
**Ads Manager**
  * Стандартные события.
  * Специально настроенные конверсии.


**Отчеты по рекламе**
  * Стандартные события, сгруппированные в разделе "Конверсии".
  * Специально настроенные конверсии.
  * Специально настроенные события, использованные ранее для оптимизации показа рекламы.


**Events Manager**
  * Стандартные события.
  * Специально настроенные события.
  * Специально настроенные конверсии.

### Атрибуция
**Ads Manager и Отчеты по рекламе**
Ads Manager и Отчеты по рекламе показывают события, атрибутированные человеку, который видел рекламу. При этом не учитываются события, которые дублируются либо нарушают правила или законы в отношении данных. [Подробнее о действиях, атрибутированных рекламе.](https://www.facebook.com/business/help/458681590974355)
**Events Manager**
Events Manager показывает как атрибутированные, так и неатрибутированные события. Результаты включают в себя органический трафик, события, атрибутированные рекламе на платформах Meta, и трафик с других платформ. После этого инструмент сбрасывает события, которые дублируются либо нарушают правила или законы в отношении данных.
Например, в отчетах Events Manager учитываются события от людей, которым не показывалась ваша реклама или которые не используют платформы Meta.
![](https://scontent-prg1-1.xx.fbcdn.net/v/t39.8562-6/146597050_455534749152435_1802453867954835002_n.svg?_nc_cat=1&ccb=1-7&_nc_sid=f537c7&_nc_ohc=aXVV8xvJ_GcQ7kNvwHlDc6r&_nc_oc=AdpxRCyIaZgWtCiyYVpouqc9iu5YqUDJMwrDTBJEFNEJZUEPEvWvXOLbI_2mhUNO9Is&_nc_zt=14&_nc_ht=scontent-prg1-1.xx&_nc_gid=NTLER37PYOle8KSFn8spIA&_nc_ss=7a389&oh=00_AfwVw8xsXntg3yckrbXhTxYA2yCUldDXOiggLyhkWmZK9g&oe=69CF18D8)
### Дедупликация
**Ads Manager и Отчеты по рекламе**
Мы выполняем дедупликацию событий.
Например, если за короткий период времени мы несколько раз получаем одно и то же событие для одного человека, мы учитываем одно из них и сбрасываем остальные, чтобы избежать ошибок в подсчетах.
**Events Manager**
Страница "Источники событий":
  * При подсчете **общего** количества события не дедуплицируются.
  * Например, в отчетах Events Manager может учитываться одно действие, выполненное человеком несколько раз.


Страница "Специально настроенные конверсии":
  * При подсчете **общего** количества специально настроенные конверсии дедуплицируются.


Ниже приведены распространенные различия между данными в Ads Manager, Отчетах по рекламе и Events Manager.
  * **Ads Manager и Отчеты по рекламе показывают меньше результатов, чем Events Manager.** Возможные причины:
    * Events Manager показывает действия людей, которые взаимодействуют с вашей компанией. Ads Manager и Отчеты по рекламе показывают действия людей, которые взаимодействуют с вашей компанией в результате просмотра вашей рекламы.
    * Кроме того, Events Manager показывает большинство полученных действий, а Ads Manager и Отчеты по рекламе — только атрибутированные действия.
  * **Количество специально настроенных конверсий в Events Manager меньше, чем число стандартных и специально настроенных событий.** При подсчете стандартных и специально настроенных событий учитывается большинство полученных событий. При подсчете специально настроенных конверсий не учитываются конверсии, которые дублируются либо нарушают правила или законы в отношении данных.
