# Настройка бюджета кампании Advantage+
[Бюджет кампании Advantage+](https://www.facebook.com/business/help/153514848493595) (другие названия — **Бюджет кампании** или **Бюджет** со включенным параметром Advantage+) позволяет задавать основной бюджет кампании. Система будет непрерывно распределять этот бюджет в режиме реального времени среди самых перспективных групп объявлений в течение всего срока действия кампании. Настроить этот параметр можно на этапе [_создания_](https://www.facebook.com/business/help/621956575422138) или [_редактирования_](https://www.facebook.com/business/help/2169779963333459) кампании.
## Настройка бюджета кампании Advantage+
Чтобы настроить бюджет кампании Advantage+ во время ее создания:
  1. Активируйте переключатель **Бюджет кампании Advantage+**. **Примечание.** Бюджет кампании Advantage+ по умолчанию включен для кампаний с целью "Продажи", "Лиды" и "Продвижение приложения".
  2. Выберите тип бюджета кампании: [_дневной_](https://www.facebook.com/business/help/190490051321426) или [_на весь срок действия_](https://www.facebook.com/business/help/1844835042445690).
  3. Задайте сумму бюджета.
  4. Выберите для кампании [_стратегию ставок_](https://www.facebook.com/business/help/1619591734742116).
  5. Если вы выбрали бюджет на весь срок действия, то можете настроить график показа рекламы для каждой группы объявлений.
  6. Нажмите **Далее**.
  7. В разделе **Бюджет и график** выберите дату начала показа.
  8. Завершите настройку группы объявлений для вашего бюджета.


**Примечание.** Между включением и выключением бюджета кампании Advantage+ должно пройти не менее 2 часов. Если вам нужно изменить настройки быстрее, приостановите кампанию, создайте новую и задайте в ней бюджеты для групп объявлений.
## Доступ к использованию бюджета кампании Advantage+
Этот параметр доступен кампаниям с:
  * Одинаковым типом бюджета для всех групп объявлений (дневным или на весь срок действия).
  * Одинаковой стратегией ставок для всех групп объявлений ("Максимальное количество", "Предельная ставка" или "Цель по цене за результат").
  * Одинаковым [_событием оптимизации показа_](https://www.facebook.com/business/help/139628083350822) для всех групп объявлений при стратегии ставок "Максимальное количество".
  * Стандартным методом показа для всех групп объявлений.


**Примечание**. Бюджет кампании Advantage+ позволяет добавлять в кампанию более 70, но менее 200 групп объявлений. Однако при таком количестве групп объявлений ваша возможность вносить правки после публикации рекламы будет ограничена. Подробнее о том, [_как использовать бюджет Advantage+ для кампании, содержащей более 70 групп объявлений_](https://www.facebook.com/business/help/519856662172206).
