# Создание A/B-тестирования с помощью инструмента "Эксперименты"
![](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=438223955063449&version=1666716831&transcode_extension=webp)
Начиная с октября 2023 года тип закупки "Охват и частота" будет называться "Резервирование". [_Подробнее о резервировании_](https://www.facebook.com/business/help/251123081984768).
В инструменте [Эксперименты](https://www.facebook.com/experiments) можно создать [A/B-тестирование](https://www.facebook.com/business/help/1738164643098669), чтобы сравнить несколько рекламных кампаний и узнать, какая из них наиболее эффективна для достижения выбранных бизнес-целей. Для тестирования можно использовать две или больше существующих кампаний либо продублировать одну текущую кампанию.
Инструмент "Эксперименты" стоит использовать для проведения A/B-тестирования, если:
  * У вас есть активные рекламные кампании, которые вы хотите сравнить.
  * Вы хотите задать для каждой кампании особые параметры перед проведением тестирования.
  * Вы хотите выбрать другие метрики в дополнение к основной. 


Чтобы создать такое тестирование:
  1. Откройте инструмент [Экспериментальные функции](https://www.facebook.com/experiments).
  2. Выберите **Создать тест** на панели инструментов в левой части экрана.
  3. Найдите раздел **A/B-тестирование** и нажмите **Начать настройку**.
  4. Выберите способ создания A/B-тестирования. Чтобы использовать несколько существующих рекламных кампаний, нажмите **Тестировать другую кампанию** , а чтобы создать копию текущей кампании — **Дублировать имеющуюся кампанию**.
  5. Если в рамках A/B-тестирования вы хотите сравнить несколько существующих рекламных кампаний, то выберите до 5 из них. 
  6. Задайте даты начала и окончания тестирования. Они могут отличаться от дат начала и окончания показа участвующих рекламных кампаний. Однако кампании должны быть активными в течение всего срока действия A/B-тестирования или времени, достаточного для получения надежных результатов.
  7. Введите название A/B-тестирования.
  8. Укажите, на основе чего нужно определять наиболее эффективную тестируемую кампанию. 
  9. Выберите **Просмотреть информацию о тесте** , чтобы увидеть параметры и график тестирования.
  10. Нажмите **Создать тест** , чтобы запустить его.


Если вы не уверены, какую переменную, бюджет или график выбрать для тестирования, ознакомьтесь с [рекомендациями по созданию A/B-тестирования](https://www.facebook.com/business/help/290009911394576).
