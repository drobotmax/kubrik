# Добавление вопроса типа "Запрос на встречу" в рекламу для сбора лидов с моментальной формой
При создании [_рекламы для сбора лидов с моментальной формой_](https://www.facebook.com/business/help/761812391313386) можно добавить вопрос типа "Запрос на встречу". В ответе на него люди смогут указать, в какой день и время они хотели бы связаться с вашей компанией или встретиться с вами.
Эта информация позволит гибче взаимодействовать с лидами и улучшить качество обслуживания клиентов. Зная ее, вы сможете связаться с человеком в удобное для него время или перенаправить его к нужному сотруднику из своей команды.
**Примечание**. Если добавить вопрос типа "Запрос на встречу", ваше объявление будет показываться только в мобильном приложении Facebook.
##  **Чтобы добавить в моментальную форму вопрос типа "Запрос на встречу"** : 
  1. Начните [_создавать моментальную форму_](https://www.facebook.com/business/help/375478503258484) для рекламы для сбора лидов.
  2. Во всплывающем окне моментальной формы нажмите **Вопросы**.
  3. Нажмите **Добавить вопрос**.
  4. В раскрывающемся меню выберите **Запрос на встречу**.
  5. Добавьте вопрос о предпочитаемом времени и дате для встречи или связи с клиентом другим способом. Вы можете включить **сообщение с подтверждением** , которое появится под этим вопросом.
  6. Завершите настройку моментальной формы.


**Примечание**. По умолчанию указано, что вы работаете с понедельника по воскресенье с 7:00 до 22:00. Пока этот график нельзя изменить.
Когда реклама будет одобрена, все люди смогут увидеть вопрос типа "Запрос на встречу" в моментальной форме, а также выбрать определенную дату и время для связи с вами.
