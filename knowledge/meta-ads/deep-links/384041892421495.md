# Способы добавления товаров в каталог
Есть несколько способов добавить товары в каталог и управлять ими в Commerce Manager. Эти объекты называются источниками данных, потому что они пополняют ваш каталог данными о товарах.
## Типы источников данных для каталога
Добавить товары в каталог можно следующими способами:
  * [Вручную.](https://www.facebook.com/business/help/1670875312991238) Добавляйте и редактируйте товары вручную в Commerce Manager.
  * [С помощью фида данных.](https://www.facebook.com/business/help/125074381480892) Загрузите электронную таблицу или файл XML с данными о товарах. Вы можете настроить запланированную загрузку (ежечасно, ежедневно или еженедельно).
  * [Через партнерскую платформу.](https://www.facebook.com/business/help/365831587397584) Если ваш ассортимент размещен на партнерской платформе, которая интегрируется с Meta (например, [Shopify](https://www.facebook.com/business/help/1046957249463415) или [WooCommerce](https://www.facebook.com/business/help/4049935305295468)), вы можете синхронизировать позиции с каталогом. Продолжайте управлять товарами на партнерской платформе. Они будут автоматически синхронизироваться с Commerce Manager.
  * [Страница Facebook](https://www.facebook.com/business/help/1573601244287366) (сейчас доступно только для английского, тайского и вьетнамского языков). Meta использует ИИ для обнаружения товаров в публикациях или объявлениях на вашей Странице и автоматически добавляет их в каталог.
  * [С помощью сайта](https://www.facebook.com/business/help/1557271764939373) (сейчас доступно только для соответствующих требованиям компаний). Meta автоматически синхронизирует товары с каталогом и регулярно их обновляет с помощью сканирования сайта краулером. 
  * [С помощью Catalog Batch API](https://developers.facebook.com/docs/marketing-api/catalog-batch). Это более сложный способ, который позволяет разработчикам добавлять и обновлять товары через вызов API. 
  * [С помощью пикселя Meta.](https://www.facebook.com/business/help/887775018036966) Вы можете автоматически добавлять и обновлять товары с сайта. Возможно, вам потребуется помощь разработчика для установки тегов микроданных на сайт.


На вкладке каталога [Товары](https://www.facebook.com/business/help/1406482412767183) можно **просмотреть** добавленные позиции, а на вкладке Commerce Manager **Источники данных** — подключенные источники данных.

## Добавление дополнительных источников данных
В Commerce Manager можно добавить другие типы источников данных, например:
  * [Фиды для стран и языков.](https://www.facebook.com/business/help/2144286692311411) Предоставьте локализованные данные о товарах, например цены в разных валютах и описания на разных языках.
  * [Дополнительные фиды.](https://www.facebook.com/business/help/609872007120910) Добавляйте или обновляйте данные о товарах в основном источнике данных, например указывайте дополнительные свойства. С помощью дополнительных фидов можно обновлять сведения о товарах, но нельзя добавлять и удалять товары.


## Настройка нескольких источников данных (расширенный вариант)
Если вы используете несколько источников данных для обновления одного товара, то можете [указать, из какого источника будут обновляться определенные свойства](https://www.facebook.com/business/help/483704294496556). Например, изображения товаров можно синхронизировать с партнерской платформой, а цены — с фидом данных. Это расширенная функция, позволяющая компаниям создавать более сложные настройки каталогов.
