# О шлюзе Conversions API
Шлюз Conversions API — это способ самостоятельной настройки, который позволит легко и быстро подключить Conversions API без редактирования кода. Этот API поможет обеспечить прямые преимущества по результативности, такие как снижение цены за действие и оптимизация функций измерения. [Подробнее об Conversions API](https://www.facebook.com/business/help/2041148702652965?id=818859032317965).
Нажмите "Шлюз Conversions API" в разделе "Платформы для подключения источников данных" в Meta Events Manager, чтобы просто интегрировать Conversions API. Этот инструмент не требует использования сторонних партнерских интеграций или привлечения штатных разработчиков.
Настроив шлюз Conversions API, вы можете начать передавать Meta веб-события прямо со своего сервера. [Подробнее](https://developers.facebook.com/docs/marketing-api/conversions-api/guides/gateway/setup).
**В чем заключаются дополнительные преимущества шлюза Conversions API?**
  * **Скорость**. Процесс интеграции Conversions API сокращается от нескольких недель до получаса. 
  * **Стоимость**. Интеграция Conversions API с помощью шлюза стоит меньше, поскольку предусматривает не так много технических ресурсов и требований. Поставщик услуг облачного хранилища взимает плату только за расходы, связанные со шлюзом Conversions API, которая составляет от 30 долл. США в месяц.
  * **Автоматическая дедупликация**. Шлюз Conversions API обеспечивает автоматическую дедупликацию между браузером и событиями сервера. [Подробнее о дедупликации событий пикселя Meta и Conversions API](https://www.facebook.com/business/help/823677331451951?id=1205376682832142).
  * **Минимальная техническая поддержка**. Вы можете самостоятельно настроить Conversions API и не привлекать к процессу разработчиков или специалистов из отдела ИТ.
  * **Низкая стоимость обслуживания**. В отличие от других способов прямой интеграции вручную, шлюз Conversions API будет автоматически обновляться при появлении новых доступных функций. Это позволит сократить расходы на постоянное обслуживание и со временем получить другие преимущества.
  * **Поддержка нескольких компаний**. Шлюз Conversions API для нескольких аккаунтов предоставляет возможность подключиться и управлять несколькими компаниями с помощью одной конфигурации Conversions API. [Подробнее о шлюзе Conversions API для нескольких аккаунтов](https://www.facebook.com/business/help/746511563926080).


**Как понять, что шлюз Conversions API подходит вашей компании?**
  * Вы уже используете пиксель Meta.
  * Вы ещё не отправляете веб-события через Conversions API.
  * Вы тратите больше 300 долл. США в месяц на кампании с оптимизацией для веб-событий.
  * Вы не сотрудничаете с партнером по электронной коммерции (например, Shopify, WooCommerce или BigCommerce).  
  
Если вы уже управляете пикселем Meta при помощи партнера по электронной коммерции и он предоставляет услуги по упрощенной интеграции Conversions API, обратитесь за помощью к этому партнеру. [Подробнее о вариантах настройки Conversions API](https://www.facebook.com/business/help/433493041367251).


Сейчас шлюз Conversions API доступен компаниям, которые используют платформу Amazon Web Services (AWS) или Google Cloud (GCP). Компании не обязательно иметь существующий аккаунт на платформе AWS или GCP, поскольку создание нового занимает всего несколько минут.
**Примечание**. Затраты на облачное хранилище могут отличаться в зависимости от региона, условий использования и объема трафика от пикселя Meta.
