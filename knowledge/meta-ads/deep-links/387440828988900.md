# Основные понятия, связанные с протоколом "Измерение на основе агрегированных данных о событиях" и SKAdNetwork
Требования iOS 14 от Apple влияют на процессы получения и обработки событий конверсий, регистрируемых такими инструментами, как пиксель Meta, Conversions API и Facebook SDK для iOS. События — это действия, которые клиенты выполняют на сайте, в мобильном приложении или в обычном магазине. Например, когда кто-то совершает покупку на вашем сайте, пиксель Meta или Conversions API регистрирует это действие как событие покупки. Подробнее о том, [как изменения в iOS 14 от Apple могут повлиять на показ вашей рекламы](https://www.facebook.com/business/help/331612538028890).
Политика Apple запрещает приложениям собирать определенные данные и делиться ими, если люди не предоставили разрешение ATT. Поэтому возможности персонализации рекламы и отчетности об ее результативности могут быть ограничены (как в приложениях, так и на сайтах). Если вы используете такие инструменты Meta для бизнеса, как [пиксель Meta](https://www.facebook.com/business/help/742478679120153), Facebook SDK для iOS или App Events API, ознакомьтесь с разделами ниже. Вы узнаете, что нужно изменить в настройке веб-событий и событий в приложении, чтобы показывать рекламу пользователям устройств с iOS 14.5 и более поздними версиями.
  * **Измерение на основе агрегированных данных о событиях**. 
    * [Подробнее о протоколе](https://www.facebook.com/business/help/721422165168355).
    * Подробнее о том, [как проверить, соответствует ли ваше приложение требованиям протокола "Измерение на основе агрегированных данных о событиях"](https://www.facebook.com/business/help/617021863580582).
    * Подробнее о том, [как подтвердить глубокую ссылку на приложение для протокола "Измерение на основе агрегированных данных о событиях"](https://www.facebook.com/business/help/1468510370209420).
  * **SKAdNetwork API от Apple для кампаний с целью "Продвижение приложения"**. SKAdNetwork API от Apple ограничивает приложения до 63 событий. Настройте события в приложении для SKAdNetwork от Apple в Events Manager, чтобы измерять и оптимизировать рекламу для ценности, событий в приложении и установок мобильного приложения. 
    * [Подробнее о SKAdNetwork API от Apple](https://www.facebook.com/business/help/188126096313109).
    * Подробнее о том, [как подготовить приложение к настройке SKAdNetwork](https://www.facebook.com/business/help/2750680505215705).
    * Подробнее о том, [как настроить SKAdNetwork](https://www.facebook.com/business/help/670955636925518).
  * **Оптимизация для ценности**. Если вы используете SKAdNetwork от Apple и хотите продолжить показывать рекламу с оптимизацией для ценности и таргетингом на пользователей устройств с iOS 14.5 и более поздними версиями, настройте группы значений ценности в конфигурации SKAdNetwork в Events Manager. 
    * [Подробнее о группах значений ценности](https://www.facebook.com/business/help/378857770047703).
    * [Подробнее об оптимизации для ценности](https://www.facebook.com/business/help/296463804090290).
    * Подробнее о том, [как настроить оптимизацию для ценности](https://www.facebook.com/business/help/437932930561569).
  * **Кампании для iOS 14+**. Это кампании с целью "Продвижение приложения", таргетированные на пользователей устройств с iOS 14.5 и более поздними версиями. [Подробнее об ограничениях кампаний для iOS 14+](https://www.facebook.com/business/help/331612538028890).
