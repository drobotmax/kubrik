# Обзор стандартных событий пикселя Meta
![](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=438223955063449&version=1666716831&transcode_extension=webp)
Вы можете получить пошаговые инструкции по настройке стандартных событий пикселя Meta от технического специалиста по решениям Meta. Узнайте, [как запланировать звонок со специалистом Meta](https://www.facebook.com/business/meta-pro-team/technical-pro?lead_source=%20sbg_2025_q4_ou_specialist_technical_helpcenter_xyz).
[События](https://www.facebook.com/business/help/964258670337005) — это действия, которые люди совершают на вашем сайте. Meta заранее определяет стандартные события для создания аудиторий, регистрации конверсий и выполнения оптимизации для них. Ниже приведен список стандартных событий для пикселя Meta. Полный список стандартных событий и их параметров можно найти на сайте [Meta for Developers](https://developers.facebook.com/docs/facebook-pixel/reference).
## Стандартные события  
| Действие на сайте  | Описание  | Код стандартного события  |  
| --- | --- | --- |  
| Добавление платежной информации  | Добавление платежной информации клиента при оформлении заказа (например, нажатие кнопки для сохранения информации для выставления счетов).  | fbq('track', 'AddPaymentInfo');  |  
| Добавление в корзину  | Добавление товара в корзину (например, нажатие кнопки "Добавить в корзину" на сайте).  | fbq('track', 'AddToCart');  |  
| Добавление в список желаний  |  Добавление товаров в список желаний (например, нажатие кнопки "Добавить в список желаний" на сайте).  | fbq('track', 'AddToWishlist');  |  
| Завершение регистрации  |  Предоставление информации в обмен на услугу, оказываемую вашей компанией (например, подписка на электронную рассылку).  | fbq('track', 'CompleteRegistration');  |  
| Контакт  |  Контакт между клиентом и вашей компанией по телефону, SMS, электронной почте, в чате или с помощью других средств.  | fbq('track', 'Contact');  |  
| Персонализация товара  |  Персонализация товаров с помощью инструмента конфигурации или другого приложения вашей компании.  | fbq('track', 'CustomizeProduct');  |  
| Пожертвование  |  Пожертвование средств в пользу вашей организации или инициативы.  | fbq('track', 'Donate');  |  
| Поиск места  |  Поиск в Интернете одной из ваших точек с намерением ее посетить (например, когда человек ищет товар и находит его в одном из ваших местных магазинов).  | fbq('track', 'FindLocation');  |  
| Начало оформления заказа  | Начало процесса оформления заказа (например, нажатие кнопки "Оформить заказ").  | fbq('track', 'InitiateCheckout');  |  
| Лид  |  Предоставление клиентом информации с пониманием того, что ваша компания позднее может с ним связаться (например, отправка формы или регистрация на пробный период).  | fbq('track', 'Lead');  |  
| Покупка  |  Завершение покупки, на которое обычно указывает получение заказа, подтверждения покупки или квитанции о проведении транзакции (например, переход на страницу с благодарностью или страницу подтверждения заказа).  | fbq('track', 'Purchase', {value: 0.00, currency: 'USD'});  |  
| Планирование  |  Запись на встречу в одной из точек вашей компании.  | fbq('track', 'Schedule');  |  
| Поиск  | Поиск на вашем сайте, в приложении или другом объекте монетизации (например, поиск товаров или предложений для путешественников).  | fbq('track', 'Search');  |  
| Начало пробного периода  |  Запуск бесплатной пробной версии вашего продукта или услуги (например, оформление соответствующей подписки).  | fbq('track', 'StartTrial', {value: '0.00', currency: 'USD', predicted_ltv: '0.00'});  |  
| Отправка заявки  |  Отправка заявки на предлагаемый вами продукт, услугу или программу (например, кредитную карту, образовательный курс или вакансию).   | fbq('track', 'SubmitApplication');  |  
| Оформление платной подписки  |  Оформление платной подписки на предлагаемый вами продукт или услугу.  | fbq('track', 'Subscribe', {value: '0.00', currency: 'USD', predicted_ltv: '0.00'});  |  
| Просмотр контента  |  Посещение нужной вам веб-страницы (например, страницы товара или целевой страницы). Благодаря этому событию вы узнаете, что человек посетил вашу веб-страницу, но не узнаете, что он на ней делал и какую информацию просмотрел.  | fbq('track', 'ViewContent');  |  
**Примечание.** Событие "Просмотр страницы" включено в базовый код пикселя. Оно сообщает, когда человек переходит на веб-страницу, на которой установлен базовый код пикселя.
## Пример стандартного события 
Ниже приведен пример кода сайта с добавленными стандартными событиями.
![Diagram showing the implementation of a Meta Pixel standard event in website code.](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=335272898611231&version=1770921948&transcode_extension=webp)
  1. **Оригинальный код сайта.** Вставьте код пикселя Meta между тегами **< head>** и **< /head>** на своей веб-странице. У вас уже может быть другой код между этими тегами, поэтому просто поместите код пикселя ниже всего остального кода, но выше тега **< /head>**.
  2. **Базовый код пикселя Meta.** Базовый код будет выглядеть так, как показано на схеме выше, но вместо 1234567890 будет указан ID вашего пикселя Meta.
  3. **Код стандартного события.** В коде пикселя Meta над тегом **< /script>** вставьте код стандартного события, которое актуально для вашей страницы (например, код события "Добавление в корзину"). Это нужно будет сделать для всех страниц, события на которых вы хотите отслеживать.


Таким образом, на каждой странице должен быть весь код, описанный в разделе 2 (базовый код), но на разных страницах будут разные фрагменты кода, описанного в разделе 3 (код стандартного события).

Укажите электронный адрес, на который вы хотите получать новости о мероприятиях и продуктах, а также другую информацию от Meta for Business.
