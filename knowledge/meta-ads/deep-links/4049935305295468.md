# Управление каталогом с помощью WooCommerce
Если вы [подключите плагин Facebook for WooCommerce к Meta](https://business.facebook.com/business/help/900699293402826), то ваши товары из WooCommerce будут синхронизироваться с вашим каталогом в Commerce Manager — платформой Meta для управления ассортиментом. Вы можете и дальше [управлять товарами в WooCommerce](https://l.facebook.com/l.php?u=https%3A%2F%2Fwoocommerce.com%2Fdocument%2Fmanaging-products%2F&h=AT5ltmlSU5IOOiw49VKXvYHYnjDtzIYjhScRhGKOSVK8J9e66YsV5zLgbjreuEZQxCNHyBcqEgDi4dRCuC3ln1q_TFWHX5poC_Hcc__RMhKKHSpOYHUWV8wJye1K34ZsKvg6kTF4Ts6kqg) — все изменения будут синхронизироваться автоматически. Узнать больше можно в справочном центре WooCommerce. Благодаря плагину у вас также будет доступ к дополнительным вкладкам и настройкам, которые помогут передавать данные о товарах в Meta.
##  Просмотр синхронизированных товаров в Commerce Manager 
В Commerce Manager можно просмотреть синхронизированные товары из интеграции WooCommerce. Для этого нажмите **Каталог** , выберите **Источники данных** и [посмотрите список позиций](https://www.facebook.com/business/help/1406482412767183) в разделе **Каталог** > **Товары**.
На платформах Meta можно продвигать только те товары, которые соответствуют [Нормам рекламной деятельности Meta](https://l.facebook.com/l.php?u=https%3A%2F%2Ftransparency.meta.com%2Fpolicies%2Fad-standards%2F&h=AT4t6FyGa3bGKqGjQQsfgowYDkOm6oe7C55cvR6a7MJG3KL8liOF15fStRXiPbpV8DjGy5SndGoNSOvTAayFBYo1m592JaQSR9I60JGCnPV3T7aN8HU36ilumRaxoWBOSQ1JiqYVb6CD-g) и [Правилам торговли](https://www.facebook.com/policies_center/commerce). Так, нельзя продавать цифровые продукты. Несоответствующие требованиям позиции представлены в каталоге как отклоненные. Если вы считаете, что позиция была отклонена по ошибке, [отправьте запрос на повторную проверку](https://www.facebook.com/business/help/2741254916111768).
##  Управление настройками синхронизации товаров из WooCommerce 
После установки плагина Facebook for WooCommerce в [редакторе товаров WooCommerce](https://l.facebook.com/l.php?u=https%3A%2F%2Fwoocommerce.com%2Fdocument%2Fmanaging-products%2Fproduct-editor-settings&h=AT4D8GoJxONrUOr9iPrB98055TzI7V-tHfXGQpu0r_hq268RuBKmx2DlMABJUTndBUs7u5aObIkYF2h3eBEGE4FeonubV1FKrpEcLbcr68gmiJjAoDj-VU5qijLHD5s6P53N0_tXUZaubQ) появится новая вкладка **Facebook**. На ней будут доступны дополнительные настройки для синхронизации с системой Meta.
Чтобы перейти на вкладку **Facebook** :
  1. На панели WordPress нажмите **Products** (Товары), а затем — **All products** (Все товары). 
  2. Найдите нужную позицию. Нажмите **Edit** (Редактировать), чтобы открыть редактор товара.
  3. В разделе **Product data** (Данные о товаре) выберите вкладку **Facebook**. Вы можете изменить следующие настройки:
     * **Facebook sync** (Синхронизация с Facebook) — укажите, будет ли этот товар синхронизироваться с Commerce Manager. Если оставить настройку по умолчанию **Sync and show in catalog** (Синхронизировать и показывать в каталоге), товар будет синхронизироваться в активном состоянии. Если выбрать **Sync and hide in catalog** (Синхронизировать и скрыть в каталоге), товар будет синхронизован, но добавлен в архив в Commerce Manager, а поэтому не будет показываться в вашей рекламе. Если выбрать **Do not sync** (Не синхронизировать), товар не будет синхронизироваться с Commerce Manager.
     * **Google product category** (Категория товара в Google) — выберите для этой позиции наиболее подходящую категорию и подкатегорию. После выбора категории вы можете добавить дополнительные свойства, присущие каждой категории.


**Примечание.** Если у товара есть варианты, то установленный статус синхронизации с Facebook будет применяться ко всем вариантам.
##  Настройка вариантов товаров 
Вариативные товары — это позиции с разными вариантами, например размеров или цветов. Если вы продаете вариативные товары, плагин Facebook for WooCommerce позволяет вам указать для каждого из вариантов определенные данные, которые будут иметь приоритет над данными родительского товара или варианта и будут видны только на платформах Meta.
  1. [Добавьте вариативные товары](https://l.facebook.com/l.php?u=https%3A%2F%2Fwoocommerce.com%2Fdocument%2Fvariable-product&h=AT7Vz5ICmJ_UHWXSVfIP5rmbSaEwn8NtExUARu-b0tcYC03R1-7vZQWVaIv8D2vMet7wAbza1v1JW-_zAUuCFuD3kfgK3rPW9Ziaz0Qq1DcyCGGqv4fo94woWqc-5rDOzA6GwMl2WCjivg) в WooCommerce. Узнать больше можно в справочном центре WooCommerce. Чтобы синхронизация с Meta прошла успешно, убедитесь, что для каждого варианта указана цена, добавлена метка "В наличии" или "Нет в наличии", а также заполнены свойства. Например, если вы указали размер и цвет для одного варианта, введите эти данные и для других. 
  2. Чтобы добавить данные Meta для вариантов товаров, найдите нужный товар в разделе **Products** (Товары) и нажмите **Edit** (Редактировать). Откроется редактор товаров.
  3. В разделе **Product data** (Данные товара) нажмите **Variations** (Варианты). 
  4. Нажмите на вариант товара.
  5. Нажмите **Facebook for WooCommerce** , чтобы открыть настройки, относящиеся к Meta, для каждого варианта.
     * **Описание на Facebook.** Введите произвольное текстовое описание для этого варианта. Если это поле не заполнено, будет использоваться описание родительского товара из WooCommerce.
     * **Изображение товара на Facebook.** Выберите изображение, которое будет синхронизироваться для этого варианта. Вы можете использовать изображение, назначенное этому варианту (по умолчанию), основное изображение родительского товара, пользовательское изображение или несколько изображений.
     * **Видео товара на Facebook.** Выберите видео товара для этого варианта. Вы можете выбрать видео из библиотеки медиафайлов или указать URL нужного видео.
     * **Цена на Facebook.** Введите цену для этого варианта. Используйте десятичный формат без символа валюты, например: 29,99. Если это поле не заполнено, будет использована цена варианта в WooCommerce.
     * **Производственный номер детали (MPN).** Введите уникальный номер для этого варианта. В [требованиях к данным о товарах](https://www.facebook.com/business/help/120325381656392) Meta рекомендуется указывать MPN для классификации товара.


**Примечание.** Вариативные товары в Commerce Manager называются [вариантами](https://www.facebook.com/business/help/363060785327110).
##  Создание групп товаров 
Группа товаров — это набор позиций, которые вы хотите продвигать в рекламе на платформах Meta. Когда вы [создаете категории в WooCommerce](https://l.facebook.com/l.php?u=https%3A%2F%2Fwoocommerce.com%2Fdocument%2Fmanaging-product-taxonomies%2F&h=AT75aLMiHwfo1goKRKUTChh6_NcWDZX43p0uaLTpj-ig3nCbTjYc5-fBD9JOFR54n3klWteKLrYmdwfz86oHURJO3KMd9dMnwpizyA3mY5E6xX1_-Dw4EyRDKME25KT7NndqF1kT0k5pZg), они синхронизируются с Commerce Manager в виде групп товаров и появляются на вкладке **Каталог** в разделе **Группы**. Категории нужно редактировать в WooCommerce. 
**Примечание**. Вы также можете [создавать группы товаров в Commerce Manager](https://www.facebook.com/business/help/620275848114281), чтобы получить больше возможностей при фильтрации или подборе товаров для каждой группы.
##  Анализ данных пикселя Meta и Conversions API 
[Пиксель Meta](https://www.facebook.com/business/help/742478679120153) и [Conversions API](https://www.facebook.com/business/help/2041148702652965) отслеживают данные о событиях, что помогает улучшить персонализацию, оптимизацию и измерение рекламы на платформах Meta. Пиксель и Conversions API определяют дубликаты событий, полученные из обоих инструментов, поэтому события не измеряются дважды.
  * Если вы подключили пиксель Meta при настройке плагина Facebook for WooCommerce, то можете использовать расширение Chrome [Pixel Helper](https://developers.facebook.com/docs/facebook-pixel/support/pixel-helper) (помощник по пикселям). Оно поможет проверить, установлен ли пиксель и правильно ли он работает.
  * Решение Conversions API интегрировано в плагин Facebook for WooCommerce, то есть оно уже внедрено при установке плагина. 


##  Использование инструментов для устранения проблем 
В плагин Facebook for WooCommerce включено несколько инструментов для устранения проблем с интеграцией с Meta. Чтобы получить доступ к этим инструментам:
  1. На панели WordPress нажмите **Marketing** (Маркетинг) в левой части экрана и выберите **Facebook** в раскрывающемся меню.
  2. Откройте вкладку **Shops** (Магазины).
  3. Прокрутите вниз до раздела **Troubleshooting** (Устранение проблем). 


В этом разделе доступны несколько важных инструментов, в том числе:
  * **Product sync** (Синхронизировать товары) — нажмите **Sync now** (Синхронизировать сейчас), чтобы синхронизировать все товары из WooCommerce с Commerce Manager. Обычно товары синхронизируются автоматически, поэтому прибегайте к этому, только если что-то пошло не так.
  * **Enable Meta diagnosis** (Включить диагностику Meta) — позволяет Meta отслеживать журналы ошибок для определения проблем с плагином Facebook for WooCommerce. Мы рекомендуем оставить эту настройку включенной.
  * **Enable debug mode** (Включить режим отладки) — позволяет использовать следующие инструменты отладки в разделе **Status (Статус) > Tools (Инструменты)** в WooCommerce: 
    * **Facebook - Reset connection settings** (Facebook: сброс настроек подключения) — очищает данные интеграции с Meta и сбрасывает ее. В таком случае нужно снова подключить WooCommerce к Meta.
    * **Facebook - Delete background sync jobs** (Facebook: удаление задач по синхронизации в фоновом режиме) — приостанавливает синхронизацию товаров, которая происходит в фоновом режиме каждый день. Эти задачи снова начнут выполняться на следующий день.
    * **Facebook - Delete all products from your catalog** (Facebook: удаление всех товаров из каталога) — удаляет все товары из Commerce Manager, синхронизированные с WooCommerce. Время на обработку этого запроса зависит от размера каталога. Товары, добавленные в каталог другими источниками данных (не из WooCommerce), не будут удалены. То же самое касается товаров в WooCommerce.
