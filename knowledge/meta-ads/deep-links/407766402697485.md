# Устранение проблем с оптимизацией для отслеживания установок мобильного приложения
Вы могли заметить, что ваша кампания для установок мобильного приложения не отслеживает установки.
  * **Прежде чем вы сможете отслеживать установки мобильного приложения, нужно добавить в него Facebook SDK**. После этого создайте новую рекламную кампанию, нацеленную на установки приложения. Убедитесь, что разработчик [настроил Facebook SDK](https://developers.facebook.com/docs/app-ads) в приложении. 
  * **Если за последние 28 дней у вас не было ни одной установки, вы не сможете выполнять оптимизацию для установок приложения**. Пока вы не получите хотя бы одну установку приложения, ваша компания будет оптимизироваться для кликов по ссылке. 


Вы можете использовать интерактивные инструменты для разработчиков, чтобы подтвердить свое приложение с помощью [Помощника по рекламе приложений](https://developers.facebook.com/tools/app-ads-helper/).
Подробнее об измерении установок приложения можно узнать в нашем [руководстве для разработчиков приложений](https://developers.facebook.com/docs/app-ads/measuring/installs-and-in-app-conversions).
