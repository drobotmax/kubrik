# Цели по результативности, доступные для разных целей рекламы и мест получения конверсий в Meta Ads Manager
Во время создания группы объявлений можно выбрать цель по результативности, чтобы помочь системе определять, кому показывать рекламу. Подробнее о том, [_какие есть типы целей по результативности и для каких действий они выполняют оптимизацию_](https://www.facebook.com/business/help/355670007911605). Доступные цели по результативности зависят от выбранной цели и места получения конверсий рекламы.
Некоторые типы рекламы будут доступны, только если вы выберете определенную цель по результативности. Например, объявления с напоминанием в Instagram доступны только пользователям, которые выбрали цель рекламы **Вовлеченность** или **Узнаваемость** и цель по результативности **Максимальное число просмотров ThruPlay** либо **Максимальное число установленных напоминаний**.
Приведенная ниже таблица поможет понять, какие цели по результативности вам доступны.
