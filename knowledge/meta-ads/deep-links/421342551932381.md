# Рекомендации по партнерской рекламе
## Для брендов
Ниже вы найдете рекомендации для партнерской рекламы, в том числе советы по таргетингу, назначению ставок, местам размещения, креативам и измерению.
  * **Создайте в сегменте каталога группы товаров** , которые хотите рекламировать. Это следует делать на этапе создания группы объявлений. Подробнее о том, [как создать группу товаров](https://www.facebook.com/business/help/620275848114281).
  * **Вместо ID используйте в качестве варианта фильтрации название бренда или товара**. Когда при создании группы товаров в качестве фильтра используется ID, изменения, вносимые продавцом, могут повлиять на позиции в группе.
  * **Используйте широкие аудитории, чтобы охватывать больше[ _людей_](https://www.facebook.com/business/help/283579896000936) и масштабировать кампании**. Это позволит привлечь на страницы ваших товаров на сайте или в приложении продавца больше потенциальных клиентов, что увеличит аудиторию для ретаргетинга.
  * **Настройте ретаргетинг**. Так ваша реклама будет показываться людям, которые просматривали ваши товары на сайте или в мобильном приложении ритейлера.
  * **Используйте отдельные рекламные аккаунты**. Для показа партнерской рекламы лучше использовать по одному аккаунту на каждого ритейлера. Не настраивайте показ партнерской и непартнерской рекламы из одного рекламного аккаунта. 
  * **Не используйте похожие аудитории**. Они ограничат ваш охват и показ. Вместо этого для поиска новых клиентов выбирайте широкие аудитории.
  * **Используйте[места размещения Meta Advantage+](https://www.facebook.com/business/help/733979527611858)** (предыдущее название — автоматические места размещения). Этот инструмент выбирает все доступные места размещения для вашей кампании, в том числе Facebook, Messenger, Instagram и Meta Audience Network.
  * **Начните с максимизации количества конверсий**. Выберите вариант "7 дней после клика" или "1 день после просмотра" для покупок и задайте предельную ставку на основе цены за целевое действие. Ваша реклама должна получить примерно 50 результатов, чтобы наша система научилась эффективно прогнозировать конверсии. Пока этого не произойдет, цена за результат может быть высокой, а показатель рентабельности инвестиций в рекламу — низким. Планируйте получить за всё время действия каждой группы объявлений хотя бы 100 конверсий. Для обучения нашей системы требуется 50 результатов. Ещё 50 нужно, чтобы использовать полученные в ходе обучения данные для высокой результативности.
  * **Просите продавцов добавлять новые позиции в сегмент вашего каталога**. У брендов нет возможности добавлять новые товары в сегмент каталога в Commerce Manager. За это отвечают продавцы, к которым нужно обращаться по поводу недостающих позиций каталога. 


## Для ритейлеров
  * **Регулярно обновляйте общие объекты**. Чтобы ваши партнеры могли использовать общие с вами объекты (например, сегменты каталога или структуры Страниц), они должны постоянно обновляться и быть действующими.
  * **Защищайте конфиденциальную информацию партнеров**. Никому не передавайте информацию об отчетности своих партнеров и отчеты по партнерской рекламы.
  * **Регулярно обновляйте свой профиль**. Потенциальным партнерам будет проще найти вас и связаться с вашей компанией, если в вашем профиле всегда будет актуальная информация.
