# Удаление товаров из каталога в Commerce Manager
Товары, которые вы удаляете из каталога, больше не появятся в объявлениях и магазинах.
**Примечание.** Если товары всё ещё доступны на вашем сайте или могут снова появиться в наличии, рекомендуем [архивировать их](https://www.facebook.com/business/help/543317109402043), а не удалять. Это поможет формировать рекомендации товаров для рекламы на основе результативности прошлых позиций. Если вы удалите товары, данные о событиях, связанных с ними, будут утеряны.
## Прежде чем начать
У вас должен быть [полный доступ к каталогу](https://www.facebook.com/business/help/400383667357991).
## Удаление товаров из каталога вручную в Commerce Manager
Чтобы удалить товары из каталога вручную:
  1. Перейдите в [Commerce Manager](https://www.facebook.com/commerce_manager) и выберите свой каталог.
  2. Откройте вкладку **Каталог** и выберите **Товары**.
  3. Установите флажки рядом с товарами, которые хотите удалить. Выберите **Удалить** и нажмите **Удалить товары** для подтверждения.


**Примечание.** Если вы добавили товары в каталог с помощью другого источника данных, например фида данных или партнерской платформы, удалите их в исходном источнике. В противном случае вручную удаленные товары могут снова появиться в каталоге при следующем обновлении из исходного источника.
## Удаление товаров с помощью фида данных
Прежде чем удалить товары с помощью фида данных, определите, какую настройку загрузки фида вы используете (замену или обновление данных). Эта настройка влияет на процесс удаления товаров. Подробнее о том, [чем отличаются замена и обновление данных](https://www.facebook.com/business/help/2284463181837648).
  * Если вы загружаете файл фида один раз, он автоматически будет настроен на замену данных. Чтобы удалить товары с помощью фида для замены данных, выполните указанные ниже действия.
  * Если вы загружаете фид по графику, убедитесь, что используете график **обновления** или **замены**. Откройте вкладку **Каталог** , перейдите в раздел **Источники данных** и проверьте фид в пункте **Следующее обновление**. Затем следуйте инструкциям по удалению товаров с помощью фида для **замены** или **обновления** данных.

### Удаление товаров с помощью фида для замены данных
Если вы настроили фид для замены данных, новый фид полностью заменит предыдущий. Товары, которых нет в новом файле, будут удалены. Чтобы удалить товары:
  1. Откройте файл данных и удалите строки с товарами, которые хотите удалить. Сохраните файл.
  2. [Загрузите отредактированный фид данных](https://www.facebook.com/business/help/1013894008673621) или сохраните файл на вашем сайте. Товары будут удалены во время следующей запланированной загрузки.

### Удаление товаров с помощью фида для обновления данных
Если вы настроили фид для обновления данных, мы удалим только те товары, которые будут соответствующим образом отмечены в фиде данных. Если вы просто уберете товары из файла, они не будут удалены. Чтобы удалить товары:
  1. Откройте файл данных.
  2. Добавьте в него поле **delete** (удалить). В этом поле укажите значение **true** (истина) для всех товаров, которые хотите удалить. Для товаров, которые хотите оставить, укажите значение **false** (ложь) или просто оставьте поле пустым, как указано в примере ниже.   
 |  
|  |  
| id  | condition  | link  | price  | delete  |  
| 63723  | new  | http://www...  | 6.99 USD  | true  |  
| 96375  | new  | http://www...  | 12.99 USD  |   |  
| 17219  | new  | http://www...  | 4.99 USD  | true  |  
  3. Сохраните отредактированный файл на сайте. Позиции будут удалены из каталога после следующей запланированной загрузки.
  4. Загрузив фид данных с товарами, которые вы отметили для удаления, вы можете убрать эти товары из последующих версий файла. 


## Удаление фида данных
Если вы хотите убрать из каталога все товары, которые содержатся в фиде данных, то можете удалить сам фид. Для этого:
  1. Перейдите в [Commerce Manager](https://www.facebook.com/commerce_manager) и выберите свой каталог.
  2. Откройте вкладку **Каталог** и нажмите **Источники данных**.
  3. Выберите фид данных и откройте вкладку **Настройки**.
  4. Прокрутите экран вниз до раздела **Удаление фида данных**. Нажмите **Удалить** и подтвердите свой выбор.


## Удаление товаров с партнерской платформы
Чтобы удалить товары, которые были добавлены в каталог с партнерской платформы, например Shopify, перейдите на нее и удалите позиции там. Все изменения будут синхронизироваться с каталогом в Commerce Manager.
## Удаление товаров, добавленных с помощью пикселя Meta
Чтобы удалить товар, [добавленный с помощью пикселя](https://www.facebook.com/business/help/887775018036966), уберите позицию с вашего сайта. Из каталога она будет удалена через 7 дней. Чтобы удалить все товары, добавленные с помощью пикселя, отключите его от каталога в разделе **Источники данных**.
