# Сравнение вариантов настройки Conversions API
![](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=438223955063449&version=1666716831&transcode_extension=webp)
Для настройки событий в мобильном приложении или в физической точке (офлайн-события) пока доступна только прямая интеграция.
Есть несколько способов настройки Conversions API. Чтобы выбрать оптимальный для вашей компании вариант, изучите описание способов настройки и таблицы, в которых сравниваются их стоимость, сложность и доступные функции.
# Варианты настройки Conversions API
  * **Партнерские интеграции с торговыми платформами**. Если вы используете торговую платформу, например [Shopify](https://www.facebook.com/business/help/646757258759189), [WooCommerce](https://www.facebook.com/business/help/900699293402826), [Wix](https://www.facebook.com/business/help/349796845717273) или [BigCommerce](https://www.facebook.com/business/help/1581202675515016), то сможете настроить Conversions API бесплатно и всего в несколько кликов. Подробнее о том, [какие торговые платформы поддерживают Conversions API](https://www.facebook.com/business/help/260370078559247).
  * **Шлюз Conversions API**. Вы можете самостоятельно настроить [шлюз Conversions API](https://www.facebook.com/business/help/387152639648383) в Meta Events Manager (этот способ не требует изменения кода). С его помощью можно легко и быстро настроить Conversions API. Если вы управляете несколькими компаниями, то также можете воспользоваться [шлюзом Conversions API для нескольких аккаунтов](https://www.facebook.com/business/help/746511563926080). Это позволит вам один раз настроить шлюз, чтобы потом использовать эту конфигурацию для всех подключенных компаний. 
  * **Другие партнерские интеграции**. 
    * Если ваш сайт расположен на одной из партнерских платформ Meta, например [Adobe](https://l.facebook.com/l.php?u=https%3A%2F%2Fexperienceleague.adobe.com%2Fdocs%2Fexperience-platform%2Ftags%2Fextensions%2Fserver%2Fmeta%2Foverview.html%3Flang%3Den&h=AT5JnDIJzvy2axNOQ1z0KxbuU_apVUaqM4KXy9WbUHmYQeMKkqv2O_fGLPYI2mWWXD4hUyZZ3zxMRRk-lj6U1qYTY9KOm0DYBNxUy1jfyUsRHd-eCPZvcEdalKOb-Ti9_GI5D0ArIJL8zQ), [Google Менеджер тегов](https://www.facebook.com/business/help/702509907046774) или [Tealium](https://www.facebook.com/business/help/892899904421859), вы сможете настроить Conversions API с помощью партнерской интеграции. Подробнее о том, [какие партнеры поддерживают события на сайте, отправляемые с помощью Conversions API](https://www.facebook.com/business/help/260370078559247).
    * Если вы работаете с партнером по решениям для обмена сообщениями, то, возможно, можете настроить Conversions API с помощью партнерской интеграции. Подробнее о том, [какие партнеры поддерживают события переписки с компанией, отправляемые с помощью Conversions API](https://www.facebook.com/business/help/1050753706106190).
  * **Прямая интеграция с использованием кода**. Чтобы лучше контролировать процесс интеграции, обратитесь к разработчику, который поможет вам создать прямую интеграцию с Conversions API. На платформе Meta for Developers можно узнать, как настроить прямую интеграцию для передачи [событий на сайте](https://developers.facebook.com/docs/marketing-api/conversions-api) или [в приложении](https://developers.facebook.com/docs/marketing-api/conversions-api/app-events), а также [офлайн-событий](https://developers.facebook.com/docs/marketing-api/conversions-api/offline-events/) или [событий переписки](https://developers.facebook.com/docs/marketing-api/conversions-api/business-messaging).  
**Примечание**. Если вам нужна более персонализированная настройка, вы можете [создать инструкции](https://www.facebook.com/business/help/232481544843294) в Events Manager и отправить их себе или своему разработчику.


# Сравнение стоимости и сложности вариантов настройки  
|   | Торговые платформы  | Шлюз Conversions API  | Другие партнеры  | Прямая интеграция  |  
| --- | --- | --- | --- | --- |  
| **Стоимость**  | Бесплатно  |  Затраты на облачное хранилище (от 30 долл. США в месяц) **Примечание**. Затраты на облачное хранилище могут отличаться в зависимости от региона, условий использования и объема трафика от пикселя Meta.  | Зависит от партнера  | Бесплатно  |  
| **Время внедрения**  | Меньше часа  | Меньше 30 минут  | Зависит от партнера  |  2–4 недели для новых интеграций 1–2 недели, если вы уже отправили события на сайте и хотите включить в свою интеграцию события в приложении, офлайн-события или события переписки  |  
| **Необходимые ресурсы**  | Маркетолог  | Маркетолог (желательно при хотя бы минимальной поддержке IT-специалиста или разработчика)  | Зависят от партнера  | Разработчик  |  
| **Прочие требования**  | Зависят от партнера  | Требует поставщика услуг облачного сервера. Сейчас поддерживаются платформы Amazon Web Services и Google Cloud  | Зависят от партнера  | Инфраструктура, необходимая для отправки событий в Meta в режиме реального времени  |  
# Сравнение функций, доступных при разных вариантах настройки  
|   | Торговые платформы  | Шлюз Conversions API  | Другие партнеры  | Прямая интеграция  |  
| --- | --- | --- | --- | --- |  
| **Поддерживаемые источники событий**  | События на сайте  | События на сайте  | Зависят от партнера  | События на сайте, в физической точке (офлайн-события) или мобильном приложении, полученные с помощью бизнес-чата, телефонных звонков или электронной почты, генерированные системой и другие  |  
|  **Выбор событий** (какие события можно отправлять)  | Варианты настройки зависят от партнера  | События, которые передает пиксель Meta, также передаются через шлюз Conversions API с выполнением автоматической дедупликации  | Варианты настройки зависят от партнера  | Можно настроить  |  
|  **Выбор и изменение параметров** (какие данные о клиентах можно отправлять с каждым событием)  | Варианты настройки зависят от партнера  | Параметры, которые передает пиксель Meta, также передаются через шлюз Conversions API  | Варианты настройки зависят от партнера  | Можно настроить  |
