# Настройка цели "Максимальная ценность конверсий" в конфигурации SKAdNetwork
Если вы используете SKAdNetwork API от Apple для охвата людей на устройствах с iOS 14.5 и более поздними версиями и хотите использовать параметр "Максимальная ценность конверсий", активируйте его в Events Manager, а также группу значений ценности для каждого события покупки, ценность которого вы хотите максимизировать. Максимальную ценность конверсий нужно включить для конфигурации событий в приложении, передаваемых через SKAdNetwork, но не для событий, передаваемых с помощью протокола "Измерение на основе агрегированных данных о событиях".
**Примечание**. Вы можете использовать максимизацию ценности конверсий для событий, которые не указали приоритетными в Events Manager, для устройств с iOS 14.5 и более поздними версиями. В этом случае вам не придется включать максимизацию ценности конверсий и группу значений ценности в Events Manager. Однако так вы рискуете не охватить людей, которые не предоставили разрешение ATT.
## Настройка цели "Максимальная ценность конверсий" в конфигурации SKAdNetwork
Ниже представлена инструкция по максимизации ценности конверсий и охвату пользователей устройств с iOS 14.5 и более поздними версиями, если вы отправляете события в приложении через SKAdNetwork:
  1. Настройте интеграцию, например Conversions API или Facebook SDK, которые будут отправлять данные о событиях покупки, выполнять оптимизацию для них, а также отслеживать соответствующие значения. 
  2. Как только ваша интеграция начнет отправлять данные, система Meta определит группу значений ценности для выбранного вами события оптимизации. Для событий в приложении, настроенных для SKAdNetwork, это делается автоматически, но вы можете отрегулировать их конфигурацию вручную. Наша система использует [группы значений ценности](https://www.facebook.com/business/help/378857770047703) для группирования конверсий по ценности и их максимизации. Чтобы подобрать группу значений ценности для определенного события, система Meta должна получить достаточное количество данных. Для этого ваша интеграция должна соответствовать [требованиям для использования максимизации ценности конверсий](https://www.facebook.com/business/help/571188993373447).
  3. Получив достаточно данных и [настроив события в приложении для SKAdNetwork](https://www.facebook.com/business/help/670955636925518), вы сможете активировать параметр "Максимальная ценность конверсий" и группы значений ценности в Events Manager. Если вы не выберете события в приложении вручную для SKAdNetwork, мы автоматически определим диапазоны групп значений ценности.
**Примечание**. Если вы включите максимизацию ценности конверсий, вам нужно будет настроить в Events Manager как минимум 4 места событий, доступных в рамках использования [SKAdNetwork](https://www.facebook.com/business/help/188126096313109). Это необходимо, поскольку эффективная максимизация ценности конверсий требует не менее 4 диапазонов значений, каждый из которых занимает одно место события. Подробнее о [группах значений ценности](https://www.facebook.com/business/help/378857770047703) и [приоритете событий](https://www.facebook.com/business/help/193250612476055).
  4. Включив параметр "Максимальная ценность конверсий" в Events Manager, вы сможете создать в Ads Manager кампанию или группу объявлений с оптимизацией для ценности. Ознакомиться с результатами можно в столбце ROAS таблицы отчетности в Ads Manager.
