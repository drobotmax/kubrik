# О Signals Gateway Pixel
Signals Gateway Pixel — это фрагмент кода, который устанавливается на ваш сайт. Он собирает сигналы с сайта и передает их на облачный сервер [Signals Gateway](https://www.facebook.com/business/help/1393493081525641?id=921478266803729). Этот инструмент расширяет ваши возможности по сбору данных и предоставляет ценную статистику для оптимизации вашей маркетинговой деятельности.
##  Преимущества Signals Gateway Pixel 
Signals Gateway Pixel обеспечивает оптимальный сбор данных с сайта, что улучшает результативность и облегчает использование данных, особенно на платформах Meta. Например, если вы запустили маркетинговую кампанию на платформах Meta, оптимизация сбора данных может значительно улучшить результаты. Собирая актуальные и качественные данные с меньшим количеством ошибок, вы сможете:
  1. **Лучше понять целевую аудиторию**. Разберитесь в поведении и предпочтениях пользователей.
  2. **Оптимизировать таргетинг и посыл рекламы**. Адаптируйте объявления, чтобы охватить подходящую аудиторию удачным посылом.
  3. **Улучшить коэффициент конверсии**. Повысьте вероятность обращения посетителей в клиентов.
  4. **Повысить общую результативность кампании**. Добейтесь лучших результатов с помощью стратегий на основе данных.


##  Пример использования 
Предположим, что вы выпускаете новое решение и хотите таргетировать рекламу на пользователей, проявившим интерес к похожим продуктам. Signals Gateway Pixel поможет вам отслеживать взаимодействия пользователей и уточнить вашу стратегию таргетинга.
##  Принцип работы 
Signals Gateway Pixel собирает данные с вашего сайта и передает их исключительно на облачный сервер Signals Gateway, не полагаясь на сторонние файлы cookie. Этот подход предоставит вам больший контроль над вашими данными и обеспечит соблюдение требований к конфиденциальности.
##  Создание пикселя Signals Gateway Pixel 
Чтобы настроить новый пиксель Signals Gateway Pixel и интегрировать его в поток данных, войдите в свой аккаунт Signals Gateway. Затем: 
  1. Перейдите в раздел **Data pipelines** ("Потоки данных") в меню.
  2. Нажмите **Create data pipeline** ("Создать поток данных"). 
  3. Выберите источник данных **Signals Gateway Pixel** и их место назначения.

##  Добавление кода и событий 
Код Signals Gateway Pixel — это фрагмент кода JavaScript, который вы добавляете в заголовок сайта. Он включает в себя базовый код для автоматического отслеживания количества просмотров страницы и дополнительный код для других веб-событий.
##  Установка кода пикселя 
  1. Получите код из своего аккаунта Signals Gateway.
  2. Найдите заголовок своего сайта или шаблон заголовка в CMS. Вставьте код в нижнюю часть раздела заголовка прямо над тегом head. Убедитесь, что код установлен на каждой странице.
  3. Укажите названия событий и отметьте их в своем коде.
  4. Отслеживайте взаимодействия пользователей, например просмотры страницы, клики по кнопке, заполнения формы или покупки.
  5. Укажите параметры для каждого события, например ценность, валюту, тип контента. Это поможет получить более подробную статистику.
  6. Воспользуйтесь инструментом тестирования событий в Signals Gateway и убедитесь, что события должным образом регистрируются.
  7. Подключитесь к месту назначения, следуя инструкциям на экране.


Подробные инструкции по реализации можно найти в [документации для разработчиков](https://developers.facebook.com/docs/marketing-api/gateway-products/signals-gateway/signals-gateway-pixel-onboarding).
