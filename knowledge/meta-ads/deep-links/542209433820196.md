# Использование купона на рекламу для кампании
Для оплаты расходов на кампании можно использовать купоны для кампании и рекламного аккаунта. Купон на рекламу покрывает только расходы, начисленные после его активации. Его нельзя применить для оплаты ранее выставленных счетов.
Если у вас есть купон на рекламу для кампании, средства за показ будут в первую очередь списываться с него. Если средства купона будут исчерпаны до завершения кампании, то для оплаты расходов будет использоваться способ оплаты по умолчанию или купоны для аккаунта. При этом сначала мы будем списывать средства с купонов, срок действия которых истечет скорее.
В ежемесячном счете на оплату можно посмотреть данные о расходах, покрытых купоном на рекламу, с разбивкой по кампаниям (независимо от того, используете вы купоны для одной кампании или всего аккаунта). Эта функция пока может быть вам **недоступна**.
