# Управление рекомендуемыми событиями в приложении в Meta Events Manager
Мы можем рекомендовать определенные стандартные события для вашего приложения. [Подробнее](https://www.facebook.com/business/help/787101178386829). 
Вы можете добавлять рекомендуемые события, а можете отключить эту функцию и перестать получать обновления о событиях, настроенных с помощью нее.
Вы можете получить электронное письмо с предложением настроить рекомендуемые события. Если вы не получили такое письмо или хотите получать рекомендации для событий другим способом, используйте Meta Events Manager.
## Как добавить рекомендуемые события в Events Manager?
  1. Откройте [Events Manager](https://business.facebook.com/events_manager2/list).
  2. Нажмите **Источники данных** ![](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=408108114296304&version=1774528167&transcode_extension=webp) в меню в левой части экрана.
  3. Выберите приложение.
  4. Выберите **Настройки**.
  5. В разделе **Рекомендуемые события** включите рекомендации, если ещё этого не сделали.
  6. Перейдите на вкладку **Обзор**.
  7. Нажмите **Добавить событие**.
  8. Выберите **Посмотреть рекомендуемые события**.
  9. Установите флажок рядом с рекомендуемым событием, которое хотите проверить. 
  10. Нажмите **Проверьте события**.
  11. Установите флажок рядом с рекомендуемым событием, которое хотите добавить.
  12. Нажмите **Добавить события**. Появится сообщение с подтверждением того, что события были добавлены.
  13. Нажмите **Готово**.
  14. На экране **Оставьте отзыв о рекомендациях событий** выберите событие, которое больше всего подходит для соответствующего текста кнопки. Если вы сделаете это, событие, сопоставленное с действием пользователя, не изменится, однако ваш отзыв поможет нам подбирать более точные рекомендации в будущем.
  15. Нажмите **Отправить отзыв**.


События должны появиться в Events Manager в течение 30 минут, после чего вы сможете использовать их для оптимизации и атрибуции рекламы, а также для создания аудиторий. На вкладке "Тестирование событий" можно проверить настройку событий.
## Как перестать получать данные о событиях в Events Manager?
  1. Откройте [Events Manager](https://business.facebook.com/events_manager2/list).
  2. Нажмите **Источники данных** ![](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=408108114296304&version=1774528167&transcode_extension=webp) в меню в левой части экрана.
  3. Выберите приложение.
  4. На вкладке **Обзор** найдите таблицу основных событий.
  5. Нажмите **Управлять** под событием.
  6. Выберите **Прекратить получение события**.
  7. Нажмите **Остановить регистрацию событий** , чтобы подтвердить свой выбор.
  8. Повторите эти шаги для всех событий, данные о которых вы больше не хотите получать.
  9. Чтобы отключить функцию рекомендуемых событий, откройте вкладку **Настройки**.
  10. В разделе **Рекомендуемые события** выключите рекомендации.


Прекратив получать данные о событиях, вы, возможно, не сможете использовать эти события для оптимизации и атрибуции рекламы, а также для создания аудиторий. Чтобы снова начать получать данные о событиях, откройте таблицу **Неактивные события** на вкладке **Обзор** и нажмите **Повторно активировать**.
**Примечание**. Параметр **Прекратить получение события** доступен только для событий, настроенных на основе рекомендаций Facebook.
