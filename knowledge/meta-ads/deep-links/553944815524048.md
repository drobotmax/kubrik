# Поддерживаемые цели и места размещения для рекламы мероприятий
Для рекламы этого типа поддерживается ряд целей и мест размещения. Доступность мест размещения зависит от выбранной вами цели.
## Доступные цели для рекламы мероприятия
Вы можете выбрать одну из следующих целей:
  * [**Вовлеченность**](https://www.facebook.com/business/help/666398006849003)**Интересует** , а человеку, который ответил на такое приглашение, — с кнопкой **Пойду**. Эту цель можно использовать как для мероприятий со свободным входом, так и для мероприятий со входом по билетам.
  * [**Трафик**](https://www.facebook.com/business/help/301780226847564)**Купить билеты** и может использоваться только для офлайн-мероприятий со входом по билетам. Чтобы использовать цель "Продажи", при создании рекламы нужно выбрать формат "Одно изображение или видео".
  * [**Продажи**](https://www.facebook.com/business/help/535561519986477)**Купить билеты** и может использоваться только для офлайн-мероприятий со входом по билетам. Чтобы использовать цель "Продажи", при создании рекламы нужно выбрать формат "Одно изображение или видео".


Цель можно менять с учетом наиболее важной для вас метрики. Цели **Трафик** и **Продажи** используются только для рекламы офлайн-мероприятий со входом по билетам. 
## Доступные места размещения для рекламы мероприятия
Доступность мест размещения для рекламы этого типа зависит от выбранной вами цели. Ниже указаны места размещения, доступные для определенных целей. 
**Примечание.** С марта 2026 года место размещения "Лента Facebook" будет включать в себя [вкладку "Друзья" на Facebook](https://www.facebook.com/help/1326838631881342). 
**Вовлеченность**
  * Лента Facebook.


**Трафик** и **Продажи**
  * Лента Facebook.
  * Facebook Marketplace.
  * Правый столбец на Facebook (для рекламы с изображением).
  * Истории на Facebook.
  * Реклама In-Stream в видео Reels на Facebook.
  * Результаты поиска на Facebook.
  * Нативная, баннерная и межстраничная реклама в Meta Audience Network.
  * Видео с вознаграждением в Audience Network (для рекламы с видео).
  * Видеореклама In-Stream в Audience Network (для рекламы с видео, недоступна для цели "Трафик").

## Будьте в курсе новостей от Meta for Business
Укажите электронный адрес, на который вы хотите получать новости о мероприятиях и продуктах, а также другую информацию от Meta for Business.
