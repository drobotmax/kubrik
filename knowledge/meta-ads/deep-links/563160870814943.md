# О рекламе с кольцевой галереей в историях в Instagram
Чтобы показать больше товаров в одном объявлении, можно использовать формат кольцевой галереи для рекламы в историях в Instagram. В такую рекламу можно добавить несколько фото или видео. Формат кольцевой галереи доступен для места размещения "Истории в Instagram". 
В одно объявление с кольцевой галереей можно загрузить до 10 карточек. В истории в Instagram автоматически показывается от одной до пяти карточек кольцевой галереи, однако зрители могут нажать кнопку **Развернуть историю** , чтобы увидеть остальные карточки (до 10). По умолчанию автоматически показываемое число карточек подбирается для каждого индивидуально.
Если вы хотите, чтобы до появления кнопки **Развернуть историю** зрителю показывалось фиксированное количество карточек, при создании объявления в Meta Ads Manager выберите истории в Instagram в качестве единственного места размещения. Затем можно выбрать количество карточек — 3 (или 2 для кольцевой галереи из 2 карточек).
