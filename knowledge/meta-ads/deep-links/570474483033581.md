# О мобильных рекламных идентификаторах для пользовательских аудиторий
Мобильные рекламные идентификаторы сообщают разработчикам приложений для мобильных устройств, кто пользуется их продуктами.
Есть три типа мобильных рекламных идентификаторов, которые доступны для [пользовательских аудиторий](https://www.facebook.com/business/help/285857478228227):
  * **Рекламный идентификатор Apple (IDFA).** Рекламный идентификатор, который Apple предоставляет как часть [рекламного фреймворка](https://l.facebook.com/l.php?u=https%3A%2F%2Fdeveloper.apple.com%2Flibrary%2Fios%2Fdocumentation%2FAdSupport%2FReference%2FASIdentifierManager_Ref%2FASIdentifierManager.html%23%2F%2Fapple_ref%2Fdoc%2Fuid%2FTP40012654&h=AT6cRSFaoA7jDwax2EiTNE3YDvrj-XWCkK5r4CEuQTHFSGPnkecaMyhnsoNozzuiK9OhBOCutzqJaZ4IgjM-KEgBe05c-A_LoS88iKq9R_WCCv6iCPne2RsyBgAXh09aTpw7zhOyAhjtmQ) для iOS.
  * **Рекламный идентификатор Android.** Рекламный идентификатор, который Google предоставляет как часть Android.
  * **Идентификатор пользователя Facebook.** Идентификатор для пользователей приложения можно получить с помощью Facebook SDK.


Мобильные рекламные идентификаторы могут пригодиться для пользовательских аудиторий, если:
  * Вам нужно охватить [_людей_](https://www.facebook.com/business/help/283579896000936), которые уже используют ваше приложение с рекламой на платформах Meta на компьютерах или мобильных устройствах.
  * Вам нужно таргетировать рекламу на людей, которые выполнили конкретное действие в вашем приложении (например, совершили покупку).
  * Вам нужно охватить людей, которые уже взаимодействовали с вашим приложением, не использующим вход через Facebook.

## Будьте в курсе новостей от Meta for Business
Укажите электронный адрес, на который вы хотите получать новости о мероприятиях и продуктах, а также другую информацию от Meta for Business. 
Отправляя эту форму, вы соглашаетесь получать по электронной почте маркетинговые материалы от Meta, в том числе новости, информацию о мероприятиях, обновления и специальные предложения. Вы можете отказаться от рассылки в любой момент. Для этого нужно, например, нажать ссылку для отмены подписки в нашем электронном письме.
