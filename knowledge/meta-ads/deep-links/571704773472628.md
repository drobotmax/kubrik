# Подключите вашу CRM-систему к Meta с помощью Conversions API для CRM
Партнеры по интеграциям Conversions API с CRM создают связь между данными о лидах и Meta Business Suite для повышения результативности рекламы. После интеграции с Meta (напрямую или через сторонний инструмент) ваша CRM-система будет отправлять нам данные о лидах. Вы сможете использовать их, чтобы оптимизировать рекламу для сбора лидов для качества, а не для объема. Партнеры по интеграциям Conversions API с CRM предлагают простое решение, для подключения которого не требуется привлекать разработчиков и писать код.
Вы также можете подключить Google Таблицы, чтобы отправлять Meta данные о конвертированных лидах. Подробнее о том, [_как подключить Google Таблицы к Meta из Meta Business Suite_](https://www.facebook.com/business/help/1095577538573756). Если вы используете Центр лидов, лиды из моментальных форм будут отправляться автоматически, поэтому настройка не требуется.
## Подключите вашу CRM-систему к Meta с помощью Conversions API для CRM
  1. Откройте [_Events Manager_](https://www.facebook.com/events_manager2/).
  2. В меню нажмите **Подключить данные**.
  3. Появится редактор **Подключение нового источника данных**. Выберите **CRM** , а затем — **Далее**.
  4. Появится сообщение **Подключите CRM, чтобы находить конвертированные лиды**. Ознакомьтесь с инструкциями и преимуществами, а затем нажмите **Далее**.
  5. Введите и выберите в поисковой строке название набора данных, в котором будут появляться события из вашей CRM-системы, или нажмите **Создать новый набор данных**.
     * Если вы **создаете новый набор данных** , введите его название и нажмите **Создать**.
  6. Выберите один из способов и подключите свою CRM-систему к Conversions API.
     * **Подключение к партнеру.** Выполните поиск партнера или введите название своей CRM-системы Затем нажмите **Далее**.
     * **Подключение вручную.** Выберите **Напишите код вручную самостоятельно** или **Пригласите разработчика для помощи в настройке** Затем нажмите **Далее**.
  7. После подключения должно быть отправлено первое событие.
  8. Дальнейшая настройка зависит от способа подключения. Если вы используете интеграцию Conversions API для CRM от бизнес-партнера Meta, ознакомьтесь с инструкциями по ссылкам ниже.

## О статусах интеграции CRM 
**Проверка статуса интеграции CRM**
На вкладке "Обзор" приводится статус интеграции вашей CRM-системы и дальнейшие указания по настройке, если вы ее ещё не завершили. Вы можете навести курсор на статус, чтобы узнать больше, и нажать **Смотреть отчеты** , чтобы просмотреть данные.

## Будьте в курсе новостей от Meta for Business
Укажите электронный адрес, на который вы хотите получать новости о мероприятиях и продуктах, а также другую информацию от Meta for Business. 
Отправляя эту форму, вы соглашаетесь получать по электронной почте маркетинговые материалы от Meta, в том числе новости, информацию о мероприятиях, обновления и специальные предложения. Вы можете отказаться от рассылки в любой момент. Для этого нужно, например, нажать ссылку для отмены подписки в нашем электронном письме. Узнать, как Meta обрабатывает вашу информацию, можно в [Политике использования данных](https://www.facebook.com/about/privacy).
