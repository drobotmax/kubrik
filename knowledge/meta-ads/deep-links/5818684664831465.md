# Создание набора данных в Meta Events Manager
Набор данных в Meta Events Manager Набор данных [можно создать несколькими способами](https://www.facebook.com/business/help/5270377362999582). Выберите нужный и следуйте инструкциям.
**Примечание**. Чтобы подключить существующий пиксель Meta к набору данных, создайте набор в мобильном приложении и выберите вариант **Создать из ID пикселя**.
Создание набора данных не должно влиять на оптимизацию кампании и отчетность в Meta Ads Manager. Однако если вы свяжете идентификатор приложения с другим набором данных, это может изменить общее количество событий в приложении и результативность рекламы приложения. Подробнее о том, [как связать свое приложение с другим идентификатором набора данных](https://www.facebook.com/business/help/2499747716861514).

### Создание набора данных в существующем мобильном приложении
## Прежде чем начать
  * Убедитесь, что мобильное приложение, которое вы хотите подключить к набору данных, принадлежит вашему бизнес-портфолио.
  * Если вы нажмете "Создать из ID пикселя", то пиксель, который вы хотите подключить к набору данных, должен принадлежать вашему бизнес-портфолио.
  * У вас должен быть полный доступ к бизнес-портфолио. Если у вас ещё нет бизнес-портфолио, узнайте, [как его создать](https://www.facebook.com/business/help/1710077379203657). Если у вас есть бизнес-аккаунт, но нет к нему полного доступа, попросите человека с полным доступом [предоставить вам его](https://www.facebook.com/business/help/299504287548592).


## Создание набора данных в существующем мобильном приложении
Чтобы создать набор данных:
  1. Откройте [**Meta Events Manager**](https://www.facebook.com/events_manager2/). 
  2. Нажмите на вкладку **Источники данных** ![](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=408108114296304&version=1774528167&transcode_extension=webp) в левой части экрана.
  3. Выберите мобильное приложение, которое хотите связать с набором данных.
  4. Нажмите **Настройки** в верхней части экрана.
  5. Откройте раздел **Связывание** и нажмите **Связать**.
  6. Ознакомьтесь с информацией о наборах данных и нажмите **Далее**.
  7. Выберите **Создать из ID пикселя** или **Создать новый**. [Подробнее](https://www.facebook.com/business/help/5270377362999582).  
  
**Примечание**. Чтобы связать события существующего пикселя Meta с набором данных, выберите **Создать из ID пикселя**.
  8. Нажмите **Далее**.
  9. Установите флажок, подтверждающий, что вы ознакомились с уведомлением и осознаете свои действия.
  10. Нажмите **Создать**.


Связав идентификатор приложения с набором данных, вы можете просматривать данные о событиях, нажав на идентификатор в Events Manager.

### Создание набора данных для подключения офлайн-событий
## Прежде чем начать
Чтобы связать мобильное приложение с набором данных:
  * Убедитесь, что мобильное приложение, которое вы хотите подключить к набору данных, принадлежит вашему бизнес-портфолио.
  * У вас должен быть полный доступ к бизнес-портфолио.


## Создание набора данных для подключения офлайн-событий
Чтобы создать набор данных:
  1. Откройте [**Meta Events Manager**](https://www.facebook.com/events_manager2/). 
  2. Найдите зеленый значок плюса и нажмите **Подключить источники данных**.
  3. Выберите **Офлайн** , а затем — **Подключить**.
  4. Введите название набора данных. Оно должно быть уникальным, чтобы вы могли отличить этот набор во время создания рекламы или пользовательской аудитории.
  5. Нажмите **ОК**.


Создав набор данных, вы можете просматривать данные о событиях, нажав на идентификатор в Events Manager.
Чтобы связать мобильное приложение с набором данных, перейдите к настройкам набора в Events Manager и откройте раздел **Связывание**. Затем нажмите **Связать** и следуйте инструкциям на экране.

### Создание набора данных вместе с группой событий переписки
## Прежде чем начать
  * Убедитесь, что ваша Страница Facebook принадлежит бизнес-портфолио, которое вы хотите подключить к набору данных.
  * У вас должен быть полный доступ к бизнес-портфолио. Если у вас ещё нет бизнес-портфолио, узнайте, [как его создать](https://www.facebook.com/business/help/1710077379203657). Если у вас есть бизнес-аккаунт, но нет к нему полного доступа, попросите человека с полным доступом [предоставить вам его](https://www.facebook.com/business/help/299504287548592).
  * Чтобы связать мобильное приложение с набором данных:
    * Убедитесь, что у вашего бизнес-портфолио есть права на мобильное приложение, которое вы хотите связать с набором данных.


## Создайте набор данных вместе с группой событий переписки.
Чтобы создать набор данных:
  1. Откройте [**Meta Events Manager**](https://www.facebook.com/events_manager2/). 
  2. Найдите зеленый значок плюса и нажмите **Подключить источники данных**.
  3. Выберите **Обмен сообщениями** , а затем — **Подключить**.
  4. Найдите и выберите свою Страницу Facebook. Вы можете найти ее по названию или [ID](https://www.facebook.com/business/help/2814101678867149). Нажмите **Далее**.
  5. Вы можете выбрать, чтобы события переписки регистрировались с существующими данными, или создать для них новый набор данных. Если у вас нет существующих данных, вы сможете только создать новый набор. Выберите нужный вариант и нажмите **Далее**.
  6. Выберите, как вы хотите подключать данные.
    1. **Партнерская интеграция**. [Подробнее о партнерских интеграциях для событий переписки](https://www.facebook.com/business/help/1050753706106190).
    2. **Прямая интеграция**. Если вы выберете прямую интеграцию, то будете перенаправлены в Events Manager, где сможете создать персонализированные инструкции по настройке Conversions API. Подробнее о том, [как внедрить Conversions API с помощью персонализированных инструкций](https://www.facebook.com/business/help/232481544843294).
  7. Нажмите **Далее**.
  8. Следуйте инструкциям на экране, чтобы начать делиться данными из бизнес-чатов.


Создав набор данных, вы можете просматривать данные о событиях, нажав на идентификатор в Events Manager. 
Чтобы связать мобильное приложение с набором данных, перейдите к настройкам набора в Events Manager и откройте раздел **Связывание**. Затем нажмите **Связать** и следуйте инструкциям на экране.
