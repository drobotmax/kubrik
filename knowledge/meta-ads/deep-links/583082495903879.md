# Об автоматической оплате ежемесячных счетов
Функция "Автоматическая оплата" позволяет автоматически оплачивать ежемесячные счета с помощью способа оплаты по умолчанию. Если включить эту функцию, в дату оплаты счета система Meta будет автоматически списывать средства с помощью выбранного способа оплаты. Автоматическая оплата пока доступна не во всех странах и регионах.
Это статья для рекламодателей, которым доступно ежемесячное выставление счетов для оплаты их объявлений. Если у вас есть вопросы о выставлении счетов и вы оплачиваете рекламу с помощью PayPal, кредитной или дебетовой карты, банковского счета либо вручную, ознакомьтесь с [этой статьей](https://www.facebook.com/business/help/716180208457684).
## Принцип работы автоматической оплаты при ежемесячном выставлении счетов
Используя автоматическую оплату [ежемесячных счетов](https://www.facebook.com/business/help/2086865811541431), вы можете:
  * **Добавить или изменить способ оплаты.** Автоматическая оплата производится согласно способу оплаты по умолчанию. Чтобы использовать для автоматической оплаты другой способ оплаты, измените способ оплаты по умолчанию.
  * **Включить или выключить автоматическую оплату.** Вы сможете сделать это в любой момент. Если вы выключите эту функцию после выставления счета, вам потребуется оплатить его вручную с помощью банковского перевода.


Если вы включите функцию "Автоматическая оплата", счета будут автоматически оплачиваться в установленный срок согласно способу оплаты по умолчанию. Выписки о платежах могут появиться в банковском счете в течение 2–3 рабочих дней. Подробнее о том, [как настроить автоматическую оплату ежемесячных счетов](https://www.facebook.com/business/help/285400536084028).
## Проблемы с платежами и их влияние на аккаунт
Если подключенный способ оплаты отклонит запланированный автоматический платеж, вы будете обязаны оплатить счет вручную. Иначе ваши рекламные аккаунты могут быть приостановлены, пока счет не будет оплачен.
## Различия между автоматической оплатой и использованием кнопки "Оплатить"
**Автоматическая оплата**
  * Периодическое выставление счетов.
  * Счета автоматически оплачиваются в установленный срок согласно способу оплаты по умолчанию.
  * Платежи происходят автоматически без настройки вручную.


**Кнопка "Оплатить"**
  * [Разовые платежи.](https://www.facebook.com/business/help/1814143615511573)
  * Доступна для счетов на оплату, выставленных в течение месяца.
  * Позволяет выполнять оплату с помощью прямого дебетования.
  * Платежи с помощью кредитных карт не поддерживаются.
  * Оплату необходимо произвести вручную через бизнес-портфолио.
  * Позволяет совершать разовые платежи вручную через бизнес-портфолио, используя связанный банковский счет.
  * Доступна только в США и Единой зоне платежей в евро (SEPA). ![](https://static.xx.fbcdn.net/rsrc.php/yX/r/-N6UxtC4nZh.webp)
