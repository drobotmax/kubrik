# Обязательные события и параметры пикселя Meta для объявлений из каталога Advantage+
Чтобы показывать объявления из каталога Meta Advantage+, вам потребуется настроить определенные события и параметры пикселя Meta. [Подробнее об использовании пикселя для объявлений из каталога Advantage+](https://www.facebook.com/business/help/2637288829930563). 
Ваш пиксель должен содержать приведенные ниже стандартные события.  
| Название события  | Описание  |  
| --- | --- |  
| ViewContent  | Кто-то просмотрел товар на вашем сайте.  |  
| AddToCart  | Кто-то добавил товар в корзину на вашем сайте.  |  
| Purchase  | Кто-то купил товар на вашем сайте.  |  
Чтобы мы могли сопоставить товары на сайте с позициями в каталоге, у этих событий должны быть определенные параметры.  
| Название параметра  | Описание  | Требование  | Тип значения  | Пример  |  
| --- | --- | --- | --- | --- |  
| content_ids  | ID определенного контента для товара или группы товаров в вашем каталоге. Не забудьте поставить букву "s" в конце "content_ids".  | Обязательно использовать "content_ids" или "contents".  | Может иметь значение ID конкретного контента или массива контента (несколько идентификаторов).  | ['1234','1853']  |  
| contents  | ID определенного контента для товара в вашем каталоге. Если вы используете этот параметр, добавьте следующие подобъекты: ID одного или нескольких товаров, а также количество добавленных в корзину или приобретенных позиций.  | Обязательно использовать "content_ids" или "contents".  | Должен иметь значение массива объектов (параметр контента, ID подобъекта и количество подобъектов).  | [{id: '1234', quantity: 2}]  |  
Например, если вы хотите видеть, когда кто-то добавляет товары в корзину, необходимо включить событие **AddToCart** в код пикселя. Код в этом случае будет выглядеть так:

```
fbq('track', 'AddToCart')
```

Код события **AddToCart** с параметром **content_ids** для определенных товаров будет выглядеть так:

```
fbq('track', 'AddToCart', {
  content_ids: ['1234','1853','9386'],
});
```

Добавив обязательные стандартные события и параметры на свой сайт, подключите пиксель к каталогу. После этого вы сможете создавать объявления из каталога Advantage+ с актуальными позициями из вашего каталога. Подробнее о том, [как подключить пиксель к каталогу в Commerce Manager](https://www.facebook.com/business/help/1044262445604547).
**Примечание**. Это обязательные параметры для показа объявлений с товарами из каталога Advantage+. Чтобы показывать рекламу [гостиниц](https://www.facebook.com/help/661142797646824), [авиабилетов](https://www.facebook.com/help/2522836671061789), [туристических направлений](https://www.facebook.com/help/2215188722143939), [автомобилей](https://www.facebook.com/help/518588851979334) или [недвижимости](https://www.facebook.com/help/1993509814217759), нужно добавить другие параметры событий пикселя.
