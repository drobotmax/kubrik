# Добавление дополнительного фида в каталог в Commerce Manager
В дополнительных фидах содержатся данные о товарах, которые дополняют данные ваших основных фидов. Если вы управляете товарами на партнерской платформе, например на Shopify, то для продажи на Facebook и в Instagram можете добавить свойства, которые могут не поддерживаться вашей партнерской платформой. В этом вам поможет дополнительный фид.
## Принцип работы дополнительных фидов
  * Дополнительный фид должен быть подключен к основному, который вы хотите дополнить.
  * К каждому основному фиду можно подключить только один дополнительный. Однако один дополнительный фид можно подключить к нескольким основным.
  * С помощью дополнительных фидов можно добавлять или обновлять большинство свойств товаров, кроме ID контента, цены, цены продажи, сведений о наличии и количестве.

### Какие есть поддерживаемые поля для дополнительных фидов?
Вы можете добавить поле **id** , чтобы указать, для какого товара вы добавляете дополнительные данные, а также любые из указанных ниже полей. При заполнении полей следуйте [этим требованиям](https://www.facebook.com/business/help/120325381656392).
    * title
    * description
    * rich_text_description
    * image_link
    * additional_image_link
    * google_product_category
    * fb_product_category
    * [поля для разных категорий](https://developers.facebook.com/docs/commerce-platform/catalog/categories#cat-spec-fields)
    * brand
    * item_group_id
    * color
    * size
    * material
    * pattern
    * gender
    * age_group
    * video
    * gtin
    * custom_label_0, custom_label_1, custom_label_2, custom_label_3, custom_label_4
    * additional_variant_attribute
    * is_customisation_required
    * custom_text_fields[0].input_guidance
    * custom_text_fields[0].field_name
    * custom_text_fields[0].is_required
    * origin_country
    * importer_name
    * importer_address
    * manufacturer_info
  * Данные в дополнительном фиде сопоставляются с соответствующим товаром в подключенном основном фиде по ID контента каждого товара, поэтому ID должны точно совпадать в обоих фидах.
  * Если одно и то же свойство товара присутствует в основном и дополнительном фидах, дополнительный в приоритете. При этом основной фид не изменится.
  * С помощью дополнительных фидов нельзя добавлять новые товары в каталог и удалять товары. Дополнительные фиды нельзя использовать в качестве отдельных фидов. Они предназначены только для улучшения или обновления существующих товаров.

## Прежде чем начать
  * У вас должен быть [полный доступ к каталогу](https://www.facebook.com/business/help/400383667357991).
  * Товары, которые вы хотите обновить, должны быть добавлены в каталог в Commerce Manager.
  * [Создайте файл для дополнительного фида](https://www.facebook.com/business/help/884650838839678) в формате XLSX, CSV, TSV, XML (RSS/ATOM) или Google Таблиц.
  * Чтобы установить график обновлений, сохраните файл сначала на сервере, а затем в хранилище, например в Dropbox или Google Таблицах (необязательно). [Подробнее о загрузках фида по графику.](https://www.facebook.com/business/help/2284463181837648)


## Добавление дополнительного фида в каталог в Commerce Manager
Чтобы загрузить дополнительный фид:
  1. Перейдите в [Commerce Manager](https://www.facebook.com/commerce_manager) и выберите свой каталог.
  2. Откройте вкладку **Каталог** и нажмите **Источники данных**.
  3. В разделе **Дополнительные фиды** нажмите **Добавить дополнительный фид**.   
**Примечание**. К каждому основному фиду можно подключить только один дополнительный. Если все основные фиды уже подключены к дополнительным, эта кнопка может быть неактивной. В таком случае нажмите **Загрузить файл** и обновите существующий дополнительный фид, а не добавляйте новый.
  4. Нажмите **Загрузить фид**. Если у вас нет готового файла, [узнайте, как его создать](https://www.facebook.com/business/help/884650838839678).
  5. Выберите способ загрузки файла из вариантов ниже.
     * **Загрузка файла**. Выполните единичную загрузку с компьютера. [Перейти к разделу](https://www.facebook.com/business/help/609872007120910?id=725943027795860#Computer).
     * **Обновление фида по графику**. Укажите ссылку на файл, сохраненный на сервере или в хранилище, например Dropbox или Google Диске, и мы извлечем его в запланированное время. [Перейти к разделу](https://www.facebook.com/business/help/609872007120910?id=725943027795860#URL).
     * **Google Таблицы**. Укажите ссылку на файл в Google Таблицах, и мы извлечем его в запланированное время. [Перейти к разделу](https://www.facebook.com/business/help/609872007120910?id=725943027795860#GoogleSheets).


## Загрузка файла с компьютера
Чтобы загрузить файл с компьютера:
  1. Перетащите файл с компьютера или нажмите **Загрузить файл**. Файл должен быть сохранен в формате XLSX, CSV, TSV или XML (RSS/ATOM), а его размер не должен превышать 100 МБ. Нажмите **Далее**.
  2. Выберите один или несколько основных фидов данных, к которым хотите подключить дополнительный фид. Основные фиды данных должны содержать все позиции, которые нужно обновить с помощью дополнительных. Чтобы избежать ошибок, подключите правильные фиды данных. Если вы не уверены в своем выборе, то можете перейти в раздел **Источники данных** , чтобы просмотреть позиции, которые содержатся в этом фиде данных. Затем нажмите **Далее**.
  3. Укажите название дополнительного фида и выберите **Загрузить**.


Ваш дополнительный фид появится в разделе **Источники данных**. Вы можете просмотреть его или отредактировать настройки, в том числе изменить основные связанные фиды. Чтобы обновить фид в будущем, откройте его настройки и загрузите новую версию файла в разделе **Единичная загрузка**.
## Обновление фида по ссылке
Чтобы загрузить файл с сервера или сайта, например из Dropbox или с Google Диска:
  1. Укажите ссылку, по которой размещен ваш файл. Ее название должно начинаться с http://, https://, ftp:// или sftp://. Ссылка должна вести на файл в формате CSV, TSV или XML (RSS/ATOM), а не на целевую страницу на вашем сайте, Страницу Facebook или в другое место. Чтобы проверить, можно ли скачать файл по ссылке, проверьте URL в своем браузере. Максимальный размер файла — 8 ГБ. Если URL начинается с ftp:// или sftp:// и защищен паролем, введите данные для входа. Закончив, нажмите **Далее**.
  2. Выберите, когда и насколько часто фид данных должен автоматически обновляться. Нажмите **Далее**.
  3. Выберите один или несколько основных фидов данных, к которым хотите подключить дополнительный фид. Основные фиды данных должны содержать все позиции, которые нужно обновить с помощью дополнительных. Чтобы избежать ошибок, подключите правильные фиды данных. Если вы не уверены в своем выборе, то можете перейти в раздел **Источники данных** , чтобы просмотреть позиции, которые содержатся в этом фиде данных. Затем нажмите **Далее**.
  4. Укажите название дополнительного фида и выберите **Загрузить**.


Ваш дополнительный фид появится в разделе **Источники данных**. Вы можете просмотреть его или отредактировать настройки, в том числе изменить график загрузки и связанные фиды данных. Чтобы обновить фид, отредактируйте файл и повторно сохраните его в хранилище. Мы извлечем файл по указанной ссылке и обновим каталог согласно графику.

### Обновление дополнительного фида, добавленного в Commerce Manager
Чтобы обновить существующий дополнительный фид:
  1. Откройте дополнительный фид и отредактируйте нужную информацию. Например, добавьте поля, в которые хотите внести дополнительные данные. Когда всё будет готово, сохраните файл. 
  2. Перейдите в [Commerce Manager](https://www.facebook.com/commerce_manager) и выберите свой каталог.
  3. Откройте вкладку **Каталог** и нажмите **Источники данных**.
  4. В разделе **Дополнительные фиды** найдите нужный дополнительный фид и нажмите **Загрузить фид**. 
  5. Перейдите в раздел **Настройки**. 
  6. Прокрутите до раздела **Единичная загрузка**. Выберите файл на компьютере или укажите ссылку файл в хранилище. Нажмите **Загрузить файл**.
