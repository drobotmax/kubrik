# Управление разрешениями для групп офлайн-событий
Чтобы загружать данные в группу офлайн-событий, потребуется доступ администратора к компании, которая ее создала, либо разрешение на управление этой группой или на ее загрузку. 
Вы можете управлять ролями и разрешениями для группы офлайн-событий в Meta Business Suite.
## Чтобы назначить полный доступ к бизнес-портфолио:
  1. Откройте [**настройки**](https://business.facebook.com/latest/settings/).
  2. В разделе **Пользователи** выберите **Люди**.
  3. Выберите имя нужного человека.
  4. Нажмите на **значок с горизонтальным троеточием** и выберите **Редактировать разрешения для бизнес-портфолио**.
  5. Внесите изменения. Когда всё будет готово, нажмите **Сохранить**.


## Чтобы предоставить доступ к группе офлайн-событий другой компании или агентству:
  1. Откройте [**настройки**](https://business.facebook.com/latest/settings/).
  2. Нажмите **Источники данных**.
  3. Нажмите **Группы офлайн-событий**.
  4. Выберите группу офлайн-событий, к которой хотите предоставить доступ.
  5. Чтобы предоставить доступ сотрудникам вашей компании:
    1. Нажмите **Назначить людей**.
    2. Выберите имена людей, которым хотите предоставить доступ.
    3. Активируйте переключатель **Загрузка офлайн-конверсий** или **Управление группой офлайн-событий**.
    4. Нажмите **Назначить**.
  6. Чтобы предоставить доступ другой компании или агентству:
    1. Нажмите **Назначить партнеров**.
    2. Укажите ID бизнес-портфолио партнера.
    3. Активируйте переключатель **Загрузка офлайн-конверсий** или **Управление группой офлайн-событий** и нажмите **Далее**.
    4. Установите флажки рядом с пунктами, которые описывают суть сотрудничества с вашим партнером, и нажмите **Далее**.
    5. Ознакомьтесь с условиями и нажмите **Принимаю**.


Вы предоставили разрешения для работы с группой офлайн-событий. Теперь люди с доступом администратора могут загружать в нее данные об офлайн-событиях.
**Примечание.** Использовать [Offline Conversions API](https://developers.facebook.com/docs/marketing-apis/offline-conversions/v2.8) могут только **системные пользователи** бизнес-портфолио.
