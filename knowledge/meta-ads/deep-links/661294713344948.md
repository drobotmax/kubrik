# Устранение ошибки 3253001: достигнут лимит объявлений
Для каждой Страницы установлен лимит объявлений, которые уже активны или находятся на проверке. Превысив установленный лимит объявлений для Страницы, вы можете увидеть код ошибки 3253001, обозначающий, что показ рекламы запрещен, пока проблема не будет устранена. [Подробнее о лимитах объявлений для Страницы](https://www.facebook.com/business/help/766697140509126).
Из этой статьи вы узнаете о распространенных причинах появления этого кода ошибки и том, как устранить их, чтобы возобновить показ рекламы.
  * **Вы достигли лимита объявлений для Страницы**. Для каждой Страницы установлен лимит объявлений, который зависит от масштаба деятельности рекламодателя. Достигнув его, вы не сможете показывать новую рекламу. Чтобы решить эту проблему, выполните следующие действия:
    * Откройте инструмент [Лимиты объявлений для Страницы](https://www.facebook.com/ads/ad_limits), чтобы узнать, насколько превышен лимит. Учитываются как активные, так и находящиеся на проверке объявления, созданные с разных аккаунтов. Чтобы просмотреть список всех объявлений для Страницы со всех аккаунтов, нужен полный доступ к этой Странице. Если вы не видите, в каких аккаунтах есть активные объявления, свяжитесь с человеком с полным доступом и попросите удалить или отключить лишнюю рекламу. [Подробнее о доступе к Странице](https://www.facebook.com/business/help/1101781386943864).
    * [Удалите](https://www.facebook.com/business/help/991073724257797) или [отключите](https://www.facebook.com/business/help/172764286113080) лишние объявления, чтобы не превышать установленный лимит.
    * [Активируйте](https://www.facebook.com/business/help/172764286113080) объявления с ошибками.
  * **Вы превысили лимит объявлений, установленный другим пользователем с полным доступом к Странице для вашего аккаунта**. Люди с полным доступом к Странице могут настраивать дополнительные ограничения для рекламных аккаунтов и бизнес-портфолио, чтобы задать лимит объявлений для определенных рекламодателей или запретить им публиковать новую рекламу. Если вы достигли установленного для вас лимита объявлений, выполните следующие действия:
    * Откройте инструмент [Лимиты объявлений для Страницы](https://www.facebook.com/ads/ad_limits), чтобы узнать, насколько превышен лимит.
    * Устранить проблему можно двумя способами:
      * Если вы владелец Страницы, откройте инструмент [Лимиты объявлений для Страницы](https://www.facebook.com/ads/ad_limits) и выберите Страницу, лимитом которой хотите управлять. Страница может быть выбрана автоматически. Нажмите **Редактировать**. Увеличьте лимит объявлений для своего аккаунта и настройте его для других аккаунтов или пользователей, чтобы общее число объявлений равнялось лимиту для Страницы.
      * Если вы не владелец Страницы, перейдите в Ads Manager и [удалите](https://www.facebook.com/business/help/991073724257797) или [отключите](https://www.facebook.com/business/help/172764286113080) лишние объявления, чтобы не превышать установленный лимит. После этого вы сможете активировать рекламу с ошибками.
  * **Вы не превышали лимит, но по-прежнему видите сообщение об ошибке**. Откройте инструмент [Лимиты объявлений для Страницы](https://www.facebook.com/ads/ad_limits), чтобы узнать, насколько превышен лимит. Если он не превышен, возможно, показ объявлений завершен или вы уже удалили некоторые из них. Это значит, что сообщение об ошибке неактуально, и вы можете вернуться к списку своих объявлений, чтобы закрыть его. Подробнее о том, [как включить или выключить рекламу на Facebook](https://www.facebook.com/business/help/172764286113080).
  * **Человек с полным доступом к Странице установил для вас лимит в 0 объявлений**. Люди с полным доступом к Странице могут настраивать дополнительные ограничения для рекламных аккаунтов и бизнес-портфолио, чтобы задать лимит объявлений для определенных рекламодателей или запретить им публиковать новую рекламу. Если для вашего аккаунта установлен лимит в 0 объявлений, вы не сможете показывать рекламу для Страницы, пока он не будет увеличен. Подробнее о том, [как установить лимит объявлений при назначении Страницы партнеру](https://www.facebook.com/business/help/3210462009207540).
  * **У вас нет необходимых разрешений для показа рекламы на Странице**. Вы сможете создавать, удалять объявления и управлять ими, только если у вас есть доступ к Странице на уровне **рекламы**. Иначе вы сможете выбрать Страницу во время создания рекламы, но не сможете опубликовать объявление.
    * Если Страница добавлена в бизнес-портфолио, [запросите доступ к Странице в Meta Business Suite](https://www.facebook.com/business/help/183277585892925).
    * Если Страница не добавлена в бизнес-портфолио, [запросите доступ к Странице на уровне Facebook](https://www.facebook.com/help/531084028683914).
