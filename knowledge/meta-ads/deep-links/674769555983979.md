# Разница между метриками "Клики (все)" и "Клики по ссылке"
Клики по ссылкам — это количество переходов по ссылкам в выбранные места назначения и на другие ресурсы на платформах Meta или вне их. Например, человек нажимает в рекламе кнопку **В магазин** и переходит на ваш сайт. Примеры кликов по ссылкам:
  * клики по изображению или по кнопке призыва к действию в рекламе с целью "Трафик";
  * клики по URL в тексте рекламы;
  * клики по рекламе, которые ведут **в** полноэкранный контент, например Холст, подборку или форму для лидов;
  * переходы на сайты и в магазины приложений непосредственно **по** ссылкам из рекламы в Ленте.


Метрика "Клики (все)" включает в себя клики по ссылке, а также по другим частям рекламы (например, человек нажимает на название вашей Страницы). В метрике "Клики (все)" могут учитываться:
  * клики по ссылке;
  * отметки "Нравится", комментарии или репосты;
  * клики для перехода на Страницу Facebook или в профиль Instagram;
  * клики для открытия фото или видео в полноэкранном режиме.

Поскольку метрика "Клики (все)" включает в себя клики по ссылке, в ваших отчетах может показываться больше кликов, чем кликов по ссылке.
