# Создание отчетов по нескольким аккаунтам в инструменте "Отчеты по рекламе"
Вы можете создать отчет по нескольким рекламным аккаунтам и предоставить доступ к нему каждому из этих аккаунтов.
## Создание отчета по нескольким аккаунтам 
Чтобы создать отчет по нескольким аккаунтам:
  1. Откройте [бизнес-портфолио](https://business.facebook.com/).
  2. В меню навигации в левой части экрана выберите **Все инструменты**. Затем в разделе **Статистика и отчеты** нажмите **Отчеты по рекламе**. 
  3. Нажмите **Создать отчет**. Укажите название отчета. Найдите и выберите один или несколько рекламных аккаунтов по названию или ID либо установите флажки рядом с нужными аккаунтами.
  4. Выберите макет отчета. Нажмите **Продолжить**.
  5. Нажмите "Создать".


Преимущества отчетов по нескольким аккаунтам:
  * Вам будут доступны данные за последние 24 месяца. В отчете по одному аккаунту можно использовать данные за последние 37 месяцев.
  * Специально настроенные конверсии будут доступны на уровне бизнес-портфолио и предоставлены всем рекламным аккаунтам в отчете.
  * При создании отчета по нескольким аккаунтам с разными часовыми поясами по умолчанию будет выбран часовой пояс бизнес-портфолио.


**Примечание**. Не все функции, поддерживаемые при создании отчетов по одному рекламному аккаунту, доступны при создании отчетов по всем аккаунтам.
