# Доступные условия для автоматических правил
![](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=438223955063449&version=1666716831&transcode_extension=webp)
Начиная с октября 2023 года тип закупки "Охват и частота" будет называться "Резервирование". [_Подробнее о резервировании_](https://www.facebook.com/business/help/251123081984768).
Создавая правила для кампаний, групп объявлений или отдельных объявлений, вы выбираете условия, при которых они должны срабатывать.
## Доступные условия
При создании правила можно выбрать следующие условия:
  * [Цена за добавление платежной информации](https://www.facebook.com/business/help/483171132183948) — средняя цена одного [добавления платежной информации](https://www.facebook.com/help/250602782215458).
  * [Цена за добавление в корзину](https://www.facebook.com/business/help/233200980642968) — средняя цена одного [добавления в корзину](https://www.facebook.com/help/291357774946130).
  * [Цена за добавление в список желаний](https://www.facebook.com/business/help/261082664677776) — средняя цена одного добавления в список желаний.
  * [Цена за завершенную регистрацию](https://www.facebook.com/business/help/477382159341024) — средняя цена одной [завершенной регистрации](https://www.facebook.com/business/help/1764313906951681).
  * [Цена за начало оформления заказа](https://www.facebook.com/business/help/1925989814089259) — средняя цена одного [начала оформления заказа](https://www.facebook.com/help/826989194166740).
  * [Цена за лид](https://www.facebook.com/business/help/999694013547805) — средняя цена одного [лида](https://www.facebook.com/help/431390884009671).
  * [Цена за установку приложения](https://www.facebook.com/business/help/963812227161566) — средняя цена одной [установки приложения](https://www.facebook.com/help/2040391276031457).
  * [Цена за покупку](https://www.facebook.com/business/help/2057725494492853) — средняя цена одной [покупки](https://www.facebook.com/help/826924984184747)[](https://www.facebook.com/business/help/2057725494492853).
  * [Цена за результат](https://www.facebook.com/business/help/762109693832964) — средняя цена за [результат](https://www.facebook.com/help/611432918970668) [после просмотра вашей рекламы](https://www.facebook.com/business/help/762109693832964) (насколько экономически выгодным было достижение цели рекламной кампании).
  * [Просмотр контента на сайте](https://www.facebook.com/business/help/651089425035526) — количество событий "Просмотр контента", отслеженных пикселем на вашем сайте и [атрибутированных вашей рекламе](https://www.facebook.com/help/458681590974355)[](https://www.facebook.com/business/help/651089425035526).
  * [CPM (цена за 1 000 показов)](https://www.facebook.com/business/help/753932008002620) — средняя цена 1 000 [показов](https://www.facebook.com/help/675615482516035)[](https://www.facebook.com/business/help/753932008002620).
  * Прогнозируемый расход бюджета — прогнозируемая процентная доля бюджета, которая будет израсходована за время показа группы объявлений.
  * Часы с момента создания — время, которое прошло с момента создания кампании, группы объявлений или отдельного объявления.
  * [Частота](https://www.facebook.com/business/help/1546570362238584) — среднее количество просмотров рекламы одним [человеком](https://www.facebook.com/business/help/283579896000936). Это [приблизительная метрика](https://www.facebook.com/help/138838180128488)[](https://www.facebook.com/business/help/1546570362238584).
  * [Показы](https://www.facebook.com/business/help/675615482516035) — количество появлений вашей рекламы на экранах. 
  * [Охват](https://www.facebook.com/business/help/710746785663278) — количество людей, которые хотя бы раз увидели вашу рекламу. Отличие охвата от [показов](https://www.facebook.com/help/675615482516035) заключается в том, что последние могут включать просмотры рекламы одним и тем же человеком. Это [приблизительная метрика](https://www.facebook.com/help/138838180128488)[](https://www.facebook.com/business/help/1546570362238584).
  * [Результаты](https://www.facebook.com/business/help/611432918970668) — количество раз, когда реклама обеспечила результат, соответствующий ее цели и выбранным настройкам.
  * Название кампании — частичное или полное совпадение названия кампании.
  * Цель — цель кампании, которой принадлежит объект.
  * Время начала — UNIX-время начала для объекта.
  * Время окончания — UNIX-время окончания для объекта.
  * [Тип закупки рекламы](https://www.facebook.com/business/help/1033727169995080) — способ оплаты и таргетинга, который вы можете использовать в вашей кампании (динамические ставки, фиксированные ставки или тип закупки "Резервирование"). Сейчас это поле доступно только на уровне кампании.
  * Событие выставления счетов — событие выставления счетов для группы объявлений, которой принадлежит объект.
  * Цель оптимизации — цель оптимизации для группы объявлений, которой принадлежит объект.
  * [Результаты](https://www.facebook.com/business/help/611432918970668) — количество раз, когда реклама обеспечила результат, соответствующий ее цели и выбранным настройкам.
  * [Дневной бюджет](https://www.facebook.com/business/help/206296442738865) — средняя сумма, которую вы готовы ежедневно тратить на определенную группу объявлений. 
  * [Бюджет на весь срок действия](https://www.facebook.com/business/help/340985466050775) — сумма, которую вы готовы потратить в течение всего срока действия группы объявлений.
  * Предел затрат — лимит затрат для кампании, которой принадлежит объект.
  * Сумма ставки — сумма ставки для объекта.
  * Время создания — UNIX-время создания объекта.
  * Время обновления — UNIX-время обновления объекта.
  * [Место размещения](https://www.facebook.com/business/help/692605680769095) — место, где показывается реклама. На Facebook реклама может показываться в правом столбце и в Ленте на компьютерах и мобильных устройствах. Мы также можем показывать рекламу в Instagram, Audience Network и Messenger.
  * Тип бюджета — период обновления бюджета для группы объявлений, которой принадлежит объект.
  * Часы с момента создания — количество часов с момента создания объекта.
  * Прогнозируемый расход бюджета, % — прогнозируемая процентная доля бюджета группы объявлений, которая будет израсходована к концу графика показа.
  * Охват аудитории, % — прогнозируемая процентная доля охвата группы объявлений по отношению к размеру аудитории.
  * Время активности, с — количество секунд, прошедших с момента получения объектом статуса "Активно". Если объект неактивен, будет возвращено значение 0.
  * Текущее время, с — текущее UNIX-время.


## Дополнительные параметры условий
Рекламодатели могут задавать дополнительные критерии для отдельных условий. Например, можно применить метрику только к родительскому объекту или к временному диапазону, который отличается от временного диапазона для всего правила.
Дополнительные параметры условий:
  * Селектор "Префикс" в RBA, который называется "Дополнительные параметры фильтра".
  * Возможность выбора диапазона времени и объекта для полей статистики отдельно для каждого условия.


## Доступные логические выражения
Логические выражения позволяют вводить в условия собственные значения. При создании правила в условие можно добавить одно из следующих логических выражений:
  * **Больше, чем.**
  * **Меньше, чем.**
  * **В диапазоне.**
  * **Вне диапазона.**


## Диапазоны времени 
Вы можете изменить диапазон времени для своего условия:
  * **Диапазон времени** — количество дней для сбора данных, которое вы хотите охватить своим правилом.
