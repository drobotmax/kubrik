# Устранение проблем с массовым импортом при использовании пользовательской таблицы Excel
Если вы используете пользовательскую таблицу Excel (таблицу Excel не из списка шаблонов Excel в Ads Manager), убедитесь, что в ней есть те же столбцы, что в шаблоне Excel из Ads Manager. Также удалите все ID объявлений в своей таблице.
Чтобы сравнить пользовательскую таблицу Excel с шаблоном Excel из Ads Manager:
  1. Откройте [Ads Manager](https://www.facebook.com/adsmanager/manage).
  2. Выберите **Импорт и экспорт** ![](https://scontent-prg1-1.xx.fbcdn.net/v/t39.2365-6/76994270_2328676033928830_3668478494049828864_n.png?stp=dst-webp&_nc_cat=111&ccb=1-7&_nc_sid=e280be&_nc_ohc=0Cpl1Exj0usQ7kNvwGQvHuR&_nc_oc=Adrwtpe4tV2L3UntOeSWpRcn7EAP-s52mRISQeu7A1VI0kPm74yk1HCXDPwgCH16wj8&_nc_zt=14&_nc_ht=scontent-prg1-1.xx&_nc_gid=ZV14gyC9grUni4wWqaA2IQ&_nc_ss=7a389&oh=00_Afz4QGaD3Gnu6bpUpteCNPKvOafWxhhXWIWZd72cBiuS7g&oe=69E3B48E), а затем — **Скачать шаблон в формате Excel**. Если такого значка в меню нет, нажмите **Больше**.
  3. Сравните пользовательскую таблицу Excel с экспортированным из Ads Manager шаблоном Excel.
     * Все столбцы в экспортированном из Ads Manager шаблоне Excel должны быть в вашей пользовательской таблице Excel.
     * Если в вашей пользовательской таблице Excel есть ID объявлений, удалите их.
  4. Сохраните пользовательскую таблицу Excel как файл .TXT в формате Unicode.
  5. Нажмите **Экспорт и импорт** и выберите **Импорт рекламы** , чтобы импортировать пользовательскую таблицу Excel назад в Ads Manager.
