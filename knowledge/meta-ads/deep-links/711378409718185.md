# Рекомендации по работе с кампаниями Advantage+ для приложений
Кампании Advantage+ для приложений помогут без лишних усилий повысить результативность рекламы для продвижения приложения. Эта функция позволяет показывать лучшие варианты креативов самым подходящим аудиториям в наиболее эффективных местах размещения. Она поддерживает результативность рекламы при повышении бюджета, оптимизируя аудиторию, места размещения и бюджет кампании по продвижению приложений.
##  Рекомендации по созданию кампаний Advantage+ для приложений 
Ниже приведены рекомендации, которые помогут вам использовать кампании Advantage+ для приложений максимально эффективно.
  * **Создание кампаний**. Количество кампаний Advantage+ для приложений не ограничено. Вы можете запустить столько кампаний, сколько необходимо для достижения ваших маркетинговых целей, будь это управление несколькими компаниями или подкатегориями, оптимизация для нескольких событий в приложении, тестирование креативов или выход на разные рынки. Meta Ads Manager поможет не дублировать кампании с одними и теми же настройками.
  * **Выбор целей результативности**. Приведенная ниже таблица поможет вам выбрать нужный тип оптимизации кампаний Advantage+ для приложений с учетом предыдущих настроек оптимизации и таргетинга, а также основных ключевых показателей эффективности.

  * **Выбор ставки**. Для достижения наилучших результатов рекомендуем использовать тип **Минимальная цена** без предельной ставки. Такая стратегия помогает контролировать расходы, но может ограничить количество полученных результатов, поскольку по ходу кампании количество возможностей для показа будет уменьшаться. При использовании [стратегии "Минимальная цена" без предельной ставки](https://www.facebook.com/business/help/317364032619502) кампании Advantage+ для приложений обычно приносят лучшие результаты и расходуют весь бюджет.
  * **Обновление креативов**. За один раз можно добавить до 50 фото или видео. Для поддержания максимальной результативности рекламы загрузите как можно больше креативов и [обновляйте их несколько раз в месяц](https://www.facebook.com/business/m/test-ads-on-facebook), чтобы избежать снижения интереса к ним. Вы также можете использовать [_интерактивную рекламу_](https://www.facebook.com/business/help/2138506633095259) и [объявления в формате Instant Experience](https://www.facebook.com/business/help/614812385653152).  
**Примечание**. Мы не рекомендуем вносить изменения в креативы, пока кампания находится на [этапе обучения](https://www.facebook.com/business/help/112167992830700).
  * **Отчеты по результативности креативов**. Показывая кампании Advantage+ для приложений, вы не сможете просматривать отчетную статистику по определенным местам размещения. Эти параметры нельзя отредактировать во время создания кампании, поэтому они недоступны и в отчетах. Поддерживается разбивка отчетности по возрасту, полу, возрасту и полу, стране, устройству показов, платформе, платформе и устройству, а также месту размещения. Кампаниям Advantage+ для приложений, направленным на пользователей устройств Android, будут доступны разбивки по Designated Marketing Area (DMA). Анализ разбивок помогает принимать более взвешенные решения об оптимизации и креативной стратегии. Вы можете посмотреть метрики по [каждому элементу креатива](https://www.facebook.com/business/help/716015512512235), чтобы узнать результативность изображений, видео, заголовков и вариантов основного текста. Такие метрики можно посмотреть для каждого объявления.
  * **Пересечение аудиторий**. [Пересечение кампаний с таргетингом на одну и ту же аудиторию](https://www.facebook.com/business/help/1679591828938781) может оказать негативное влияние на показ. Поэтому не стоит одновременно показывать одной и той же аудитории несколько кампаний Advantage+ для приложений. Чтобы проверить, подходит ли вам кампания Advantage+ для приложения, проведите A/B-тестирование и сравните ее результаты с результатами рекламы с более узкими настройками.
