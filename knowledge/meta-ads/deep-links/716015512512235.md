# Рекомендации и отчетность по креативам в кампаниях Advantage+ для приложений
Кампании Advantage+ для приложений помогут без лишних усилий повысить результативность рекламы для продвижения приложения. Эта функция позволяет показывать лучшие варианты креативов самым подходящим аудиториям в наиболее эффективных местах размещения. Она поддерживает результативность рекламы при повышении бюджета, оптимизируя кампании по продвижению приложений в целом, а также креативы и аудитории в частности.
При создании кампаний Advantage+ для приложений рекомендуем использовать максимальное количество креативов, чтобы достичь лучших результатов. Вы можете добавить до 50 уникальных фото или видео, до 5 вариантов основного текста и до 5 заголовков. Вы можете создать до 50 объявлений одновременно, что исключает необходимость дублировать рекламу. Рекомендуем менять креатив 2–4 раза в месяц, чтобы поддерживать результативность рекламы.
## Просмотр отчетов по результативности креативов в кампаниях Advantage+ для приложений
Вы можете посмотреть метрики, чтобы узнать результативность изображений, видео, заголовков и вариантов основного текста.
Чтобы узнать результативность каждого элемента креатива в кампании Advantage+ для приложения:
  1. Откройте [_Meta Ads Manager_](https://www.facebook.com/ads/manage/campaigns).
  2. Выберите рекламный аккаунт, связанный с нужным приложением.
  3. Выберите кампании, которые хотите просмотреть.
  4. Нажмите на вкладку **Объявления**.
  5. Вы увидите метрики результативности для выбранного объявления.
  6. Чтобы продолжить настройку метрик, откройте меню **Столбцы** и выберите один из предустановленных вариантов или нажмите **Настроить столбцы**.


**Примечание**. Если вы выберете пользовательскую группу столбцов, сохраните ее как отдельный отчет. Так вы сможете использовать его для дополнительных кампаний.
