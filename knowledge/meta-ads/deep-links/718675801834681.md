# Устранение проблем с пикселем Meta для рекламы ассортимента автосалона
Для запуска рекламы ассортимента автосалона нужно создать каталог, пиксель Meta и шаблон рекламы. Чтобы успешно настроить пиксель, изучите руководство ниже. 
## Часто задаваемые вопросы
**Что, если я хочу отслеживать действия, которые не соответствуют стандартному событию?**
Мы расширили [список поддерживаемых стандартных событий](https://www.facebook.com/business/help/402791146561655) с учетом тех событий на сайте, которые нужны для продажи автомобилей, но учесть всё невозможно. Поэтому мы рекомендуем создать [специально настроенное событие](https://developers.facebook.com/docs/facebook-pixel/api-reference#events) для каждого дополнительного значимого действия на сайте. Вы можете оптимизировать показ рекламы для специально настроенных событий, используя цели "Вовлеченность", "Лиды" и "Продажи".
**Как по отдельности отслеживать несколько типов действий в рамках стандартного события (например, лида)?**
Чтобы по отдельности отслеживать несколько действий в рамках одного стандартного события (например, лида), вы можете установить специальные параметры и получать информацию, уникальную для каждого действия. 
В описанном случае можно добавить параметр в формате eventname_type ("названиесобытия_тип"). Список стандартных и обязательных событий для рекламы ассортимента автосалона, а также рекомендации по разным типам событий доступны в [документации Meta for Developers](https://developers.facebook.com/docs/marketing-api/dynamic-ads-auto/auto-event). Предположим, вы хотите по отдельности отслеживать 2 типа события "Лид": записи на тест-драйв и запросы специальной цены. Вы можете настроить событие "Лид" и параметр для каждого из действий указанным ниже способом.
1. Записи на тест-драйв:

```

fbq('track', 'Lead', {
lead_type: 'Test Drive'});
```

2. Запросы специальной цены:

```

fbq('track', 'Lead', {
lead_type: 'Finance Quote'});
```

Убедитесь, что параметры настроены, а события активно срабатывают на сайте. Затем создайте специально настроенную конверсию на основе события и добавьте правила включения нужных параметров и значений, следуя [этому руководству](https://www.facebook.com/business/help/462806641140703?id=1205376682832142). Снова воспользовавшись приведенным примером (для отслеживания 2 типов события "Лид"), вы можете:
  1. Создать специально настроенную конверсию на основе события "Лид", выбрав параметр lead type ("тип лида") и значение Test Drive ("тест-драйв").
  2. Создать специально настроенную конверсию на основе события "Лид", выбрав параметр lead type ("тип лида") и значение Finance Quote ("запрос цены").


**Как отслеживать действия, связанные с определенной маркой или моделью транспортного средства, либо выполнять оптимизацию для таких действий?**
Вы можете настраивать события и параметры таким образом, чтобы отслеживать действия, связанные с определенной маркой или моделью транспортного средства, а также оптимизировать показ рекламы для таких действий. Предположим, вы хотите отслеживать просмотры страницы с информацией о модели Acura MDX или выполнять оптимизацию для этого действия. Чтобы реализовать это, следуйте инструкциям ниже.
  1. Добавьте параметры make ("марка") и model ("модель") в код события "Просмотр контента" пикселя Meta на всех страницах с информацией об автомобиле, чтобы учесть эти значения. 
  2. Создайте специально настроенную конверсию на основе события "Просмотр контента", выбрав параметр model ("модель") и значение MDX (инструкции доступны [здесь](https://www.facebook.com/business/help/462806641140703?id=1205376682832142)). Также при желании можно добавить второе правило, чтобы указать марку ('make' = 'Acura').
  3. Когда вы создадите специально настроенную конверсию и дадите ей название, при [настройке столбцов](https://www.facebook.com/business/help/1452248935049010) в инструменте "Отчеты по рекламе Meta" появится столбец с ее названием. Вы также увидите эту конверсию, когда при создании рекламы будете выбирать событие конверсии для оптимизации. 
В одном рекламном аккаунте можно создать до 100 специально настроенных конверсий, поэтому мы рекомендуем использовать их только для самых важных для вас марок и моделей.


**Я хочу привлечь как можно больше людей на страницы, посвященные определенной модели (например, Nissan Sentra). Как это лучше сделать?**
Убедитесь, что добавили событие "Просмотр контента" на каждую страницу, посвященную Sentra, и указали необходимые параметры (в соответствии с таблицей). 
  * Не забудьте добавить параметр "model" ("модель"). Ваш разработчик может настроить этот код таким образом, чтобы название модели показывалось как значение этого параметра. Если всё настроено правильно, то при каждом посещении посвященных Sentra страниц пиксель будет регистрировать событие ViewContent и определять модель как Sentra.
  * Другим полезным параметром может быть page_type ("тип страницы"), для которого значением будет VDP ("страница с информацией о транспортном средстве"). 

После этого вы создадите в Meta Events Manager **специально настроенную конверсию** на основе этих параметров. 
Поскольку ваша цель — привлечь на страницы как можно больше людей, мы рекомендуем расширить целевую аудиторию и выбрать для кампании цель "Вовлеченность", "Лиды" или "Продажи". При выборе события конверсии для оптимизации вы можете использовать специально настроенную конверсию, которую только что создали для просмотров страниц о Sentra, или более общее событие — "Просмотр контента" (если использование предыдущего варианта принесло меньше показов, чем планировалось).
**Какие события следует использовать для создания аудитории,**отчетности** и оптимизации?**
Выбор событий для каждой из этих целей целиком зависит от ключевых показателей эффективности и приоритетов вашей компании. Однако мы можем дать некоторые рекомендации, основанные на опыте других автосалонов.
**Создание аудитории**
  * Ретаргетинг — создавайте пользовательские аудитории на основе данных о посетителях вашего сайта для ретаргетинга с учетом действий и поведения [людей](https://www.facebook.com/business/help/283579896000936) на сайте. Пример аудитории: люди, которые посетили страницу с информацией об определенной модели автомобиля, но не записались на тест-драйв.
  * Привлечение новых клиентов — создавайте похожие аудитории и охватывайте потенциальных клиентов, похожих на людей, которые совершают ключевые действия на вашем сайте. Пример аудитории: люди, которые не посещали ваш сайт, но очень похожи на тех, кто в последние 30 дней записывался на тест-драйв онлайн.

**Отчетность**
  * Любое стандартное событие, которое вы настраиваете для сайта, автоматически учитывается в отчетах. Вы можете использовать функцию [настройки столбцов](https://www.facebook.com/business/help/1452248935049010) в инструменте "Отчеты по рекламе", чтобы выбрать событие, результаты по которому хотите видеть в отчетах. Для каждого события можно добавить параметр cost per result ("цена за результат"). 

**Оптимизация**
  * Выбирать события для оптимизации показа рекламы следует на основе действий, наиболее важных для вашей компании (от которых, соответственно, вы хотите получить максимальную выгоду). Это могут быть, например, записи на тест-драйв или предварительные регистрации в связи с выпуском новой модели. 
**Примечание**. Если важное для вас событие происходит редко (например, покупка автомобиля или покупка дорогостоящего товара в рассрочку), рекомендуем выполнять оптимизацию для более частого события (например, добавления в корзину). В ином случае существует вероятность недостаточного показа, поскольку нашим системам машинного обучения нужно анализировать значительное количество данных о конверсиях, чтобы находить подходящих людей.
