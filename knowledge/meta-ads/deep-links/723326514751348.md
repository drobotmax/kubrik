# Использование специально настроенных конверсий для регистрации стандартных событий без изменения кода
Если на вашем сайте установлен базовый код пикселя Meta, вы можете создать специально настроенную конверсию для регистрации стандартных событий без изменения кода. [Подробнее о специально настроенных конверсиях](https://www.facebook.com/business/help/780705975381000).
## Принцип работы
Чтобы регистрировать стандартные события без изменения кода, в качестве события конверсии выберите вариант **Весь трафик для URL**. Параметр "Весь трафик для URL" определяется с помощью событий просмотра страницы PageView, которые по умолчанию входят в базовый код пикселя Meta.
![Meta Pixel base code highlighting page view events](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=198566762443690&version=1738696630&transcode_extension=webp)
Так вы сможете регистрировать посещения всех URL, для которых установлено событие PageView, в том числе веб-страниц, на которые пользователи заходят на разных этапах воронки продаж.
Чтобы указать, какие веб-страницы соответствуют этому действию пользователя, вам необходимо настроить правила для URL. Вы также можете выбрать категорию стандартного события, наиболее схожую с действием, для которого вы выполняете оптимизацию. После этого мы сможем показывать вашу рекламу [людям](https://www.facebook.com/business/help/283579896000936), которые с большей вероятностью выполнят это действие.
## Прежде чем начать
[Установите пиксель Meta на своем сайте](https://www.facebook.com/business/help/952192354843755).
## Настройка стандартных событий с помощью специально настроенных конверсий без редактирования кода
Чтобы с помощью специально настроенных конверсий задать стандартные события, не редактируя код:
  1. Откройте [Meta Events Manager](https://business.facebook.com/events_manager2/list).
  2. Нажмите **Специально настроенные конверсии** ![](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=1725407184250087&version=1773385167&transcode_extension=webp) в меню в левой части экрана.
  3. Выберите **Создать специально настроенную конверсию**.
  4. Укажите название специально настроенной конверсии.
  5. Добавьте описание (необязательно).
  6. Выберите источник данных.
  7. Для параметра **Событие конверсии** выберите **Весь трафик для URL**.
  8. Для пункта **Выберите стандартное событие для оптимизации** рекомендуем использовать категорию, указанную по умолчанию. Если вы хотите использовать другую категорию, выберите ту, которая лучше всего описывает оптимизируемое событие конверсии. [Подробнее](https://www.facebook.com/business/help/2375212726097833). 
  9. Настройте правила для URL. 
    1. В первом раскрывающемся меню в разделе **Правила** выберите **URL**.
    2. Выберите **содержит** , **не содержит** или **равен** во втором раскрывающемся меню, чтобы отфильтровать события для определенных действий. 
       * Если вы выберете **URL содержит** или **не содержит** , в текстовое поле введите часть URL, которая представляет вашу специально настроенную конверсию.
       * Если вы выберете **URL равен** , скопируйте и вставьте в текстовое поле точный URL, который представляет вашу специально настроенную конверсию. Укажите домен полностью, включая https://www.
  10. Чтобы добавить другие правила, нажмите значок **+**. 
  11. Если для событий в вашей специально настроенной конверсии ещё не указана ценность конверсии, установите флажок рядом с параметром **Указать ценность конверсии** и введите целое число без каких-либо дополнительных символов (необязательно).
  12. Нажмите **Создать**.


Специально настроенная конверсия создана.
**Примечание**. В рекламный аккаунт можно добавить не более 100 специально настроенных конверсий, но вы можете удалять конверсии, которые вам больше не нужны, и создавать новые.
##  Активация специально настроенных конверсий
Чтобы активировать созданную специально настроенную конверсию, перейдите на страницу своего сайта, к которой относится конверсия. Затем откройте вкладку "Специально настроенные конверсии" в Events Manager и проверьте, сработала ли конверсия должным образом (если да, вы увидите статус **Активно**).
## Отчеты и оптимизация
Чтобы узнать, сколько раз происходила определенная конверсия, перейдите на вкладку "Специально настроенные конверсии" в Events Manager. [Подробнее о разнице в подсчете событий в Meta Ads Manager, Отчетах по рекламе и Events Manager](https://www.facebook.com/business/help/337196340694086).
При создании рекламной кампании в Ads Manager вы также можете выбрать специально настроенную конверсию в качестве цели по результативности.
