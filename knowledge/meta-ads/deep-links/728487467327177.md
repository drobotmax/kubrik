# Начало работы с Meta Ads Manager for Excel
Чтобы использовать Meta Ads Manager для Excel, выполните вход с помощью своего аккаунта Facebook. После этого вы сможете управлять отчетами по рекламе.
## Прежде чем начать
  * Если вы экспортировали данные о результативности рекламы из Ads Manager в Ads Manager для Excel, возможно, вы уже выполнили вход в эту надстройку. Если это так, в надстройке вы увидите свои фото профиля и имя на Facebook.
  * Ads Manager для Excel поддерживает последнюю версию Excel 2016 для Windows и Mac. Если ваша программа устарела или при создании отчета появляется сообщение об ошибке, перейдите на сайт Microsoft и обновите Excel. 


## Вход в Meta Ads Manager for Excel
  1. Откройте Excel на компьютере.
  2. Откройте вкладку **Вставка**.
  3. Раскройте меню **Мои надстройки** ![](https://scontent-prg1-1.xx.fbcdn.net/v/t39.2365-6/15730161_102913770218804_4921325483699732480_n.png?stp=dst-webp&_nc_cat=108&ccb=1-7&_nc_sid=e280be&_nc_ohc=d0hPm0C52ZgQ7kNvwERbGog&_nc_oc=AdpRrzrHqCDcsPmq_S2ZQ-QNf8iDFXdOCmI-uUQk31PBFB-7TuXdSEDe4iUMrAOCj24&_nc_zt=14&_nc_ht=scontent-prg1-1.xx&_nc_gid=FI4Yay5y0f3pDffLNJqDXw&_nc_ss=7a389&oh=00_AfzMt7NTkJM0ZQ8LXnri7cbsK7wibdXGp2X0eBuXYGXXkQ&oe=69E3BA21) и выберите **Ads Manager для Excel**.
![](https://scontent-prg1-1.xx.fbcdn.net/v/t39.2365-6/77938381_1365763650263487_2373971270816497664_n.jpg?stp=dst-webp&_nc_cat=111&ccb=1-7&_nc_sid=e280be&_nc_ohc=_n61a-6lONcQ7kNvwHBVs54&_nc_oc=AdqiyaaPWcmSrxsCzPtXHEUTFAQEmxcA8Rn7-5bLFb4d8Sdbz75dHf-O6hMmShFeYTk&_nc_zt=14&_nc_ht=scontent-prg1-1.xx&_nc_gid=FI4Yay5y0f3pDffLNJqDXw&_nc_ss=7a389&oh=00_AfzXmcEw19yWdKGmbnNrCHQdiGHWMhgljZBIUPGUF-fGCA&oe=69E3AF65)
**Примечание**. Если в списке надстроек нет Ads Manager для Excel, перейдите в магазин Microsoft Office Store и загрузите ее.
  4. В Ads Manager для Excel нажмите **Вход через Facebook**.
![](https://scontent-prg1-1.xx.fbcdn.net/v/t39.2365-6/17625920_185622875285114_4066864181364654080_n.png?stp=dst-webp&_nc_cat=100&ccb=1-7&_nc_sid=e280be&_nc_ohc=1t8BBbRMtpwQ7kNvwHZ8qN7&_nc_oc=Adp1rK2ZKOFD0purfb4QVYItQl7mWEjAFWobwchyF5uY-fLtHLTxZmBKhGolnTatQqc&_nc_zt=14&_nc_ht=scontent-prg1-1.xx&_nc_gid=FI4Yay5y0f3pDffLNJqDXw&_nc_ss=7a389&oh=00_AfyRbyF2zpkvArXWkV_RvnQIhdmTVKZT0oO9OSYKdCbhzw&oe=69E3A60F)
  5. После этого
  6. Нажмите значок ![](https://scontent-prg1-1.xx.fbcdn.net/v/t39.2365-6/17636717_251431088659396_3346735982736048128_n.png?stp=dst-webp&_nc_cat=105&ccb=1-7&_nc_sid=e280be&_nc_ohc=i_skpHuHvcgQ7kNvwGexzoA&_nc_oc=Adrt2L6Jzf8ILrTcRXzRWwlT1PA-XUVwFyFAzRGS5HqraJWHwEI_u1SXNnUOWYBnFbc&_nc_zt=14&_nc_ht=scontent-prg1-1.xx&_nc_gid=FI4Yay5y0f3pDffLNJqDXw&_nc_ss=7a389&oh=00_Afz-OziS2kowYbmkmNszJqi0-GJb9ObeEZNAgP1la7IPAw&oe=69E3B28D), чтобы скопировать код.
  7. Откройте в браузере страницу [www.facebook.com/device](https://www.facebook.com/device) и вставьте код в поле.
**Примечание**. Если войти на Facebook не удается, введите электронный адрес и пароль, связанные с вашим аккаунтом. Проверьте правильность введенных данных. Если вы не помните свой текущий пароль, то можете [изменить его](https://www.facebook.com/help/213395615347144?helpref=faq_content).
  8. Нажмите **Далее** и следуйте инструкциям на экране, чтобы завершить вход в Ads Manager для Excel.

## Будьте в курсе новостей от Meta for Business
Укажите электронный адрес, на который вы хотите получать новости о мероприятиях и продуктах, а также другую информацию от Meta for Business. 
Отправляя эту форму, вы соглашаетесь получать по электронной почте маркетинговые материалы от Meta, в том числе новости, информацию о мероприятиях, обновления и специальные предложения. Вы можете отказаться от рассылки в любой момент. Для этого нужно, например, нажать ссылку для отмены подписки в нашем электронном письме.
