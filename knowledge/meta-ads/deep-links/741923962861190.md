# Советы по созданию групп товаров в каталоге и управлению ими
Группа товаров — это совокупность позиций в каталоге, которую можно использовать для показа рекламы. Группу товаров можно создать в Commerce Manager или Meta Ads Manager.
## Фильтры для групп товаров: принцип работы
Во время [создания группы товаров](https://www.facebook.com/business/help/620275848114281) фильтры помогут вам определить, какие позиции будут добавлены в нее из каталога. Вам нужно будет выбрать свойство товаров (например, название) и фильтр, который вы хотите к нему применить.  
| Фильтр  | Описание  |  
| --- | --- |  
| **—**  | Учитывает все товары, которые точно отвечают поисковому запросу. Например, фильтрация **—** по цвету со значением "синий" включит в группу только товары синего цвета.  |  
| **соответствует одному из этих вариантов**  | Учитывает все товары, которые точно отвечают хотя бы одному поисковому запросу. Например, фильтрация **соответствует одному из этих вариантов** по цветам "красный", "зеленый", "синий" охватит все товары, значение цвета которых соответствует одному или нескольким из указанных запросов.  |  
| **не**  | Учитывает все товары, которые полностью не отвечают поисковому запросу. Например, фильтрация **не** по материалу "кожа" включит в группу только те товары, которые не изготовлены из кожи. Однако товары из материала "искусственная кожа" всё равно будут учтены.  |  
| **содержит**  |  Учитывает все товары, которые содержат поисковый запрос. Этот запрос должен соответствовать целому слову, а не его части. Следуйте советам ниже.
  * Если поисковый запрос содержится в более длинном слове, фильтр не будет учитывать это слово. Например, фильтрация **содержит** по названию "скейт" не покажет результаты для значения "скейтборд" и наоборот.
  * Учитывайте множественное число слов. Например, фильтрация **содержит** по названию "ботинки" не покажет результаты для значения "ботинок" и наоборот.
  * Учитывайте цифры. Например, фильтрация по пользовательскому номеру 1 с помощью параметра **содержит** и значения "2024" не покажет результаты для значения "120243" и наоборот.

При добавлении в запрос знаков препинания или специальных символов следуйте советам ниже.
  * Поскольку пробел и определенные знаки (,?@$%^*()+=~[{}];"<>|—/®™【】) считаются разделителями, содержащие их слова будут рассматриваться как отдельные. Например, фильтрация **содержит** по названию "робот+пылесос" покажет результаты одновременно для значений "робот" и "пылесос".
  * Знаки &:/._#- также являются разделителями, но фильтрация покажет результаты и для целого слова. Например, фильтрация "содержит" по названию "робот-пылесос" покажет результаты одновременно для значений "робот", "пылесос" и "робот-пылесос". Фильтрация **содержит** по внутреннему ярлыку "осень_2026" покажет результаты одновременно для значений "осень", "2026" и "осень_2026".
  * Апостроф (') не считается разделительным знаком. Например, фильтрация **содержит** по названию "Johnson's" не покажет результаты для значения "Johnson" и наоборот.
  * Если поисковый запрос содержит несколько слов, результат поиска должен будет содержать все эти слова. При этом их порядок не имеет значения. Например, фильтрация **содержит** по названию "джинсы синие" покажет результаты для значения "синие укороченные джинсы", но не для "джинсы" или "черные джинсы".
  * Поскольку знак > считается разделительным, этот фильтр не работает с поисковыми запросами, которые содержат знак >, например категориями товаров. Например, фильтрация "содержит" по категории товара в Google "Уход за собой > Косметика" будет недействительной. Перепишите поисковый запрос в виде самостоятельных слов (например, чтобы фильтр **содержит** использовал категорию товара в Google "косметика") или примените фильтр **начинается с**.

 |  
| **не содержит**  |  Учитывает все товары, которые не содержат поисковый запрос. Этот запрос должен соответствовать целому слову, а не его части. В описании к параметру **содержит** можно узнать, что мы подразумеваем под словом. Этот фильтр показывает для товара все результаты, которые не включают в себя целый поисковый запрос. Например:
  * Фильтрация **не содержит** по названию "вязаный свитер" включит в группу товары под названием "полосатый свитер", но не "толстый белый вязаный свитер".
  * Фильтрация **не содержит** по названию "свитер" не включит в группу товаров ни "вязаный свитер", ни "толстый белый вязаный свитер".

 |  
| **начинается с**  |  Учитывает все товары, которые начинаются с поискового запроса. Например, фильтрация **начинается с** по категории товара в Google "Одежда и аксессуары" охватит все товары, начиная с этой категории, например "Одежда и аксессуары > Одежда > Платья" и "Одежда и аксессуары > Сумки, кошельки и чехлы > Сумки". Этот фильтр доступен только для свойства "Категория товара в Google". Для других свойств используйте фильтры **содержит** или **—**.  |  
| **больше, чем**  |  Учитывает все товары с числовыми значениями, которые превышают значение поискового запроса. Например, фильтрация **больше, чем** по цене "50" охватит все товары с ценой выше 50. Этот фильтр не поддерживает числа с десятичным разделителем. **Примечание.** Чтобы охватить акционные товары, выполните фильтрацию **больше, чем** по цене со скидкой и укажите значение "0". Это покажет все товары со скидкой в вашей группе товаров.  |  
| **больше или равно**  | Учитывает все товары с числовыми значениями, которые превышают значение поискового запроса или равны ему. Например, фильтрация **больше или равно** по цене "50" охватит все товары с ценой, значение которой будет больше или равно 50. Этот фильтр не поддерживает числа с десятичным разделителем.  |  
| **меньше, чем**  | Учитывает все товары с числовыми значениями, которые ниже значения поискового запроса. Например, фильтрация по пользовательскому номеру 0 с помощью параметра **меньше, чем** и значения "100" охватит все товары с числовым значением пользовательского номера 0, которое будет меньше 100. Этот фильтр не поддерживает числа с десятичным разделителем.  |  
| **меньше или равно**  | Учитывает все товары с числовыми значениями, которые ниже значения поискового запроса или равны ему. Например, фильтрация по пользовательскому номеру 0 с помощью параметра **меньше или равно** и значения "100" охватит все товары с числовым значением пользовательского номера 0, которое будет меньше или равно 100. Этот фильтр не поддерживает числа с десятичным разделителем.  |  
| **присутствует**  | Учитывает все товары с выбранным свойством. Например, фильтрация **присутствует** по видео охватит все товары, для которых есть видео.  |  
| **не присутствует**  | Учитывает все товары без выбранного свойства. Например, фильтрация **не присутствует** по видео охватит все товары, для которых нет видео.  |  
## Советы по фильтрации групп товаров
  * Фильтры не чувствительны к регистру.
  * Используйте большую группу товаров, чтобы наша система могла показывать в рекламе наиболее актуальные для аудитории позиции. Не сужайте фильтры слишком сильно, поскольку это приводит к ограничению рекомендаций товаров. При создании группы можно включить параметр **Добавить рекомендуемые товары** , чтобы добавлять в нее наиболее популярные позиции или товары с недавними конверсиями. [Подробнее об использовании рекомендаций товаров в рекламе.](https://www.facebook.com/business/help/120193522268735)
  * Не выполняйте для рекламы фильтрацию **—** со значением "в наличии". Товары, которых нет в наличии, а так не показываются в рекламе.
  * Вы можете указать, как должны совпадать товары: по **всем фильтрам** или **хотя бы одному фильтру**. Убедитесь, что вы выбрали правильный вариант.
  * Вы можете создавать собственные фильтры. Для этого добавьте индивидуальную информацию о своих товарах с помощью [внутренних ярлыков и пользовательских номеров](https://www.facebook.com/business/help/120325381656392?id=725943027795860#Optional). Позже эти свойства можно будет выбрать во время фильтрации. Если вы фильтруете группы товаров по пользовательским меткам, рекомендуем перейти на **внутренние ярлыки**. При обновлении ярлыков товар не проверяется на соответствие правилам.
  * Вы не сможете отфильтровать товары по свойствам, если не предоставили соответствующую информацию. Например, если вы не указали для товаров свойство цвета, применение фильтра по цвету не даст результатов.
  * Если в вашем каталоге очень много позиций, не создавайте группы товаров с помощью большого количества фильтров **содержит** или **не содержит** , иначе фильтрация может выполняться долго.


## Советы по добавлению достаточного количества позиций в группы товаров
  * Следите за тем, чтобы в группе товаров всегда было достаточно позиций для показа в рекламе. Реклама в формате подборки должна содержать не менее 4 товаров, а в формате кольцевой галереи — не менее 2. Это должны быть разные товары (варианты одной позиции засчитываются как один товар). Позиции также не могут быть [недействительными](https://www.facebook.com/business/help/2687033028210472), то есть содержать товары, которых нет в наличии, архивированные товары, нарушать правила и т. д. 
  * Фильтры групп товаров работают динамично, поэтому группы время от времени могут обновляться вместе с каталогом. Например, вы можете случайно удалить все или некоторые товары из группы.
  * Если в группе есть несколько вариантов одной позиции, например товары разного размера или цвета, то учтите, что в каждом объявлении появится только один из этих вариантов. [Подробнее о вариантах.](https://www.facebook.com/business/help/2256580051262113)
  * Загружайте разные изображения для каждого товара. Если одно изображение используется для нескольких товаров в группе, в рекламе будет показываться лишь один из этих товаров.

Если в группе окажется недостаточно товаров, мы можем приостановить показ вашей рекламы. В таком случае вы увидите соответствующее сообщение в Ads Manager. [Подробнее об устранении проблем с недостаточным количеством позиций в группе товаров.](https://www.facebook.com/business/help/599014736904594)
