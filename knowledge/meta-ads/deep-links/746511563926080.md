# О шлюзе Conversions API для нескольких аккаунтов
Шлюз Conversions API для нескольких аккаунтов — это функция шлюза Conversions API. Она позволяет один раз настроить шлюз, а затем подключиться к нему и управлять несколькими компаниями с помощью этой конфигурации. Шлюз Conversions API позволяет легко и быстро настроить Conversions API. [Подробнее о шлюзе Conversions API](https://www.facebook.com/business/help/387152639648383).
После настройки шлюза Conversions API для нескольких аккаунтов вы можете пригласить другие компании подключиться к нему. Когда они примут ваше приглашение, то смогут подключить источники данных своих сайтов и получить преимущества шлюза Conversions API без потребности в самостоятельной настройке или управлении. Каждый владелец компании может использовать службы шлюза Conversions API через собственные аккаунты. Узнать больше о принципе работы шлюза Conversions API для нескольких аккаунтов можно на сайте [Meta for Developers](https://developers.facebook.com/docs/marketing-api/conversions-api/guides/gateway-multiple-accounts).
Потенциальные преимущества использования шлюза Conversions API:
  * **Расширьте портфолио своих услуг и приносите своим клиентам больше результатов**. Агентства, партнеры, торговые посредники и другие компании могут предоставлять шлюз Conversions API в виде услуги, чтобы диверсифицировать свои предложения и повышать результативность кампаний своих клиентов.
  * **Упростите разным аккаунтам доступ к Conversions API**. После того, как хост настроит шлюз Conversions API для нескольких аккаунтов, другие компании могут использовать службы шлюза без потребности в создании или содержании собственной инфраструктуры. Хост шлюза Conversions API может легко интегрировать с Conversions API несколько клиентов.
  * **Настраивайте Conversions API легко и просто**. Шлюз Conversions API — это способ самостоятельной настройки, который позволит быстро подключить Conversions API без большого объема технических ресурсов. При этом обеспечивается автоматическая дедупликация между браузером и событиями сервера. Подробнее о [дедупликации](https://www.facebook.com/business/help/823677331451951) и [шлюзе Conversions API](https://www.facebook.com/business/help/387152639648383).
  * **Повышайте результативность**. Использование шлюза Conversions API предоставляет такие преимущества, как понижение цены за результат, повышение надежности и оптимизация измерения. [Подробнее об Conversions API](https://www.facebook.com/business/help/2041148702652965).
  * **Понизьте расходы на общее использование ресурсов**. К одной конфигурации шлюза можно подключить больше одного пикселя Meta, что позволяет сэкономить средства за счет распределения вычислительных ресурсов между несколькими компаниями.


Сейчас шлюз Conversions API для нескольких аккаунтов доступен компаниям-хостам, которые уже используют или могут создать аккаунт Amazon Web Services (AWS) или аккаунт на платформе Google Cloud (GCP). Создать новый аккаунт AWS или GCP можно за считанные минуты.
