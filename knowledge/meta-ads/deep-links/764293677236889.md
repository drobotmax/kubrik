# Изменение лимита затрат для рекламного аккаунта
![](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=438223955063449&version=1666716831&transcode_extension=webp)
Если в вашем рекламном аккаунте настроено [использование доступных средств](https://www.facebook.com/business/help/350519228479855), вы не сможете установить для него лимит затрат.
Лимит затрат для рекламного аккаунта можно изменить в настройках оплаты. По достижении лимита затрат показ ваших объявлений будет приостановлен, а средства аккаунта перестанут расходоваться. Чтобы сразу возобновить показ рекламы, измените лимит затрат рекламного аккаунта.
Лимит затрат для рекламного аккаунта — это максимальная сумма денег, которую ваш рекламный аккаунт может потратить на все кампании. Вы можете сбросить лимит затрат для рекламного аккаунта вручную или автоматически, например в конце месяца.
**Примечание**. Вы сами устанавливаете лимит затрат для рекламного аккаунта. Если ваша реклама приостановлена и бюджет не израсходован или лимит затрат для рекламного аккаунта не достигнут, возможно, вы исчерпали [дневной лимит затрат](https://www.facebook.com/business/help/563129151097553), который устанавливается Meta. Дневной лимит затрат сбрасывается каждый день.
## Прежде чем начать
Чтобы изменить лимит затрат для рекламного аккаунта, вы должны быть его администратором. [Подробнее о разрешениях и ролях для рекламы](https://www.facebook.com/business/help/155909647811305).
## Изменение лимита затрат для рекламного аккаунта
  1. Перейдите в [настройки платежей](https://www.facebook.com/ads/manager/account_settings/account_billing/) в Ads Manager. Вам может потребоваться снова ввести пароль. 
  2. Прокрутите вниз до раздела **Лимит затрат для аккаунта** и нажмите **Параметры** ![](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=250425836038370&version=1773072306&transcode_extension=webp).
  3. Нажмите **Изменить**.
  4. Введите новый лимит затрат.
  5. Нажмите **Сохранить**.
  6. Нажмите **Готово** , чтобы подтвердить изменения.


Лимит затрат для рекламного аккаунта будет обновлен примерно через 15 минут. После этого показ вашей рекламы продолжится. Сумма, которую вы уже потратили, будет применена к новому лимиту затрат.
**Примечание**. Если вы получите сообщение об ошибке, в котором указано, что вы внесли слишком много изменений в лимит затрат, подождите приблизительно час и попробуйте снова.
