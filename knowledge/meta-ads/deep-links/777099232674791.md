# Работа с инструментом Meta для настройки событий на сайтах
![](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=438223955063449&version=1666716831&transcode_extension=webp)
Вы можете получить пошаговые инструкции по настройке веб-событий от технического специалиста по решениям Meta. Узнайте, [_как запланировать звонок со специалистом Meta_](https://www.facebook.com/business/meta-pro-team/technical-pro?lead_source=sbg_2025_q4_ou_specialist_technical_helpcenter_xyz).
[Добавив базовый код пикселя Meta на сайт](https://www.facebook.com/business/help/952192354843755), вы можете использовать инструмент настройки стандартных событий и параметров без написания кода. Пока он работает только со стандартными событиями, но не со специально настроенными.
## Прежде чем начать
  * [Убедитесь, что на сайте установлен пиксель Meta.](https://www.facebook.com/business/help/952192354843755)
  * Убедитесь, что у вас есть права администратора для этого пикселя.
  * Отключите блокировку рекламы в браузере.
  * Чтобы вы могли настроить параметры ID и типа контента, в тексте или URL веб-страницы должен содержаться уникальный идентификатор товара (например, SKU или UPC). Параметры ID и типа контента доступны только для следующих событий: "Просмотр контента", "Добавление в корзину", "Начало оформления заказа", "Покупка", "Поисковый запрос", "Добавление в список желаний" и "Лид". 


## Использование инструмента настройки событий
  1. Откройте [**Meta Events Manager**](https://www.facebook.com/events_manager2/). 
  2. Нажмите **Наборы данных** ![](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=408108114296304&version=1774528167&transcode_extension=webp) в меню в левой части страницы. 
  3. Выберите название и ID нужных данных. 
  4. Откройте вкладку **Настройки** в верхней части страницы. 
  5. В разделе **Настройка события** нажмите **Открыть инструмент настройки событий**.
  6. Укажите URL и нажмите **Открыть сайт**. Инструмент настройки событий запустится в браузере. 
  7. Для каждого рекомендуемого события нажмите **Проверка** , а затем — **Подтвердить** или **Отклонить**. 
  8. Чтобы добавить событие, которого нет в списке рекомендуемых, перейдите на сайт и найдите нужные кнопки или страницы. Выберите **Отслеживать новую кнопку** или **Отслеживать URL** , а затем следуйте инструкциям на экране.  
**Примечание.** Настраивая событие для кнопки, вы добавляете его ко всем кнопкам на сайте, у которых аналогичный или похожий текст. 
  9. Выберите событие.
  10. Настройте параметры для события.

### Настройка параметров значения и валюты
  1. В разделе **Значение** нажмите **Выбрать значение на странице** , если хотите добавить параметр значения. 
  2. Инструмент выделит цены на странице сайта. Выберите значение (цену), которое соответствует событию. После того как вы в первый раз выберете значение, инструмент будет автоматически заполнять значения на других страницах.
  3. Выберите валюту в раскрывающемся меню в разделе **Валюта**. 
**Примечание.** Если вы уже настроили соответствующее событие "Начало оформления заказа", то сможете выбрать вариант **Использовать значение из события "Начало оформления заказа"** , когда будете настраивать событие "Покупка". Инструмент будет автоматически добавлять в событие "Покупка" значение, установленное для события "Начало оформления заказа". Если вариант **Использовать значение из события "Начало оформления заказа"** недоступен, укажите вариант **Выбрать значение на странице**.

### Настройка ID контента и параметров типа контента
  1. Нажмите **Добавить ID и тип контента** , чтобы настроить эти параметры для события.
  2. Нажмите **Выбрать ID контента на странице** , если ID контента доступен на странице. Выберите **Извлечь ID контента из URL** , если ID контента содержится в URL сайта.
**Примечание.** Если на странице несколько ID контента, добавьте любые два из них, выбрав вариант **Добавьте ещё один ID контента**. Мы постараемся найти все похожие ID контента на странице.
  3. Настройте ID контента, следуя инструкциям на экране.
  4. Выберите тип контента.
  5. Нажмите **Подтвердить**.


Чтобы настроить [объявления из каталога Meta Advantage+](https://www.facebook.com/business/help/733979527611858), выберите один и тот же ID контента для всех этапов пути пользователя. Например, выберите один и тот же ID контента для событий "Просмотр контента", "Добавление в корзину" и "Покупка". 
  1. Когда всё будет готово, нажмите **Завершить настройку**.


Чтобы посмотреть события, перейдите в Meta Events Manager, выберите пиксель и посмотрите график и таблицу для каждого события. Чтобы проверить, правильно ли регистрируются события, выберите пиксель, откройте вкладку **Тестирование событий** и следуйте инструкциям на экране. Подробнее о том, [как использовать инструмент тестирования событий](https://www.facebook.com/business/help/2040882565969969?id=1205376682832142).
