# О специально настроенных конверсиях для веб-событий
![](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=438223955063449&version=1666716831&transcode_extension=webp)
Это статья о специально настроенных конверсиях для веб-событий. Специально настроенные конверсии пока недоступны для событий в приложении. Подробнее о том, [как создать специально настроенную конверсию с помощью офлайн-событий](https://www.facebook.com/business/help/182135238967018).
Специально настроенные конверсии позволяют создавать правила для событий. Благодаря этому вы можете измерять более конкретные действия клиентов. Например, можно отфильтровать все события покупок, чтобы сосчитать только покупки женских туфель дороже 40 долл. США. Затем можно использовать эти специально настроенные конверсии в качестве целей по результативности и охватывать [людей](https://www.facebook.com/business/help/283579896000936), которые с наибольшей вероятностью совершат важные для вас действия.
## Преимущества специально настроенных конверсий
С помощью специально настроенных конверсий можно:
  * **Фильтровать события.** Добавляйте правила к стандартным событиям, специально настроенным событиям и событию "Весь трафик для URL", чтобы более подробно анализировать действия клиентов. Например, можно создать правило, позволяющее считать количество людей, купивших товары определенного цвета или стоимости. [Подробнее.](https://www.facebook.com/business/help/462806641140703)
  * **Настраивать стандартные события без дополнительного кода.** Если на сайте есть базовый код пикселя Meta, вы можете использовать правила для URL, чтобы регистрировать стандартные события, не добавляя дополнительный код. [Подробнее.](https://www.facebook.com/business/help/723326514751348)
  * **Выбирать в качестве цели по результативности специально настроенную конверсию.** Создавая объявление в Meta Ads Manager, вы можете выбрать в качестве цели по результативности специально настроенную конверсию.
  * **Контролировать, какие данные передаются партнерам.** Передавайте деловым партнерам данные по отдельным специально настроенным конверсиям, а не источники данных целиком. Подробнее о том, [как делиться специально настроенными конверсиями в Events Manager](https://www.facebook.com/business/help/3313524248726315).


## Пример специально настроенной конверсии
Например, интернет-магазин хочет повысить продажи мужской одежды дороже 50 долл. США. Владелец может создать специально настроенную конверсию, которая будет регистрироваться каждый раз, когда кто-либо совершит покупку. Для этого потребуется выбрать событие конверсии "Покупка" и создать 2 указанных ниже правила.
  1. **Правило для URL** : URL страницы, на которой происходит покупка, содержит ключевое слово "mens" (мужская).
  2. **Правило для параметра** : для параметра указано значение стоимости больше 50 долл. США. 


Если специально настроенная конверсия произойдет достаточное количество раз, ее можно будет использовать в качестве цели по результативности. Это значит, что мы сможем показывать рекламу людям, которые с наибольшей вероятностью купят мужскую одежду дороже 50 долл. США.

## Управление специально настроенными конверсиями
Удалять специально настроенные конверсии, редактировать их и управлять ими можно в Meta Events Manager.
**Примечание.** Если вы создаете специально настроенную конверсию с помощью рекламного аккаунта, который входит в бизнес-портфолио, то:
  * Люди с доступом к этому портфолио смогут просматривать специально настроенные конверсии и управлять ими, используя рекламный аккаунт или бизнес-портфолио.
  * Люди с доступом только к рекламному аккаунту смогут просматривать лишь те специально настроенные конверсии, которые были созданы в этом аккаунте, а также управлять ими.
