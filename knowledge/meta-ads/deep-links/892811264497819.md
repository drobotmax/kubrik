# Объявления из каталога Advantage+: обязательные события в приложении для Facebook SDK
Чтобы показывать объявления из каталога Advantage+ от Meta (предыдущее название — динамическая реклама) пользователям вашего мобильного приложения, добавьте в него Facebook SDK для Android или iOS. Вы сможете узнавать, какие действия совершают определенные люди (например, просматривают ваши товары или добавляют их в корзину), и решать, кому именно показывать рекламу. Без этих данных объявления из каталога Advantage+ не смогут давать полезные рекомендации людям, которые, вероятно, купят ваши товары.
Чтобы использовать объявления из каталога Advantage+ для мобильного приложения, в него нужно добавить три обязательных события:  
| События для iOS  | События для Android  |  
| --- | --- |  
| FBSDKAppEventNameViewedContent  | EVENT_NAME_VIEWED_CONTENT  |  
| FBSDKAppEventNameAddedToCart  | EVENT_NAME_ADDED_TO_CART  |  
| FBSDKAppEventNamePurchased  | EVENT_NAME_PURCHASED  |  
Узнать, как добавлять эти события в приложении для [Android](https://developers.facebook.com/docs/app-events/getting-started-app-events-android) и [iOS](https://developers.facebook.com/docs/app-events/getting-started-app-events-ios), можно на сайте Meta for Developers.
