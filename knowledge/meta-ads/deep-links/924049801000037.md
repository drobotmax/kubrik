# Создание групп похожих аудиторий для оптимизации ставок
![](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=438223955063449&version=1666716831&transcode_extension=webp)
Похожие аудитории ограничены или недоступны для использования в рекламе финансовых продуктов и услуг, а также жилья и трудоустройства. [Подробнее.](https://www.facebook.com/business/help/298000447747885)
Созданные аудитории не пересекаются, поэтому вам не придется исключать другие более или менее схожие аудитории. 
  * Аудитория с наибольшим уровнем сходства (0–1 %). 
  * Аудитория со средним уровнем сходства (1–3 %). 
  * Аудитория с наименьшим уровнем сходства (3–6 %). 


## Создание группы похожих аудиторий
Чтобы создать группу похожих аудиторий:
  1. Откройте раздел [Аудитории](https://www.facebook.com/ads/manage/audiences.php).
  2. Нажмите **Создать аудиторию** и выберите **Похожая аудитория**.
  3. Выберите исходную аудиторию и страну для похожей аудитории. 
  4. Выберите количество создаваемых аудиторий. 
  5. Перемещайте ползунки, чтобы выбрать степень сходства ваших аудиторий с исходной аудиторией.
  6. Нажмите **Создать аудиторию**.


Вы можете создать группу похожих аудиторий, чтобы назначать разные ставки для аудиторий с разной ценностью.
