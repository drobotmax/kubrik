# Об условиях для автоматических правил
![](https://static.xx.fbcdn.net/mci_ab/public/cms/?ab_b=e&ab_page=CMS&ab_entry=438223955063449&version=1666716831&transcode_extension=webp)
Новинка: добавлено 21 новое условие. Также добавлены новые расширения для фильтров. 
  

Создавая правила для кампаний, групп объявлений или отдельных объявлений, вы выбираете условия, при которых они должны срабатывать. Выбрав нужное условие, вы можете вводить для него собственные значения.
Предположим, вы хотите знать, когда кто-то нажимает на вашу рекламу и совершает на вашем сайте покупку на сумму от 5 до 10 долл. США. При создании правила вы, скорее всего, выберете условие **[Цена за начало оформления заказа](https://www.facebook.com/business/help/1925989814089259)**.
В правило можно добавить несколько условий. Однако для срабатывания правила ваша кампания, группа объявлений или отдельное объявление должны соответствовать всем условиям. Если вы хотите, чтобы действие совершалось при выполнении нескольких условий, создайте для каждого условия по отдельному правилу.
