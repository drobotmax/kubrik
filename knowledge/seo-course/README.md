# SEO Course Knowledge Base

SEO-курс от Сергея Кудрявцева (компания "Новые клиенты", @newclients1).
Курс ориентирован на предпринимателей и маркетологов, которые хотят понимать SEO-продвижение
для коммуникации с подрядчиками и контроля качества работ.

## Modules

1. **Что такое SEO** - определение, органическая выдача, SEO vs контекст, факторы ранжирования
2. **Семантика** - семантическое ядро, маркерные запросы, сбор вручную/автоматом, кластеризация
3. **Внутренняя оптимизация** - технические факторы, мета-теги, структура, контент, юзабилити
4. **Внешняя оптимизация** - ссылочная масса, методы линкбилдинга, соцсети, репутация
5. **Результат в SEO** - связь позиций и трафика, метрика, видимость сайта
6. **Взаимодействие клиента и SEO-студии** - стоимость, сроки, чек-лист, roadmap 12 мес
7. **Отчетность в SEO** - виды отчетов, KPI, план работ
8. **Ответственность клиента** - оплата, помощь бизнесу в SEO
9. **Продажи SEO-услуг** - поиск клиентов, каналы, воронка продаж, документы
