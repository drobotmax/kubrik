# Playbook: копирайтинг для холодной аудитории

> Источники: [[books/sugarman-copywriting.md]], [[books/cialdini-influence.md]], [[books/building-a-storybrand.md]], [[books/breakthrough-advertising.md]], [[copywriting-course/]], [[team-playbooks/anti-patterns.md]]
> Последнее обновление: 2026-04-12
> Тип: playbook

## Суть

Пошаговый процесс написания рекламного текста для аудитории, которая не знает о продукте (Problem Aware и Solution Aware по Schwartz). Синтез из 4 книг + курса копирайтинга + team experience.

## Шаг 1: Определить уровень осведомленности

Холодная аудитория = Problem Aware (~20%) или Solution Aware (~10%).

- **Problem Aware:** знают боль, не знают что решение есть -> начинать с проблемы
- **Solution Aware:** знают что решения есть, не знают о нашем -> начинать с результата

См. [[awareness-levels]] для полной таблицы.

## Шаг 2: Сформулировать Job (JTBD)

"Когда я [ситуация], я хочу [мотивация], чтобы [результат]"

Три измерения:
- Функциональное: что конкретно нужно (купить квартиру, запустить рекламу)
- Эмоциональное: как хочу себя чувствовать (уверенность, спокойствие)
- Социальное: как хочу выглядеть (успешный инвестор, заботливый родитель)

**Копирайтер адресует все три.** Эмоциональное - главный драйвер покупки (StoryBrand).

## Шаг 3: Построить StoryBrand каркас

1. **Герой** (клиент) хочет [job]
2. **Проблема:** External + Internal + Philosophical
3. **Наставник** (наш клиент) проявляет эмпатию ("мы понимаем") + авторитет (цифры, кейсы)
4. **План:** 3 простых шага
5. **CTA:** прямой + переходный
6. **Провал:** что будет если не действовать
7. **Успех:** трансформация

## Шаг 4: Собрать текст по Slippery Slide

**Hook (заголовок):**
- Problem Aware: "Почему [симптом]?" / "Если вы [ситуация] - вот что происходит"
- Solution Aware: "Как [результат] за [срок] - без [боли]"

**Engage (тело):**
- 1-2 принципа Cialdini: Social proof + Authority (самые универсальные для cold)
- Семена любопытства в замедляющихся местах
- Для B2B/инвесторов - стилистические переходы, не прямые затравки

**CTA:**
- Прямой: "Оставьте заявку" / "Запишитесь на просмотр"
- Переходный: "Скачайте подборку" / "Получите расчет"

## Шаг 5: Checklist перед сдачей

- [ ] Заголовок цепляет за 2 секунды?
- [ ] Все три уровня проблемы затронуты?
- [ ] Есть минимум 2 принципа Cialdini?
- [ ] Горка без обрывов - каждое предложение тянет к следующему?
- [ ] CTA ясный и конкретный?
- [ ] Нет фейкового дефицита или манипулятивных таймеров?
- [ ] Текст соответствует техтребованиям платформы? (см. platform-specs.md)

## Примеры по вертикалям

### Недвижимость РФ (Problem Aware)
**Hook:** "Квартира в Москве за 8 млн - миф или реальность в 2026?"
**Engage:** Social proof (X семей уже нашли) + план из 3 шагов
**CTA:** "Получите подборку квартир в вашем бюджете"

### Bali инвестиции (Solution Aware)
**Hook:** "Как получать 12-15% годовых в $ на вилле, которой можно пользоваться"
**Engage:** Authority (ROI конкретного объекта) + Unity (сообщество инвесторов)
**CTA:** "Скачайте каталог объектов с расчётом доходности"
