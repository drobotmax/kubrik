## Что это

Лендинги в Яндекс Директе - бесплатные одностраничные сайты, создаются прямо в интерфейсе Директа на основе описания бизнеса или ссылки на существующий сайт. В апреле 2026 вышли из беты. С момента запуска создано более 90 000 лендингов.

Лендинги можно использовать как:
- Сайт-визитку (описание бизнеса, контакты, форма заявки, кнопки действия)
- Посадочную страницу для объявлений Директа и других каналов
- **Резервную посадочную страницу** в ЕПК - ключевая новая функция

## Что нового (2026)

### Резервная посадочная страница в ЕПК

В Единой перформанс-кампании (ЕПК) теперь можно указать резервную посадочную страницу в настройках кампании. При нестабильном интернет-соединении пользователя она отображается вместо страниц основного сайта.

**Важно для настройки:** если используется резервная посадочная, нужно указать счётчик Метрики и добавить цели из него - Директ будет оптимизировать трафик под вероятность целевого действия.

### Адаптивная структура

Структура сайта формируется индивидуально с учётом специфики бизнеса и отрасли (ранее был единый шаблон).

### Обновлённый алгоритм подбора изображений

Нейротехнологии автоматически подбирают релевантные изображения по тематике бизнеса.

### Уникальная ссылка (кастомный URL)

Можно создать уникальный адрес лендинга, ассоциирующийся с брендом.

### SEO-продвижение

Улучшения видимости в поиске Яндекса добавлены автоматически - дополнительная настройка не нужна.

### Настройка цветов

Оформление в фирменном стиле без привлечения дизайнера.

### Галерея изображений

Блок для ключевых предложений, акций, примеров работ или преимуществ.

### Быстрая навигация по странице

Якорные ссылки для перехода к нужным разделам.

### Кнопки CTA в любом блоке

Призывы к действию можно добавить в любую часть страницы.

### Установка логотипа и фона

Брендинг страницы без дизайнера.

### Экспорт заявок в API

Заявки с лендингов экспортируются в CRM через API. Сервис: **Leads**, метод: **get**.

## Для кого актуально

- Клиенты без сайта (или со слабым сайтом)
- Запуск акций и выход в новые ниши
- Кампании в условиях нестабильного интернета (актуально для регионов)
- Быстрое создание и тестирование посадочных страниц

## Применение для клиентов KUBRIK

**Резервная посадочная в ЕПК** - важный инструмент для клиентов с нестабильным хостингом или при тестовых запусках без полноценного лендинга. Позволяет не терять показы и конверсии при проблемах с основным сайтом.

**Алгоритм настройки резервной посадочной:**
1. Создать лендинг в Директе (или использовать существующий)
2. В настройках ЕПК указать лендинг как резервную посадочную страницу
3. В стратегии указать счётчик Метрики
4. Добавить цели из счётчика для оптимизации трафика

**Экспорт заявок в API** - закрывает потребность в CRM-интеграции без программирования со стороны клиента (при наличии CRM с поддержкой API).
